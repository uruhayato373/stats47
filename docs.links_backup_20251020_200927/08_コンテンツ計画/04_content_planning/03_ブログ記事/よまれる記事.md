# 読まれる記事執筆ガイドライン

## 分析結果：週間300ビュー vs 20ビューの差異

### 高ビュー記事（日照時間ランキング）の成功要因

#### 1. タイトルの魅力

- **具体的な問いかけ**：「最も太陽の恵みを受ける県はどこか」
- **普遍的な関心事**：天気・気候は誰もが経験する身近なテーマ
- **検索されやすいキーワード**：「日照時間」「都道府県」「ランキング」

#### 2. 冒頭の構成

- **読者への関連性を明示**：農業、再生可能エネルギー、健康への影響を提示
- **専門用語の丁寧な説明**：「日照時間とは」セクションで定義を明確化
- **データの見方を解説**：統計指標の意味を事前に説明

#### 3. コンテンツの充実度

- **詳細な分析**：各都道府県について数値と背景を丁寧に解説
- **地理的考察**：なぜその結果になったのか地形や気候の理由を説明
- **複数の視点**：ベスト5、ワースト5、地域的特徴、前年比較など多角的
- **約3,500文字の充実したボリューム**

#### 4. ビジュアル要素

- **データの視覚化**：ランキング表やグラフで理解を助ける
- **見出し画像**：記事の内容を象徴する画像

#### 5. 実用的価値

- **購入者特典**：CSV生データのダウンロード提供
- **応用可能性**：太陽光発電、農業計画などへの活用を示唆

---

### 低ビュー記事（平均初婚年齢）の課題

#### 1. タイトルの弱さ

- **サブタイトルのみ**：「結婚の地域性」というぼんやりした表現
- **具体性の欠如**：何を知れるのか不明確
- **検索性の低さ**：「からみる」という曖昧な表現

#### 2. 冒頭の不足

- **いきなり本題**：導入や背景説明がない
- **読者への訴求なし**：なぜこのデータが重要か説明されていない
- **専門用語の説明不足**：平均初婚年齢の定義や意義が不明

#### 3. コンテンツの薄さ

- **淡白な記述**：都道府県名と数値の羅列が中心
- **分析の浅さ**：なぜその結果になったのか十分に考察されていない
- **約1,200文字と短い**：読み応えに欠ける

#### 4. 構成の問題

- **同じパターンの繰り返し**：夫と妻で同じ内容を2回書いている
- **深掘りの不足**：「社会的背景」や「今後の課題」が箇条書きのみ

## 読まれる記事を書くための実践ガイドライン

### 1. タイトル設計の原則

#### やるべきこと

- **具体的な問いを立てる**：「〜は何か？」「どこが一番〜か？」
- **数字を入れる**：「ベスト5」「2023年度」など
- **検索キーワードを意識**：読者が検索しそうな言葉を使う
- **好奇心を刺激**：予想外の結果や意外性を示唆

#### 避けるべきこと

- 抽象的な表現（「〜からみる」「〜について」）
- 専門用語だけのタイトル
- 長すぎるタイトル（60文字以内推奨）

### 2. 冒頭（リード文）の黄金パターン

```markdown
1. 読者の日常・関心事との接点（2-3文）
2. 記事で扱うテーマの重要性（2-3文）
3. 記事の内容予告（1-2文）
4. 専門用語の定義（必要に応じて）
```

**例：**

> 「〇〇について気になったことはありませんか？実は都道府県によって大きな差があります。 本記事では2023年度の最新データから、〇〇の実態を徹底分析します。」

### 3. コンテンツの充実方法

#### 必須要素（最低3,000文字以上）

1. **導入**（10%）：背景・定義・記事の目的
2. **データ提示**（30%）：ランキングや数値の提示
3. **詳細分析**（40%）：各項目の深掘り・理由の考察
4. **横断的考察**（15%）：地域的特徴・傾向の分析
5. **まとめ**（5%）：要点の再確認・今後の展望

#### 各都道府県の記述で含めるべき内容

- **数値データ**：具体的な数字と偏差値
- **前年比較**：増減率や変化の傾向
- **地理的要因**：なぜその結果になったのか
- **社会的背景**：産業、文化、政策との関連
- **具体例やエピソード**：読者がイメージしやすい描写

### 4. 構造化のテクニック

#### 見出しの使い方

- **H2（##）**：大きなセクション（ベスト5、ワースト5など）
- **H3（###）**：個別項目（各都道府県、各テーマ）
- **H4（####）**：補足説明や詳細

#### 箇条書きの効果的活用

- 複数の要因を列挙する際
- 特徴を整理して示す際
- 読者への提案やアドバイス

**注意：** 箇条書きだけに頼らず、文章での説明も充実させる

### 5. ビジュアル要素の活用

#### 必ず含めるべき要素

- **見出し画像**：記事の内容を象徴する魅力的な画像
- **データ図表**：ランキング表、グラフ、チャート
- **地図**：地域性を扱う場合は必須
- **比較表**：前年比、地域比較など

#### 画像の配置タイミング

- 冒頭：読者の関心を引く
- データ提示直後：理解を助ける
- 各大セクションの始め：視覚的な区切り

### 6. 読者を惹きつける文章技法

#### やるべきこと

- **問いかけから始める**：「〜をご存知でしょうか？」
- **具体例を挙げる**：「例えば〜」「具体的には〜」
- **対比を使う**：「一方で〜」「それに対して〜」
- **意外性を示す**：「予想外の結果として〜」
- **読者に語りかける**：「私たちの〜」「あなたの地域では〜」

#### 避けるべきこと

- 事務的・報告書的な文体
- 専門用語の連続使用
- 淡白すぎる箇条書きの羅列
- 同じパターンの繰り返し

### 7. SEO対策とキーワード戦略

#### タイトル・本文に含めるべきキーワード

- **年度**：「2023年度」「最新」
- **地域**：「都道府県」「ランキング」
- **具体的なテーマ**：該当する統計指標名
- **関連用語**：そのテーマに関連する検索されやすい言葉

#### キーワードの配置場所

1. タイトル
2. 最初の段落（冒頭100文字以内）
3. 各見出し
4. まとめセクション

### 8. 付加価値の提供

#### 読者が「保存したい」「共有したい」と思う要素

- **データのダウンロード提供**：CSV、PDFなど
- **時系列データの比較**：過去数年分のトレンド
- **実用的な示唆**：ビジネス、生活への応用方法
- **予測や展望**：今後の変化についての考察
- **関連記事へのリンク**：類似テーマの記事紹介

### 9. 更新・改善のチェックリスト

記事公開前に確認すべき項目：

- [ ] タイトルは具体的で魅力的か？
- [ ] 冒頭で読者の関心を引いているか？
- [ ] 最低3,000文字以上あるか？
- [ ] 各データに背景説明と考察があるか？
- [ ] 画像・図表が適切に配置されているか？
- [ ] 専門用語は説明されているか？
- [ ] まとめセクションで要点を再確認しているか？
- [ ] 検索されやすいキーワードが含まれているか？
- [ ] 読者にとっての実用的価値があるか？
- [ ] 誤字脱字はないか？

## テンプレート例

### 都道府県ランキング記事の基本構造

```markdown
# [年度] [指標名]の都道府県ランキング — [魅力的なサブタイトル]

## 導入（300-500文字）
- 読者の関心事との接点
- テーマの重要性・社会的意義
- 記事の内容予告

## [指標名]とは（200-300文字）
- 定義の説明
- 測定方法
- なぜ重要なのか

## 統計データの見方（150-200文字）
- 使用する指標の説明
- 偏差値、ランク、差分などの意味

## [指標名]ランキング ベスト5（1,200-1,500文字）
### 1位：[都道府県名]（[数値]）
- 数値とランキング
- 偏差値と前年比
- 地理的・気候的要因
- 社会的・経済的背景
- 具体例やエピソード

[2位〜5位も同様に記述]

## [指標名]ランキング ワースト5（1,200-1,500文字）
[ベスト5と同じ構造で記述]

## 地域的特徴（400-600文字）
- 地域ごとの傾向
- 意外な結果
- 地理的要因の分析

## 前年度からの変化（300-400文字）
- 変化の大きかった地域
- 全国的なトレンド
- 変化の要因考察

## まとめ（300-500文字）
- 主要な発見の要約
- 実用的な示唆
- 今後の展望

## 購入者特典（必要に応じて）
- ダウンロード可能なデータセット
- 詳細な分析資料
```

## 最後に

**読まれる記事の本質は「読者への価値提供」**です。

- データを示すだけでなく、**なぜそうなのか**を説明する
- 数字を並べるだけでなく、**読者の生活との接点**を示す
- 情報を提供するだけでなく、**理解と発見の喜び**を与える

この3つを意識すれば、自然と読者に支持される記事が書けるようになります。
