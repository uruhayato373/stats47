# Ranking Utils

このディレクトリは、ランキング機能の様々な場所で再利用される、汎用的なヘルパー関数（ユーティリティ）を格納します。

## 責務

- **汎用的な処理の提供**: データ形式の変換、文字列操作、計算など、特定のビジネスドメインに依存しない、純粋で再利用可能な関数を提供します。
- **状態を持たない (Stateless)**: `utils` の関数は、原則として副作用を持たない純粋関数であるべきです。同じ入力に対しては、常に同じ出力を返します。
- **自己完結**: アプリケーションの他のレイヤー（特に `services`）に依存せず、単体で機能するように設計します。

## 原則

- `utils` 内の関数は、特定のビジネスロジックを含んではいけません。
- 外部リソースへのアクセス（API通信やDBクエリなど）は、`../services` ディレクトリが担当します。
