# Google Earth Studio (GES) 47都道府県 旋回動画生成

Google Earth Studio 用のプロジェクトファイル（.esp）を47都道府県分生成し、手動でレンダリングするための独立アプリ。

## ⚠️ 重要な制約

- **GESはバッチレンダリング非対応**: 47都道府県の動画は1つずつレンダリングする必要があります
- **推定レンダリング時間**: 1動画 5-10分 × 47 = **4-8時間**
- **公式API非提供**: 手動レンダリングが最も確実な方法です

## ワークフロー

```
Step 1: .esp ファイルを47都道府県分生成
Step 2: Google Earth Studio で1つずつ手動レンダリング
```

---

## Step 1: プロジェクトファイル生成

### 実行方法

```bash
cd apps/ges
pnpm generate
# または縦動画 (1080x1920) を生成（デフォルト）
npx tsx scripts/generate-projects.ts

# 横動画 (1920x1080) を生成
npx tsx scripts/generate-projects.ts --aspect 1920-1080
```

### オプション

| オプション | 説明 | 例 |
|-----------|------|-----|
| `--aspect 1080-1920` | 縦動画 (デフォルト) | `pnpm generate` |
| `--aspect 1920-1080` | 横動画 | `pnpm generate -- --aspect 1920-1080` |
| `--limit N` | 先頭 N 件のみ生成（テスト用） | `pnpm generate -- --limit 3` |

### テンプレート

| ファイル | サイズ | 用途 |
|---------|--------|------|
| `templates/1080-1920.esp` | 縦 (9:16) | Reels / TikTok 向け |
| `templates/1920-1080.esp` | 横 (16:9) | YouTube / Web 向け |

テンプレートは神戸（lat=34.691269, lng=135.183025）を基準とした GES プロジェクトです。GES で調整・保存した .esp ファイルをこのパスに置くと、本番に近い動画になります。

### 出力先

```
.local/r2/ges/
  1080-1920/   # 縦動画用
    01000.esp
    02000.esp
    ...
    47000.esp
  1920-1080/   # 横動画用
    01000.esp
    ...
```

## Step 2: 手動レンダリング

1. **Google Earth Studio にアクセス**
   ```
   https://www.google.com/earth/studio/
   ```

2. **プロジェクトを開く**
   - 「Open Project」をクリック
   - `dist/ges_projects/1080-1920/` から .esp ファイルを選択してアップロード

3. **プレビュー確認**
   - タイムラインを再生してカメラ旋回動作を確認

4. **レンダリング設定**
   - 画面右上の「Render」ボタンをクリック
   - **解像度**: 1080x1920 (縦) または 1920x1080 (横)
   - **フレームレート**: 30fps または 60fps
   - **フォーマット**: MP4 (H.264) 推奨
   - **品質**: High または Maximum

5. **レンダリング開始 → ダウンロード**
   - 「Start Render」をクリック
   - ⏱️ 所要時間: 5-10分程度
   - レンダリング完了後、自動ダウンロード開始

6. **47都道府県分繰り返す**

### 所要時間の目安

| フェーズ | 所要時間 |
|---------|---------|
| .esp ファイル生成（全47件） | **5分以内** |
| 手動レンダリング（47都道府県） | **4-8時間** |
