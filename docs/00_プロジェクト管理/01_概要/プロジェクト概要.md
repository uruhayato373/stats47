# プロジェクト概要

## プロジェクトについて

「統計で見る都道府県」は、日本の地域統計データを可視化するWebアプリケーションです。

e-Stat APIを中心に、政府統計、自治体データ、民間統計など多様なデータソースから47都道府県の統計データを取得し、直感的なグラフとチャートで表示します。

複雑な統計データを分かりやすい形で提供し、誰もが地域の現状を理解できるプラットフォームを構築することで、データドリブンな地域理解を促進し、より良い地域社会の構築に貢献することを目指しています。

また、統計データの背景や解釈、地域の特徴を解説するブログ機能により、データの理解を深め、地域への関心を高めることを目指しています。

## ブランドコンセプト

### プロジェクト名称

- **ブランド名**: 統計で見る都道府県
- **技術名**: stats47
- **キャッチフレーズ**: データで知る、地域の今

### ミッション

複雑な統計データを分かりやすく可視化し、誰もが地域の現状を理解できるプラットフォームを提供することで、データドリブンな地域理解を促進し、より良い地域社会の構築に貢献する

### ビジョン

地域統計データの「民主化」を実現し、すべての人が統計データを使って地域を理解・比較・分析できる社会を目指す

### コアバリュー

1. **わかりやすさ**: 専門知識がなくても理解できる
2. **信頼性**: 公的統計に基づく正確なデータ
3. **中立性**: 客観的なデータ提示
4. **アクセシビリティ**: 誰もが無料で利用可能

### ブランドパーソナリティ

- **知的で親しみやすい**: 統計の専門性を保ちつつ、親しみやすい表現
- **誠実で信頼できる**: データの正確性と出典の明示
- **革新的**: 新しい可視化手法とユーザー体験
- **包括的**: 専門家から一般ユーザーまで幅広く対応

### トーン＆マナー

#### 文章スタイル

- **丁寧だが堅苦しくない**: 「です・ます」調、専門用語には解説を添える
- **具体的でわかりやすい**: 抽象的な表現は避け、具体例を示す
- **中立的**: 特定の地域を過度に持ち上げたり貶めたりしない

#### 避けるべき表現

- 過度に専門的な統計用語（解説なし）
- センセーショナルな表現
- 断定的すぎる主張

#### 推奨する表現

- 「データから読み取れること」
- 「統計で見ると」
- 「他の地域と比較すると」

## 主要機能

### データ可視化機能

- **地域選択機能**: 都道府県レベルでの地域選択
- **多様なデータソース**: e-Stat API、政府統計、自治体データ、民間統計の統合
- **データ可視化**: 棒グラフ、折れ線グラフ、円グラフ、コロプレス地図
- **地域比較**: 複数地域のデータを同時に比較表示
- **時系列分析**: 過去データとの比較とトレンド分析

### ブログ機能

- **統計解説記事**: データの背景や解釈を詳しく解説
- **地域特集**: 各都道府県の特徴や魅力を紹介
- **データストーリー**: 統計データから読み取れる地域の物語
- **専門家コラム**: 統計学や地域研究の専門家による寄稿

### ユーザー体験

- **レスポンシブデザイン**: デスクトップ、タブレット、モバイル対応
- **ダークモード**: 自動システム設定検出と手動切り替え
- **SEO最適化**: 検索エンジンでの発見性向上
- **アクセシビリティ**: 誰もが利用しやすい設計
