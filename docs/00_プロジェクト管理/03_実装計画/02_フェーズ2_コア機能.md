# フェーズ 2: 機能拡張（3-6 ヶ月）

## 概要

**期間**: 3-6 ヶ月  
**目標**: エンゲージメント機能の追加とユーザー体験の向上  
**収益目標**: 月 5-10 万円（広告 + 有料コンテンツ）

## 目標

- エンゲージメント機能の追加
- ユーザー体験の向上
- 分析・改善機能の実装
- 有料コンテンツ販売の開始

## 完了基準

### 技術指標

- [ ] 平均セッション時間 5 分以上
- [ ] 直帰率 50%以下
- [ ] ユーザーフィードバック 4.0/5.0 以上

### ビジネス指標

- [ ] 月間 PV 10 万以上
- [ ] 広告収入 月 2 万円以上
- [ ] 有料コンテンツ収入 月 2 万円以上
- [ ] 月次収益 5-10 万円達成

- [ ] 前年比・前年同期比計算
- [ ] 期間選択 UI（スライダー）
- [ ] 複数地域の時系列比較（最大 5 地域）

- [ ] データポイントのツールチップ表示
- [ ] レスポンシブ対応

- [ ] 市区町村 ID と e-Stat 地域コードの対応表作成
- [ ] TopoJSON ファイルの検証とテスト

**地図コンポーネント実装**:

- [ ] react-leaflet のセットアップ
- [ ] 歴史的行政区域データセットの TopoJSON 読み込み実装
- [ ] コロプレス地図コンポーネント実装
- [ ] カラースケール実装（Sequential、Diverging）
- [ ] 分類方法実装（等間隔、分位数、自然分類）
- [ ] インタラクティブ機能（ズーム、パン、ホバー）
- [ ] ツールチップ表示（地域名、統計値、順位）
- [ ] 凡例とカラーバー
- [ ] 統計指標・年度選択 UI
- [ ] クリックで詳細表示機能
- [ ] 全画面表示モード
- [ ] アクセシビリティ対応（色覚異常対応、キーボードナビゲーション）
- [ ] レスポンシブ対応

**国土地理院タイル統合**:

- [ ] GsiTileLayer コンポーネント実装
- [ ] ベースマップ選択 UI 実装（標準地図、淡色地図、白地図、航空写真）
- [ ] ベースマップ切り替え機能
- [ ] 国土地理院の出典表示追加
- [ ] タイルレイヤーの最適化（遅延読み込み、キャッシュ）

**データソース表記**:
