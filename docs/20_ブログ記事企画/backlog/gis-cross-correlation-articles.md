# 掛け合わせ相関記事 ブログ企画 (型B)

## 概要

掛け合わせコンテンツ戦略 (型B = 統計×統計相関) の記事企画。相関全再計算
(1,789,251 ペア) の mining で精選した「意外な相関」9 件を、ブログ記事 8 本に企画
(テレワーク関連 2 相関は 1 記事に統合)。検索流入 (SEO) 向け。

データ源: stats47 の `correlations` テーブル (2026-05-20 再計算)。各記事は公開前に
当該 2 指標の県別値を集計し、上位/下位/外れ値を確定してから本文を書く。

## 共通の執筆規律 (全記事必須)

- **相関 ≠ 因果**: 相関があっても一方が他方の原因とは限らない。「○○だから××」と
  断定しない。「○○の県は××も多い傾向」という観測事実ベースで書く
- **集計してから筋を決める**: 企画の仮説は出発点。県別データを集計し、外れ値・
  パターンを見てから記事の物語を確定する
- **r の扱い**: 相関係数を正確に (例 +0.90 は「強い正の相関」、+0.4 台は「弱い」)。
  誇張しない
- **第三の変数に触れる**: 多くは「都市度」「所得」「高学歴層の厚さ」など背景要因が
  両方を動かしている。背景要因の存在に必ず言及する

## 各記事の企画

### A. 大学卒の多い県は海外旅行も多い
- metric: `final-education-university-graduate-school-ratio` × `overseas-travel-annual-participation-rate-15plus`
- 検索意図: 「海外旅行 都道府県」「学歴 地域差」
- 構成案: 冒頭フック (素朴な予想) → 2 ランキング対比 → 一致する県/外れる県 → 背景要因 (高学歴層は所得・都市度も高い=海外旅行のハードルが低い) → 相関は強いが因果でない注記 → まとめ
- 注意: 学歴・所得・都市度が絡む。「学歴が海外旅行を生む」と書かない

### B. テレワークの県は高学歴で学び続けている
- metric: `telework-rate` × `final-education-university-graduate-school-ratio` / `study-participation-rate-english`
- mining 候補 #2 (テレワーク×英語学習 +0.92) と #3 (大学卒×テレワーク +0.86) を統合
- 検索意図: 「テレワーク 都道府県」「テレワーク 普及率」
- 構成案: テレワーク実施率ランキング → 高学歴割合との一致 → さらに英語学習など自己研鑽の行動者率とも連動 → 「在宅で働ける職種=知識労働=高学歴・学び続ける層」という共通基盤 → まとめ
- 注意: 3 指標が同じ「知識労働層の厚さ」を映している可能性。指標の独立性に注意

### C. 人口が増える県はスポーツも盛ん
- metric: `future-population-change-rate-2050` × `sports-annual-participation-rate-10plus`
- 検索意図: 「将来人口 都道府県」「スポーツ 実施率」
- 構成案: 将来人口増減率ランキング → スポーツ行動者率との対比 → 一致の解釈 (人口が増える=若い県/都市部=スポーツ環境も) → 外れ値 → 注記 → まとめ
- 注意: 「若年人口比率」が両方を動かす第三の変数。年齢構成に必ず触れる

### D. 音楽を聴く県は旅行にも出る
- metric: `hobby-participation-rate-popular-music` × `travel-participation-rate-overnight`
- 検索意図: 「趣味 都道府県」「旅行 行動者率」
- 構成案: 2 つの「行動者率」ランキング対比 → 文化的にアクティブな県という共通像 → 背景 (余暇時間・可処分所得・都市度) → まとめ
- 注意: 「行動者率」系は余暇全般の活発さの代理。音楽と旅行の直接の因果ではない

### E. 通勤圏の広い県は海外旅行も多い
- metric: `commuter-ratio-to-other-municipalities` × `overseas-travel-annual-participation-rate-15plus`
- 検索意図: 「通勤 都道府県」「海外旅行 地域」
- 構成案: 他市区町村通勤者比率ランキング → 海外旅行行動者率との一致 → 解釈 (大都市圏近郊=通勤圏が広く所得も移動志向も高い) → まとめ
- 注意: 三大都市圏近郊効果。「移動に慣れた県」という解釈は推測なので断定しない

### F. 資産を持つ県は高学歴
- metric: `current-securities-balance-ratio-multi-person-households` × `final-education-university-graduate-school-ratio`
- 検索意図: 「有価証券 都道府県」「資産 地域差」
- 構成案: 有価証券保有割合ランキング → 大学卒割合との対比 → 解釈 (高学歴=高所得=金融資産形成) → 注記 → まとめ
- 注意: 所得が両方の背景。資産形成の地域差は所得格差の反映、と正確に

### G. 持ち家の県は単独世帯が少ない
- metric: `owner-occupied-housing-ratio` × `single-person-household-ratio`
- 検索意図: 「持ち家比率 都道府県」「単独世帯」
- 構成案: 持ち家比率ランキング → 単独世帯割合との負の相関 → 解釈 (持ち家=ファミリー世帯中心の県/借家=単身者の多い都市) → まとめ
- 注意: 負の相関。住宅形態と世帯構成は表裏。都市/地方の構造差

### H. マンションの県は高齢世帯が少ない
- metric: `apartment-ratio` × `household-ratio-with-65plus`
- 検索意図: 「共同住宅 都道府県」「高齢世帯」
- 構成案: 共同住宅比率ランキング → 65歳以上世帯割合との負の相関 → 解釈 (マンション中心=都市部=若い世帯/戸建て中心=地方=高齢世帯) → まとめ
- 注意: 負の相関。#2 日照×太陽光の「都市度=戸建て率」と同じ構造。記事間で重複しすぎないよう切り口を変える

## 優先順位 (案)

A・B が検索ボリューム・話題性ともに高そう → 先行。G・H は #2 と構造が近い (都市/地方) ため後ろ。
ただし優先は GSC データや反応を見て調整。
