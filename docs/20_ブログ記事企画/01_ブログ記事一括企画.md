# ブログ記事企画管理

## 方針

- 3つのアプローチ（カテゴリ / トレンド / アフィリエイト）で企画を生成
- 企画確定後、記事を執筆・公開したらそのエントリを削除（ライフサイクルルール）
- `/update-blog-plan` で定期的に本ファイルを最新化

- バッチ1（7カテゴリ × 5本 = 35本）: **全件公開済み**
- バッチ2（7カテゴリ × 5本 = 35本）: **全35本公開済み（2026-03-09）**
