# ターゲットペルソナ

## 概要

stats47プロジェクトのターゲットユーザーを明確化し、機能開発、コンテンツ戦略、UI/UX設計の指針とするためのペルソナ定義です。

4つの主要ペルソナを設定し、それぞれのニーズと期待を明確にすることで、プロダクト開発の優先順位付けとマーケティング戦略の最適化を図ります。

## ペルソナ1: 一般ユーザー

### 基本情報

**名前**: 田中 美咲（たなか みさき）  
**年齢**: 35歳  
**職業**: 会社員（マーケティング職）  
**居住地**: 東京都  
**家族構成**: 独身  

### 背景・ライフスタイル

- 地方移住や地域活性化に興味を持っている
- 週末は地方のイベントに参加することも
- SNSで情報収集し、友人とシェアすることが多い
- データに基づいた意思決定を好む

### 技術リテラシー

- **レベル**: 中程度
- **デバイス**: スマートフォン（iPhone）、ノートPC
- **利用サービス**: SNS、ニュースアプリ、動画配信サービス
- **統計知識**: 基礎的な理解はあるが、専門用語は苦手

### 課題・ペインポイント

- 統計データは難しく感じる、どこを見ればいいかわからない
- 移住先を検討したいが、地域の実態がよくわからない
- グラフや表が多すぎて、何が重要なのか判断できない
- 専門的すぎる説明は理解できない

### ニーズ

- わかりやすいグラフと視覚的な表現
- 専門用語の平易な解説
- 地域の「生活実感」に近い指標
- 他の地域との比較
- スマートフォンで快適に閲覧できる

### stats47の利用シーン

1. **移住先検討**: 地方移住を検討する際の情報収集
2. **地域理解**: 自分の住む地域の特徴を知りたい
3. **話題収集**: SNSでシェアできる面白いデータ探し
4. **趣味の研究**: 興味のある地域について深く知る

### 期待する機能・コンテンツ

**機能**:
- 直感的で美しいデータビジュアライゼーション
- モバイル対応（スマホファースト）
- 簡単な地域比較機能
- SNSシェア機能

**コンテンツ**:
- 平易な言葉で書かれたブログ記事
- 「住みやすさ」「子育て環境」など生活に密着したテーマ
- データから読み取れる「地域の物語」
- ビジュアル重視のインフォグラフィック

### 成功指標

- 月1-2回の訪問
- 1セッションあたり5分以上の滞在
- 2-3記事の閲覧
- SNSでのシェア

## ペルソナ2: 地域研究者

### 基本情報

**名前**: 佐藤 健太（さとう けんた）  
**年齢**: 28歳  
**職業**: 大学院生（地域社会学専攻）  
**居住地**: 京都府  
**家族構成**: 独身  

### 背景・ライフスタイル

- 地域社会学の博士課程に在籍
- 地方の過疎化と都市化について研究
- 週に2-3回は図書館や研究室で作業
- 学会発表や論文執筆が主な活動

### 技術リテラシー

- **レベル**: 高い
- **デバイス**: ハイスペックノートPC、タブレット
- **利用サービス**: 学術データベース、統計ツール（R、Python）、論文管理ツール
- **統計知識**: 専門的な統計知識を持つ

### 課題・ペインポイント

- 複数の統計を横断的に見たいが、データソースが分散している
- データのダウンロードや加工に時間がかかる
- e-Stat APIは直接使うのが面倒
- データの出典や更新日が不明確なサイトが多い

### ニーズ

- 詳細で正確なデータ
- 時系列データと地域間比較
- データのエクスポート機能（CSV、JSON）
- API利用による自動データ取得
- データの出典と更新日の明記

### stats47の利用シーン

1. **研究データ収集**: 論文執筆のためのデータ収集
2. **仮説検証**: 研究仮説を検証するためのデータ確認
3. **プレゼン資料作成**: 学会発表用のグラフ・図表作成
4. **文献レビュー**: 既存研究との比較データ取得

### 期待する機能・コンテンツ

**機能**:
- CSVダウンロード機能
- API提供（データアクセス）
- 詳細な統計情報とメタデータ
- カスタマイズ可能なグラフ
- 複数年次データの比較
- データの出典と更新日の表示

**コンテンツ**:
- 統計手法の解説
- データの限界や注意点の明記
- 専門的な分析記事
- 研究事例の紹介

### 成功指標

- 週1回以上の訪問
- 1セッションあたり15分以上の滞在
- データのダウンロード
- API利用

## ペルソナ3: 地域政策担当者

### 基本情報

**名前**: 鈴木 隆（すずき たかし）  
**年齢**: 42歳  
**職業**: 市役所企画課職員  
**居住地**: 愛知県（人口20万人の地方都市）  
**家族構成**: 既婚、子供2人  

### 背景・ライフスタイル

- 地域振興政策の立案・実行を担当
- 市議会への説明資料作成が重要な業務
- 他自治体の事例研究を日常的に実施
- 住民説明会での質疑応答に備える必要がある

### 技術リテラシー

- **レベル**: 中程度
- **デバイス**: 業務用PC（Windows）
- **利用サービス**: Excel、PowerPoint、メール、政府統計サイト
- **統計知識**: 基礎的な理解はあるが、高度な分析は苦手

### 課題・ペインポイント

- 他自治体との比較データ収集に時間がかかる
- 根拠データの収集に手間がかかる
- データの信頼性を示す必要がある
- グラフ作成に時間がかかる

### ニーズ

- 信頼できるデータソース
- 地域間比較（特に同規模自治体）
- トレンド分析（過去10年程度）
- そのまま使えるグラフやチャート
- データの出典明記

### stats47の利用シーン

1. **政策立案**: 新規政策の根拠データ収集
2. **議会資料作成**: 市議会提出資料の作成
3. **住民説明**: 住民説明会での説明資料作成
4. **他自治体比較**: 近隣自治体や同規模自治体との比較分析

### 期待する機能・コンテンツ

**機能**:
- 全国平均との比較
- 周辺自治体との比較
- データの信頼性表示（出典、更新日）
- グラフの画像ダウンロード
- 印刷に適したレイアウト

**コンテンツ**:
- 政策に関連する統計解説
- 他自治体の成功事例
- データの読み方ガイド
- 住民向けの平易な解説

### 成功指標

- 月2-3回の訪問（政策立案時）
- 1セッションあたり10分以上の滞在
- 複数の統計データの閲覧
- グラフのダウンロード

## ペルソナ4: 地域ビジネス経営者

### 基本情報

**名前**: 山田 裕子（やまだ ゆうこ）  
**年齢**: 50歳  
**職業**: 地域企業経営者（小売チェーン）  
**居住地**: 福岡県  
**家族構成**: 既婚、子供2人（成人）  

### 背景・ライフスタイル

- 地域密着型の小売チェーンを経営（店舗数15）
- 新規出店計画を検討中
- 商工会議所の役員も務める
- 地域経済の動向に敏感

### 技術リテラシー

- **レベル**: 中程度
- **デバイス**: PC、タブレット（iPad）
- **利用サービス**: ビジネスニュース、経済統計サイト、会計ソフト
- **統計知識**: ビジネスに必要な基礎知識

### 課題・ペインポイント

- 市場調査データが高額
- 地域の経済動向把握が難しい
- 出店候補地の商圏分析に時間がかかる
- 無料で信頼できるデータソースが少ない

### ニーズ

- 人口動態（年齢構成、世帯構成）
- 経済指標（所得、消費支出）
- 産業構造データ
- 商圏分析に使えるデータ
- 将来予測データ

### stats47の利用シーン

1. **出店計画**: 新規出店候補地の市場調査
2. **マーケティング戦略**: ターゲット顧客層の分析
3. **経営判断**: 地域経済トレンドの把握
4. **事業計画策定**: 中長期計画の根拠データ収集

### 期待する機能・コンテンツ

**機能**:
- 商圏分析（人口密度、年齢構成）
- 産業別データ
- 人口推移予測
- 消費動向データ
- 地域経済指標の可視化

**コンテンツ**:
- ビジネス活用事例
- 地域経済の解説記事
- 消費トレンド分析
- 業界別の統計解説

### 成功指標

- 月1回以上の訪問
- 1セッションあたり10分以上の滞在
- 複数地域の比較
- データのダウンロード

## マーケティング戦略への影響

### ターゲット別アプローチ

**ペルソナ1向け**:
- **チャネル**: SNS（Instagram、Twitter）、ブログメディア
- **訴求ポイント**: 「地域の魅力を再発見」「データで見る地域の姿」
- **コンテンツ**: ビジュアル重視のインフォグラフィック

**ペルソナ2向け**:
- **チャネル**: 学術コミュニティ、研究者ネットワーク
- **訴求ポイント**: 「研究に使える信頼性の高いデータ」「API提供」
- **コンテンツ**: 技術ブログ、データセット紹介

**ペルソナ3向け**:
- **チャネル**: 自治体ネットワーク、行政向けメディア
- **訴求ポイント**: 「政策立案を支援」「効率的なデータ収集」
- **コンテンツ**: 活用事例、他自治体の成功事例

**ペルソナ4向け**:
- **チャネル**: ビジネスメディア、商工会議所
- **訴求ポイント**: 「ビジネス判断を支援」「無料で市場調査」
- **コンテンツ**: ビジネス活用事例、ROI分析
