### 優先度1: 薬局テーマ（表示 803 回）

#### B記事案: 「なぜ佐賀県は人口あたりの薬局数が日本一なのか」

- **シリーズ**: B（ランキング読み物）
- **A記事リンク先**: `a-pharmacy-count-per-100k`（note 内リンク）
- **stats47 リンク先**: /socialsecurity/pharmacies/ranking/pharmacy-count-per-100k

**使えるデータ（DB既存）**:

| データ | ranking_key | 備考 |
|---|---|---|
| 薬局数（人口10万人対） | pharmacy-count-per-100k | Top: 佐賀(64.5), 高知(60.4), 山形(60.0) / Bottom: 沖縄(39.4), 千葉(42.7), 埼玉(43.7) |
| 薬局数（面積100km²対） | pharmacy-count-per-100km2 | 面積あたりで見ると都市部が上位に逆転する可能性 |
| 薬局数（実数） | pharmacy-count | 人口あたりと実数のギャップ |
| 精神科病床数（人口10万人対） | psychiatric-bed-count-per-100k | 相関 r=0.55 |
| 精神科病院数（人口10万人対） | psychiatric-hospital-count-per-100k | 相関 r=0.54 |

**記事の切り口**:
1. クイズの答え「佐賀県」を冒頭で明示 → 読者の検索意図を即座に満たす
2. 「なぜ地方に薬局が多いのか」── 医薬分業の進展度、門前薬局、高齢化率との関係
3. 面積あたりで見ると景色が変わる ── 都市部の薬局密集 vs 地方の1薬局カバー面積
4. 「詳しいランキング全47都道府県版はこちら」→ A記事へ

**検索意図との相性**: ★★★（クイズ回答需要に B 記事で「なぜ」を深掘り → A 記事で全順位確認）

#### B記事案: 「日本一暑い県はどこか ── 最高気温・猛暑日・熱帯夜で見る暑さの正体」

- **A記事リンク先**: `a-maximum-temperature`（note 内リンク）

**使えるデータ（DB既存）**:

| データ | ranking_key | 備考 |
|---|---|---|
| 最高気温 | maximum-temperature | 2024年最新 |
| 最低気温 | lowest-temperature | 寒い県との対比 |
| 年平均気温 | average-temperature | 平均と極端値の違い |
| 年間日照時間 | annual-sunshine-duration | 暑さと日照の関係 |

**記事の切り口**:

3. フェーン現象と内陸性気候の解説（なぜ海沿いでなく内陸が暑いのか）
