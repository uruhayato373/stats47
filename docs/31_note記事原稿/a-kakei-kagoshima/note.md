## 鹿児島県の消費の個性

祭具・墓石が全国平均の5.4倍、葬儀関係費やつきあい費も高水準。先祖供養と地域の付き合いを大切にする鹿児島の文化が、家計にはっきりと表れています。
揚げかまぼこ（さつま揚げ）は全国1位で、郷土の食文化が日常の支出に直結しています。一方、かれいやかき(貝)など本州で好まれる魚介の消費は極端に少なく、地元で獲れるさばやあじ中心の魚食スタイルがうかがえます。
女子用学校制服が全国1位、国公立中学校・高校への支出も高く、公教育を軸にした子育ての姿が見えます。温暖な気候のもと、灯油やコートの支出は全国最低水準です。
