## 千葉県の消費の個性

バス通学定期代が全国1位、鉄道通学定期代も全国上位と、郊外から都心へ通う通勤・通学費用が家計に大きくのしかかっています。
遊園地入場・乗物代も全国1位で、テーマパークを身近に楽しめる地の利が消費に直結しています。
教育費は私立中学校・大学ともに全国平均の2倍以上で、都心の教育機関に子どもを通わせる首都圏郊外ならではの支出構造です。
アクセサリーが全国1位、婦人服も全国上位と、ファッション消費が高い点も千葉の家計の特徴です。
