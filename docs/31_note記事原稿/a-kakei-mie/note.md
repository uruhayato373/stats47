葬儀関係費が全国平均の4倍超──。三重県の家計には、冠婚葬祭を大切にする地域性が色濃く刻まれています。

三重県（津市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、三重県の消費の「個性」を徹底的に洗い出しました。


## 三重県の消費支出の全体像

年間消費支出は**3,472,557円**で全国**35位**（全国平均3,602,915円の**0.96倍**）。
食料費の割合（エンゲル係数）は**28.9%**です。

690品目のうち**7品目で全国1位**、20品目で全国最下位を記録しています。

## 三重県の消費の個性

葬儀関係費が全国平均の4倍超、設備器具・設備材料も約3倍と、住まいと冠婚葬祭への支出が突出しています。持ち家に腰を据え、住宅の設備投資を惜しまない堅実な暮らしぶりが浮かび上がります。
教育では国公立志向が鮮明で、私立大学の支出は全国平均のわずか9%。一方で楽器や諸会費など地域活動への支出は厚く、ゲーム機の支出は全国最低水準です。
賃貸家賃の低さや酒類の少なさも含め、三重県の家計は「地に足のついた暮らし」を映し出しています。
