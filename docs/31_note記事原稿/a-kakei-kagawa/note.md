日本そば・うどんの外食費が全国平均の2.15倍で全国1位──。「うどん県」の看板は、家計データでもしっかり裏付けられています。

香川県（高松市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、香川県の消費の「個性」を徹底的に洗い出しました。


## 香川県の消費支出の全体像

年間消費支出は**3,564,469円**で全国**27位**（全国平均3,602,915円の**0.99倍**）。
食料費の割合（エンゲル係数）は**28.5%**です。

690品目のうち**8品目で全国1位**、7品目で全国最下位を記録しています。

## 香川県の消費の個性

うどん関連の支出(外食・乾麺・生麺)がすべて全国1位で、「うどん県」の名に恥じない消費パターンです。
住居の工事費が全国平均の3〜6倍と突出しており、持ち家の維持・改修にしっかりお金をかける姿勢が際立ちます。
一方で被服費は全国平均の0.85倍、外食(うどん以外)も控えめで、日々の暮らしは堅実そのもの。理美容費が高いのも特徴で、カット代やパーマネント代が全国平均を大きく上回っています。
「うどんと住まいにはお金を惜しまない」という、香川県らしいメリハリのある家計が浮かび上がります。
