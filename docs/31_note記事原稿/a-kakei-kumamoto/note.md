「他の生鮮肉」が全国平均の2.1倍で全国1位、飲酒代も2.1倍で全国1位──。馬肉文化と酒を囲む交際の活発さが、熊本の家計を特徴づけています。

熊本県（熊本市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、熊本県の消費の「個性」を徹底的に洗い出しました。


## 熊本県の消費支出の全体像

年間消費支出は**3,904,288円**で全国**8位**（全国平均3,602,915円の**1.08倍**）。
食料費の割合（エンゲル係数）は**28.9%**です。

690品目のうち**12品目で全国1位**、3品目で全国最下位を記録しています。

## 熊本県の消費の個性

「他の生鮮肉」と飲酒代がともに全国1位。馬肉文化と酒を囲む活発な交際が、熊本の消費を語るうえで欠かせないキーワードです。
被服では婦人用ファンデーションが全国平均の4.4倍で全国1位、ネクタイも4.2倍で全国1位と、おしゃれへの投資意欲が目立ちます。
鉄道通勤定期代は全国平均の0.15倍と極端に低い一方、バスやタクシーの支出は高水準。路面電車とバスが市民の足となっている熊本ならではの交通構造が表れています。
年間消費支出は全国8位と高く、九州の中でも消費意欲の旺盛な県であることがデータから読み取れます。
