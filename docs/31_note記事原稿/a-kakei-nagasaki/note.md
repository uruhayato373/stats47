## 長崎県の消費の個性

婚礼関係費が全国平均の16倍超で堂々の全国1位。冠婚葬祭を盛大に行う長崎の文化が、家計の数字にもはっきりと表れています。
カステラは全国平均の7.3倍で、南蛮文化の伝統がいまも暮らしに息づいていることがわかります。
坂の街ならではの交通事情として、自転車購入はほぼゼロに近い一方、バス通勤定期代・タクシー代は全国トップクラス。地形が移動手段を規定し、家計構造そのものを形づくっています。
国内遊学仕送り金が全国平均の1.85倍と高く、進学のために子どもを県外に送り出す家庭が多いことも長崎の特徴です。
