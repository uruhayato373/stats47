鉄道通勤定期代が全国平均の2.26倍で全国1位──。琵琶湖のほとりから京阪神へ通う「通勤県」の姿が、家計データにくっきり表れています。

滋賀県（大津市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、滋賀県の消費の「個性」を徹底的に洗い出しました。


## 滋賀県の消費支出の全体像

年間消費支出は**3,853,247円**で全国**11位**（全国平均3,602,915円の**1.07倍**）。
食料費の割合（エンゲル係数）は**29.7%**です。

690品目のうち**13品目で全国1位**、6品目で全国最下位を記録しています。

## 滋賀県の消費の個性

鉄道の通勤・通学定期代がともに全国1位。京阪神への通勤圏という立地が、家計に最も大きな影響を与えています。
家具・インテリアへの支出も突出しており、テーブル・ソファーは全国1位の3.7倍。新興住宅地での新生活への投資意欲が数字に表れています。
葬儀関係費や信仰祭祀費の高さは、琵琶湖周辺に根づく伝統行事や地域の絆の強さを感じさせます。
「通勤は都市、暮らしは地方」という滋賀県ならではのライフスタイルが、690品目の消費データから浮かび上がります。
