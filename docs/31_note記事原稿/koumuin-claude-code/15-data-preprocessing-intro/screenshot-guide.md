# 統計データ加工: 公務員のための Claude Code × データ前処理入門 — スクリーンショット撮影ガイド

## 撮影前準備

### macOS コマンド
- 全画面: `Cmd + Shift + 3`
- 範囲指定: `Cmd + Shift + 4`
- ウィンドウ単体: `Cmd + Shift + 4` → `Space`
- クリップボード保存: 上記に `Ctrl` 追加

### Excel 推奨
- フォント: 游ゴシック / メイリオ 11pt
- ウィンドウサイズ: 1400×900
- 表示倍率: 100% (文字が潰れない)
- 行高: 標準のまま (e-Stat 原本の雰囲気を再現)

### マスキング原則
- e-Stat 公開データは元から個人情報なし (撮影対象として安全)
- ただし Excel タイトルバーのフルパス: `/Users/user/projects/stats-demo/inputs/c01.csv`
- Office アカウント名 (右上) があれば塗りつぶし
- 開いているサイドバー・最近使ったファイル一覧に個人情報が混じる場合は閉じる

### 保存先
- ディレクトリ: `docs/31_note記事原稿/koumuin-claude-code/15-data-preprocessing-intro/images/`
- 命名規則: `screenshot-N-<short>.png`
- 圧縮: 500KB 超は `pngquant --quality=70-90` で再圧縮

## 撮影リスト

### Shot 1: e-Stat 国勢調査 CSV の Excel 表示 (1 画面に全体構造)

- **本文位置**: 「### Step 2: 元データを観察させる」末尾、プロンプト例の直後
- **撮影対象**: e-Stat からダウンロードした国勢調査 CSV (例: 統計表 ID `0003448237` 都道府県別年齢別人口) を Excel で開き、1 画面に以下すべてが見える状態:
  - 上部 (1-3 行目): タイトル「令和 2 年国勢調査 男女別人口及び世帯数」/ 基準日「2020-10-01 現在」/ 単位「単位: 人」
  - 中央 (5-6 行目): 結合セル絡みの 2 段ヘッダ (「都道府県」「総数」「男」「女」「世帯数」+ 再掲列)
  - 中央下: データ本体 (北海道〜数県のデータ行)
  - 下部: 末尾の注釈行 (「注 1) ...」「資料: 総務省統計局」)
- **準備するもの**:
  1. e-Stat (https://www.e-stat.go.jp) から対象 CSV をダウンロード
  2. 文字コード: 多くは Shift_JIS (`cp932`) → Excel で文字化けする場合は `iconv -f cp932 -t utf-8 input.csv > input-utf8.csv` 変換
  3. Excel で開き、必要なら行高を縮めて 1 画面に上 (タイトル) + 中 (ヘッダ + データ数行) + 下 (注釈) を入れる
  4. 注釈までスクロールで届かない場合は Excel の「ウィンドウの分割」または 1 行飛ばし表示で対応
- **マスキング項目**:
  - Excel タイトルバーのフルパス → `/Users/user/projects/stats-demo/inputs/c01.csv`
  - Office アカウント (右上) を塗りつぶし
  - サイドバー / 最近使ったファイル一覧を非表示
  - データ自体は公開統計のためマスキング不要
- **推奨ファイル名**: `screenshot-1-estat-csv-structure.png`
- **撮影手順**:
  1. e-Stat から CSV ダウンロード → Excel で開く
  2. Excel ウィンドウを 1400×900 にリサイズ
  3. 表示倍率を 100% に設定、必要なら 90% に縮小し全体構造が 1 画面に入るよう調整
  4. 1-3 行目のメタ情報 / 5-6 行目のヘッダ / 数行のデータ / 末尾注釈が同時に見えるよう行高調整 (注: Excel 「ホーム > 書式 > 行の高さ」で 18px 程度)
  5. リボン UI を最小化 (`Ctrl + F1` で表示切替)
  6. `Cmd + Shift + 4` → `Space` → Excel ウィンドウクリックで撮影
  7. プレビューでタイトルバー個人情報を塗りつぶし、不要な周辺領域を切り抜き
  8. 撮影画像を見て「読者が e-Stat 原本の障害ポイント (タイトル/単位/結合ヘッダ/注釈) を一目で理解できるか」をチェック

## 撮影後手順

1. PNG を `images/` ディレクトリに保存
2. 500KB 超なら `pngquant --quality=70-90` で圧縮
3. `draft.md` のマーカー行を `![e-Stat 国勢調査 CSV の典型レイアウト (タイトル/ヘッダ/データ/注釈)](./images/screenshot-1-estat-csv-structure.png)` に置換
4. 個人情報残存チェック:
   - Excel タイトルバーに実ユーザー名 (`/Users/<実名>/`) が残っていないか
   - Office アカウント名・組織名 (右上) が映っていないか
   - サイドバーの最近使ったファイル一覧に個人ファイル名が映っていないか
   - `pngcheck -t` でメタデータ (Author / Title) 削除確認
