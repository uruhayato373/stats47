## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

「会議が終わるたびに 30-60 分かけて議事録を起こしている」「速記が苦手で、後から聞き返すと細部が抜ける」——自治体職員の典型的な悩みです。

人口 5-30 万人規模の市役所では、係長級が週 3-5 本の会議を抱える例が珍しくありません。1 本の議事録に 30-60 分、月の累計で 5-10 時間が消える計算です。

解決策はシンプル。
録音 mp3 を Whisper で文字起こし → Claude Code で議事録形式に整形。
1 時間の会議で実作業 5 分、Whisper 処理を含めても 10 分以内。

ただし、最初に絶対のルールを 1 つ。
庁内会議録は対象外。個人情報・特定個人情報・決裁前検討・人事案件を含む会議は、絶対にこのフローに乗せない。

対象は「私的勉強会・公開研修・自主研究グループ・自治体間連絡会 (公開議事録扱い)」に限定します。守秘の線引きを最初に確認することが、AI 活用の前提です。

本記事は無料で全文公開。録音機材選び、Whisper API vs whisper.cpp の精度比較、CLAUDE.md でのテンプレ化まで一気に解説しました。
