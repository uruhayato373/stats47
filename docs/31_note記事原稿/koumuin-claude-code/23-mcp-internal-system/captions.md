## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

MCP (Model Context Protocol) は、Anthropic が 2024 年に公開した「Claude Code から外部システムを呼ぶ標準プロトコル」。

公務員の業務に当てはめると、庁内システム (財務会計・人事給与・税務・住基・GIS など) を Claude Code から呼べる可能性が見えてきます。

ただし、LGWAN 系 (基幹系) で動かすには制約があります。
現実解は「Claude Code 本体はインターネット系、MCP server だけ LGWAN 系」という分離。3 層分離の中で、データを越境させない設計です。

実装難度は中程度。Node.js + JSON-RPC の知識があれば、1 週間で骨格は組めます。

セキュリティ上、MCP server には絶対条件があります。
(1) 読み取り専用 (書き込み権限を持たせない)
(2) ホワイトリスト (呼べる関数を限定)
(3) 監査ログ完備 (誰が・いつ・何を呼んだか全記録)

本記事は実証実験レベル。本番展開には情報セキュリティ委員会の承認と、ベンダー保守契約の見直しが必須です。

それでも、MCP の方向性は「内部システム連携の標準」になる可能性が高い。早期に触っておく価値があります。
