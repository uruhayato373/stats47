# 撮影ガイド: 決算書類の前年比較分析を 5 分で出す手順

## 撮影前準備

### macOS スクリーンショットコマンド

```bash
# 範囲選択 (推奨): Shift + Command + 4
# ウィンドウ単一: Shift + Command + 4 → Space → クリック
# 全画面: Shift + Command + 3
# クリップボード保存: 上記コマンドに Control を追加
```

### ターミナル / エディタ推奨設定

- ターミナル: iTerm2 または macOS Terminal、フォント `JetBrains Mono 14pt`、配色 `Solarized Dark`
- 財務会計システム画面が撮れない場合は **ダミー画面を Keynote / Figma で自作** する (本記事は実システム撮影を強く推奨しない)
- ウィンドウ幅は 1280 × 720 程度に整える
- 実システムを撮る場合は最低限のマスキング前提でテスト環境を使うこと

### マスキング原則

- システム名 (富士通・NEC・内田洋行 等のロゴ・製品名) → 完全黒塗り or ダミーUI に置換
- 自治体名・部署名 → `〇〇市財政課` 等の架空名
- 職員名・担当者ID → 黒塗り
- 決算金額 → CSV と整合する架空値 (例: `12,000,000`)
- ファイルパスの `/Users/<実名>/` → `/Users/user/`
- システムのバージョン番号・契約 ID → 黒塗り

## 撮影リスト

### Shot 1: 財務会計システムから決算 CSV をエクスポートする画面

- **本文位置**: Step 1 直後 (draft.md 67 行目)
- **撮影対象**: 財務会計システムの「決算データ出力 / CSV エクスポート」画面。年度・款項目選択 → 出力形式 (CSV UTF-8) → エクスポートボタンが見える状態
- **準備するもの**:
  - 推奨: Keynote / Figma で **架空の財務会計システム UI** を 1 画面作る (リスク回避)
  - 必須要素: 「決算年度: 2024 / 2025」「出力範囲: 款項目別」「形式: CSV (UTF-8)」「エクスポート」ボタン
  - 代替: 実システム撮影なら、テスト環境 + 全マスキング前提
- **マスキング項目**:
  - システム名・ベンダーロゴ → 黒塗り or 削除
  - 自治体名 (画面ヘッダ・タイトル等) → `〇〇市` 等の架空名
  - ログイン中の職員名・職員ID → 黒塗り
  - 一覧に表示される事業名・金額が実データの場合はすべてダミーへ
  - ウィンドウタイトルバーのシステム識別 URL → 黒塗り
- **推奨ファイル名**: `shot-01-zaimu-csv-export.png`
- **撮影手順**:
  1. **推奨ルート**: Keynote で 1280×720 の架空 UI を作成 → スライドを PNG 書き出し
  2. **実システムルート**: テスト環境にログイン → 決算データ出力画面へ遷移 → 年度範囲・科目を選択 → エクスポート直前で撮影
  3. `Shift + Command + 4` で範囲選択撮影
  4. プレビュー.app でシステム名・自治体名・職員名を矩形ツールで黒塗り
  5. 一覧の数字が実データなら全行をダミーで上書き (Keynote 再編集 or プレビュー注釈)
  6. 公開前にもう一度フルサイズで個人情報残りをチェック
