## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

庁内で「AI 勉強会をやってください」と頼まれる場面が増えています。

ところが 30 分という限られた時間で職員を Claude Code に入門させるのは、構成を間違えると破綻します。

成功する黄金比は次の通り。
(1) 説明 5 分: なぜやるか / 何ができるか
(2) デモ 5 分: 実演 1 つだけ (議事録要約が最強)
(3) ハンズオン 15 分: 参加者が自分で触る
(4) 振り返り 5 分: 質疑応答 + 次のステップ

成功条件は「事前準備」です。
参加者のサインアップ + Claude Code インストールを当日やると、それだけで 30 分が消えます。1 週間前にメール案内 + サポート時間設定で済ませておきます。

ハンズオン題材は「議事録 1 段落の要約」が最も成功率が高い。短い・誰でも分かる・効果が即実感できる、の 3 拍子が揃っています。

IT 苦手層への配慮も重要。コマンド入力ゼロの「ブラウザ版シナリオ」を用意して、2 トラック並走するのが現実解です。

最後の 5 分で「明日試したい業務を 1 つ書いて提出」+ 翌週メールフォローで、「やってみた率」が体感 3 倍になります。やりっぱなしの勉強会では効果が出ません。
