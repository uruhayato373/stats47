## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

「AI 活用のログを監査で提出してください」と内部監査・外部監査から求められる時代が来ます。

ログを残していなければ「組織として把握していない」と判定され、業務利用そのものに歯止めがかかる可能性も。

Claude Code の `.claude/settings.json` で Hook を組むと、以下を自動でログ化できます。
(1) プロンプト送信 (本文は残さず、ハッシュとトークン数だけ)
(2) ファイル変更 (パスと変更前後のサイズ)
(3) セッション終了 (時刻と総トークン数)

ログ内容:
「実行者・日時・対象ファイル・プロンプトハッシュ・トークン消費数」

本文を残さないのは、ログ自体が個人情報漏えいの新たなリスクにならないため。

改ざん検知のため、各ログエントリに前エントリの SHA-256 を含めます (チェーンハッシュ)。1 エントリを書き換えると以降のチェーンがすべて崩れる仕組みです。

月次レポート自動生成スクリプトで「監査担当者が読める形式」に変換すれば、提出依頼に即応できます。
