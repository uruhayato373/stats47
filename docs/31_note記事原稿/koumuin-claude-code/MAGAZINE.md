# note マガジン設定: 公務員 × Claude Code

note.com でマガジンを作成する際の設定値とコピー。Profile 5（note.com/stats47）で作成する。

## 課金モデル

- 種類: **買い切りマガジン**
- 価格: **¥1,980**（有料 23 本をバンドル）
- 個別記事: 有料 23 本は各 ¥300、無料 8 本は ¥0
- 訴求: 個別に全部買うと ¥300 × 23 = ¥6,900 → マガジンなら ¥1,980

## タイトル（30 文字以内）

```
公務員 × Claude Code 実務活用ガイド
```

（25 文字。代替案: 「自治体職員のための Claude Code 実務ガイド」27 字 / 「公務員のための Claude Code 活用ガイド」25 字）

## 説明文（案・note のマガジン説明欄にペースト）

```
「Claude Code を職場で使いたいけれど、何から始めればいいかわからない」——そんな自治体職員のための、実務特化の全 31 本シリーズです。

環境構築（庁内プロキシの突破）から、議事録・議会答弁・条例レビュー・補助金書類のチェック、個人情報を送らないセキュリティ設定、Excel 集計の自動化、上司への導入説明まで。「ターミナルって何？」というレベルの未経験者でも読める導入記事から、中級の実務適用までをカバーします。

執筆者は元自治体職員。現在は Claude Code で 47 都道府県の統計サイト stats47.jp（約 2,000 のランキングを毎日自動更新）を個人で開発・運用しています。「AI で本当に実務が回るのか」の答えは、実際に動いているものを見てもらうのが早い、という考えで書いています。

▼ このマガジンについて
・全 31 本（無料 8 本 + 有料 23 本）
・有料記事は個別 ¥300。本マガジン（¥1,980）なら有料 23 本すべて読めます
・自治体特有の制約（プロキシ閉域・管理者権限なし・稟議・監査）を踏まえた実務手順

まずは無料記事から。Claude Code が初めての方は「Claude Code とは何か — ターミナル未経験の公務員のための導入ガイド」をどうぞ。
```

## 作成後の手順

1. マガジン作成 → URL を取得（`https://note.com/stats47/m/m...`）
2. `node .claude/scripts/note/inject-magazine-url.cjs <マガジンURL>` で全 31 本の `{{MAGAZINE_URL}}` を実 URL に置換
3. `/publish-note` で公開（Phase 0 ガードが未注入を検知）
4. 公開した 31 本をマガジンに追加

収録記事の一覧は [INDEX.md](INDEX.md) の「31 本一覧」を参照。
