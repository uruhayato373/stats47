## 撮影前準備 (任意撮影に進む場合の共通事項)

### macOS 標準コマンド

- `Cmd + Shift + 3`: 画面全体
- `Cmd + Shift + 4`: 範囲選択 (推奨)
- `Cmd + Shift + 4` → `Space`: ウィンドウ単位

### 推奨設定

- ターミナル / エディタ: フォント 14pt、ウィンドウ 1200×800
- ライトテーマで統一
- 通知 OFF (`集中モード` ON)
- ブラウザ撮影時はシークレットウィンドウで個人プロファイル情報を排除

### マスキング原則

- **自治体名 / 部署名 / 退職者氏名 / メールアドレス / 電話番号 / 住所 / 実在企業名は完全マスキング**
- 退職者人物 → 「公務 太郎 (架空)」「自治 花子 (架空)」等の架空名で統一
- 自治体名 → 「○○市」「サンプル町」等の架空名
- 再就職先 → 「A 社 (民間コンサル)」「B 法人 (NPO)」等の匿名表記
- 給与水準・年収 → 「○○○万円台」程度のレンジ表現か、出典明示のうえ参考値として扱う
- 求人サイト URL のクエリパラメータも黒塗り (ユーザー ID 等が含まれることがある)
- ファイルパス `/Users/<実名>/...` → `/Users/user/...`

### 保存先

- 配置先: `docs/31_note記事原稿/koumuin-claude-code/30-post-retirement-career/images/`
- 命名規則: `screenshot-N-<short-keyword>.png`
- 圧縮: `pngquant --quality=70-90 --ext=.png --force *.png`

## 撮影後手順 (任意撮影を実施した場合)

1. **PNG 保存**: 撮影画像を `images/` 配下に推奨ファイル名で保存
2. **pngquant 圧縮**: `cd images && pngquant --quality=70-90 --ext=.png --force screenshot-*.png`
3. **draft.md マーカー置換**: 該当マーカーを `![alt](./images/screenshot-N-xxx.png)` に置換
4. **個人情報残存チェック**:
   - `grep -rE "([一-龥]{2,4}(市|町|村|区|県))|(@[a-z0-9.-]+\.(jp|com))|(0[0-9]{1,4}-[0-9]{1,4}-[0-9]{4})" images/` で何もヒットしないこと
   - 画像を 100% 表示で再確認 (求人サイトの細かい固有情報・条文の細字に注意)
   - 必要に応じて `Preview.app` のマーカー機能で黒塗り追記
