## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

公務員の退職後キャリアは、選択肢が広がっています。

特に「AI × 公的セクター経験者」は、近年急速に需要が伸びています。

選択肢 4 つ。
(1) 民間 DX 領域: 年収 700-1200 万円。「行政営業ができる技術者」の需要が高い
(2) GovTech スタートアップ: 公務員 OB 採用が増加、ストックオプションも選択肢
(3) NPO・社団法人: 「DX 顧問」ポジションで非常勤対応可能 (年金併用前提)
(4) 大学実務家教員: 拡大中。博士号がなくても公務員経験 + 技術スキルで採用例あり (JREC-IN で月 10-20 件公募)

退職後の市場価値は 3 軸の掛け算です。
(1) 公務員経験
(2) 技術スキル
(3) 対外実績

技術スキルだけでは若手エンジニアに勝てません。公務員経験と組み合わさってはじめて希少性が出ます。

ところが「対外実績ゼロ」だと、せっかくの組み合わせも見えません。在職中に next の 3 つのいずれかで実績を作っておくのが必須。
(1) note 発信
(2) 庁外講演
(3) GitHub 公開

対外実績ゼロ vs ありで、年収差 200-300 万。

年金 (年金払い退職給付) との併用設計、再任用との比較も記事内で整理しました。

note 有料記事 (¥1,200) では、4 選択肢別の年収レンジ、応募ルート、在職中の準備リスト 15 項目を公開しました。
