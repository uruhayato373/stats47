## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

「個人情報を AI に送らないでください」と研修で言われても、忙しい現場では人間の注意力が必ずどこかで切れます。

Claude Code の `UserPromptSubmit` hook は、プロンプト送信を「送信前に書き換え or 中止」できる唯一の仕組みです。人間の注意力に頼らない、自動マスキングが組めます。

マスキング対象は 5 種類。
(1) マイナンバー (12 桁数字)
(2) 電話番号 (固定・携帯)
(3) メールアドレス
(4) 郵便番号
(5) 氏名 + 敬称 (山田太郎様、田中花子さん など)

正規表現で検出してマスキング、あるいはブロックします。

監査面でも重要。マスキングログを `.claude/logs/masking/YYYY-MM-DD.jsonl` に残せば、監査時の「何が止まったか」「いつ・誰が試したか」が即答できます。

「完璧な防止」ではなく「人間の注意力に頼らない多層防御の 1 枚目」として設計するのがポイント。100% を目指すと運用負荷で挫折します。

note 有料記事 (¥1,200) では、5 種マスキング hook の完全な実装コード、監査ログのスキーマ、CI でのテスト方法までを公開しました。
