## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

条例改正案のレビューは、政策法務担当の負荷が最も高い業務の 1 つ。

定義の整合、引用条文の参照切れ、項番号のずれ、上位法 (地方自治法・関連個別法) との不整合、文体の統一、公用文表記基準への準拠——確認項目は多岐にわたります。

Claude Code を使えば、これらのチェックを 1 コマンドで自動化できます。

手順はシンプル。
(1) 改正案 + 現行条例 + 上位法 + 他自治体類似条例を 1 つの作業フォルダに集約
(2) `current/` `revision/` `reference/` `peers/` の 4 つに分けて配置
(3) `.claude/skills/ordinance-review/` を作って `/ordinance-review` で実行

出力は「該当条文番号・現状・想定対応・根拠」の 4 列 markdown 表。起案文の「政策法務協議結果」欄にそのまま貼れる形式です。

条例文は公開情報なので、個人情報マスキング設定の追加は不要。庁内 PC で動かない場合は、自宅 PC + 公開情報のみで作業可能です。

AI は「見落としを減らす補助線」。最終的な政策判断・上位法解釈の責任は人間が持ちます。
