## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

起案文・決裁文の差し戻しは、自治体職員の時間を最も奪う「見えない残業」の 1 つ。

「法令引用の表記が違う」「予算科目との整合が取れていない」「添付資料の参照番号が間違っている」——上司決裁直前で戻されて、修正・回し直しで半日が消えることも珍しくありません。

Claude Code に起案文・決裁文を投げて、20 項目のチェックリストで査読させると、形式エラーは 1 件あたり 5 分で潰せます。

チェック項目の例:
(1) 法令根拠の条文番号の表記統一
(2) 予算科目 (款・項・目・節) の整合
(3) 関連する過去決裁文の参照漏れ
(4) リスク記載 (財政・法務・住民影響) の網羅
(5) 決裁ルート (専決区分) と起案者印の整合
(6) 添付資料の参照番号と実ファイルの一致
(7) 起案理由の論理性 (根拠 → 必要性 → 効果)
...他 13 項目

最終的な政策判断・予算判断は人間が行います。AI は「形式エラーで戻される時間」を消すための補助線です。
