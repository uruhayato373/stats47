## 品質に関する注記

- **AI 4 並列フェーズ (リライト → SVG → 撮影ガイド → 第三者視点加筆) を計 ~2 時間で完遂**
- 全 30 本が「読んだ人がそのまま使える」レベル。一次情報マーカーは「第三者視点の典型例・規模感・公開資料引用」で完全置換
- 「私は」「自分は」等のナレーター一人称は本文から完全排除 (虚偽記載リスク回避)
- 加筆スタイル: 「人口 5-30 万人規模の市役所では…」「典型的な総務課のケースでは…」「2023 年地方公務員白書によれば…」など
- 公的統計引用 (総務省、e-Stat、地方公務員白書、Graffer 2025 等) で具体性を担保
- 「公務員語彙の正確性」「Claude Code 固有機能の網羅」「守秘配慮」の 3 原則を維持
- SVG は flow / structure / infographic の 3 タイプ。色使い・フォント (#FAFAFA / Hiragino) は全記事統一
- 撮影ガイドは 35 ショット分の具体手順 (マスキング指示・推奨ファイル名・撮影順序) を網羅
