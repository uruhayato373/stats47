## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

Claude Code を業務で使うには、情シス (IT 担当) の合議が要ります。

ところが「何を説明すればいいか」が分からないまま資料を作ると、突き返されて何往復もすることに。

情シスが本当に気にしているのは、技術仕様ではなく次の 3 点です。
(1) データはどこに行くか
(2) ログは取れるか
(3) 止められるか

Anthropic 公式の Data Usage Policy / Trust Center / SOC 2 Type II を引用すれば、技術論の 7 割は通ります。残り 3 割は組織としての運用ルール (責任・教育・停止手順)。

説明資料は A4 3-5 ページが黄金比。長すぎず短すぎず、想定 Q&A を 10 個別添。自治体規模 (政令市 / 中核市 / 一般市 / 町村) で情シスの判断基準が違うので、その差分も解説しています。
