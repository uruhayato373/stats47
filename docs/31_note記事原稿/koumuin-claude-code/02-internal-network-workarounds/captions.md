## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

「Claude Code を使いたいのに、庁内ネットワークが許してくれない」。自治体職員なら経験のある壁です。

技術的な抜け道は大きく 3 つ。
(1) 私物 PC + モバイル WiFi / テザリング (最も安全・確実)
(2) 庁内 PC + プロキシ正攻法 (情シス交渉 2-3 ヶ月)
(3) VDI / リモートデスクトップ経由 (監査責任が曖昧)

どれを選ぶかは、自治体の 3 層分離の運用と、自分の所属部署のリスク許容度で決まります。

絶対のルールは 1 つだけ。
「庁内データを送らない」。
LGWAN 端末・住基ネット端末では試さない。これだけは守る。

本記事 (無料) では各ルートの比較マトリクスと、ありがちな失敗例を 6 件紹介しています。
