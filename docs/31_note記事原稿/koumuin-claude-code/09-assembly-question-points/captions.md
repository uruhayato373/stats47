## Instagram — フィード投稿用キャプション (2200 字以内、改行は 2-3 行ずつ)

議会一般質問の通告書が出てくると、議会事務局では一気にバタつきます。

「この質問は総務課マターか、財政課か」「過去 5 年の答弁を遡って整合チェック」「想定される追加質問は何か」——経験者の暗黙知が支えてきた業務ですが、若手が育つまで時間がかかるのが課題でした。

Claude Code に通告書を投げると、論点・関係課 (機能名)・過去答弁参照キーワード・想定追加質問を一括抽出してくれます。所要時間 60 分 → 10 分が現実的です。

ポイント。
(1) 通告書は「質問項目」「背景説明」「想定答弁先」に分解できる定型構造
(2) 出力を関係課への振り分けメール本文にそのまま貼れる markdown 形式に
(3) 通告書は議会の公開資料なので個人情報設定の追加は不要
(4) 最終判断は議会事務局の経験で行う (経験者の暗黙知を AI が完全代替するのは現状不可)

AI は「最初の振り分け案」を作るだけ。経験者がレビューする構成にすれば、若手が経験を積みながら時短もできます。
