婚礼関係費への支出が全国平均の6.08倍──。「名古屋の結婚式は派手」という定評を、家計データが裏づけています。

愛知県（名古屋市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、愛知県の消費の「個性」を徹底的に洗い出しました。
