地代への支出が全国平均の4.38倍で全国1位──。持ち家志向が強い岐阜では、土地にしっかりとお金をかける家計が浮かび上がります。

岐阜県（岐阜市）は何にお金を使い、何にお金を使わないのか。

総務省の家計調査には**690品目にわたる消費支出データ**が都道府県庁所在市ごとに公表されています。
このデータを全品目・全国平均と比較して、岐阜県の消費の「個性」を徹底的に洗い出しました。


## 岐阜県の消費支出の全体像

年間消費支出は**3,437,158円**で全国**37位**（全国平均3,602,915円の**0.95倍**）。
食料費の割合（エンゲル係数）は**31.9%**です。

690品目のうち**13品目で全国1位**、6品目で全国最下位を記録しています。

全国平均を下回る内訳: **調理食品**（0.89倍） **酒類**（0.87倍）

## 岐阜県の消費の個性

地代が全国1位、パック旅行費や音楽月謝も全国トップと、住まいにも教養にもしっかりお金をかけるのが岐阜の家計の特徴です。
外食では和食・中華食への支出が目立ち、地元の柿やウイスキーへの高い消費も東海地方らしい豊かさを感じさせます。
中学校の補習教育費が全国平均の約2倍に達する教育投資の手厚さも特徴的で、名古屋圏の教育熱心な気風が岐阜にも色濃く及んでいます。
