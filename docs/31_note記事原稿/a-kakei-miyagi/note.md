## 宮城県の消費の個性

かまぼこ全国1位に象徴されるように、食の面では笹かまぼこやチューハイなど仙台ならではの消費が光ります。中華そばの支出も全国平均の約1.8倍と、ラーメン文化の根強さがうかがえます。
住居費では民営家賃が全国平均の約3倍と突出しており、東北の拠点都市としての賃貸需要の高さが表れています。
国内遊学仕送り金が全国平均の2倍超と高い一方、幼児教育費用や保育費用も高水準。子どもの教育にお金をかける姿勢が、仙台の家計の大きな特徴です。
