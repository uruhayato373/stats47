# DB 管理ツール（drizzle-kit studio）

admin アプリ廃止後、ローカル D1 (SQLite) の閲覧・編集には `drizzle-kit studio` を使用する。

## 起動方法

```bash
npm run db:studio
```

ブラウザで `https://local.drizzle.studio` が開く。

## できること

| 機能 | 説明 |
|---|---|
| テーブル一覧 | 全テーブルのスキーマ（カラム名・型・PK・NOT NULL）を表示 |
| レコード閲覧 | ページネーション付きでレコードを一覧表示 |
| フィルタリング | カラム値での絞り込み（例: `is_active = true`） |
| ソート | 任意のカラムで昇順・降順ソート |
| レコード編集 | セルをクリックして値を直接編集・保存 |
| レコード追加・削除 | GUI でレコードの追加・削除が可能 |
| SQL 実行 | 任意の SQL クエリを直接実行（SELECT / UPDATE / DELETE 等） |
| リレーション表示 | 外部キーで関連するレコードを表示 |

## 主な用途（旧 admin 機能の代替）

### ランキング項目管理（旧 `/ranking/items`）

```sql
-- 都道府県ランキングの一覧（有効のみ）
SELECT ranking_key, title, unit, is_active, is_featured
FROM ranking_items
WHERE area_type = 'prefecture' AND is_active = 1
ORDER BY category_id, title;

-- is_active のトグル
UPDATE ranking_items SET is_active = 0 WHERE ranking_key = 'xxx';

-- 項目数の確認
SELECT area_type, COUNT(*) as total,
       SUM(is_active) as active, SUM(is_featured) as featured
FROM ranking_items GROUP BY area_type;
```

### カテゴリ管理（旧 `/categories`）

```sql
SELECT * FROM categories ORDER BY display_order;
```

### 広告管理（旧 `/affiliate-ads`）

```sql
SELECT * FROM affiliate_ads ORDER BY created_at DESC;
```

## 注意事項

- drizzle-kit studio は **ローカル D1 のみ**を操作する
- **リモート D1 は Phase 10 (2026-04-29) で完全撤廃済み**。本番反映は以下のフロー:
  1. ローカル D1 を編集（drizzle-kit studio または `/populate-*`, `/register-ranking` 等のスキル）
  2. `/export-snapshots` で R2 snapshot を再生成
  3. `/push-r2` でリモート R2 に push
  4. `/deploy` で Cloudflare Workers デプロイ
- 詳細フロー: **`docs/01_技術設計/11_データ基盤設計.md`**
- バッチ処理: 相関分析は `/compute-correlation`、AI コンテンツは `/generate-ai-content` でスキル化済み
