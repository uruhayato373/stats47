# 地図実装ガイド

## 概要

Leaflet ライブラリを使用した地図コンポーネントの実装方法について説明します。コロプレスマップ、インタラクティブ地図、地理データの扱い方について詳しく解説します。

## 基本コンポーネント

### 1. ChoroplethMap（コロプレスマップ）

統計データを色分けして表示する地図コンポーネントです。

```typescript
import { useEffect, useRef } from "react";
import L from "leaflet";
import "leaflet/dist/leaflet.css";

interface ChoroplethMapProps {
  geoData: TopoJSON;
  values: Map<string, number>;
  config: MapConfiguration;
}

export const ChoroplethMap: React.FC<ChoroplethMapProps> = ({
  geoData,
  values,
  config,
}) => {
  const mapRef = useRef<L.Map>(null);
  const mapContainerRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (!mapContainerRef.current) return;

    // マップの初期化
    const map = L.map(mapContainerRef.current, {
      center: [35.6762, 139.6503], // 東京
      zoom: 6,
      zoomControl: true,
      attributionControl: true,
    });

    // ベースマップの追加
    L.tileLayer(config.basemap.url, {
      attribution: config.basemap.attribution,
      maxZoom: config.basemap.maxZoom,
    }).addTo(map);

    // TopoJSONデータの処理
    const geoJsonLayer = L.geoJSON(geoData, {
      style: (feature) => {
        const areaCode = feature.properties.areaCode;
        const value = values.get(areaCode);
        const color = value
          ? getColorForValue(value, config.colorScale)
          : "#f0f0f0";

        return {
          fillColor: color,
          weight: 1,
          opacity: 1,
          color: "white",
          dashArray: "",
          fillOpacity: 0.7,
        };
      },
      onEachFeature: (feature, layer) => {
        const areaCode = feature.properties.areaCode;
        const value = values.get(areaCode);

        if (value !== undefined) {
          layer.bindPopup(`
            <div>
              <h3>${feature.properties.areaName}</h3>
              <p>値: ${value.toLocaleString()}</p>
            </div>
          `);
        }
      },
    }).addTo(map);

    // マップの境界に合わせてフィット
    map.fitBounds(geoJsonLayer.getBounds());

    mapRef.current = map;

    return () => {
      map.remove();
    };
  }, [geoData, values, config]);

  return (
    <div
      ref={mapContainerRef}
      className="w-full h-96 rounded-lg border border-gray-200"
    />
  );
};
```

### 2. InteractiveMap（インタラクティブ地図）

ユーザーが操作できる地図コンポーネントです。

```typescript
interface InteractiveMapProps {
  geoData: TopoJSON;
  onAreaClick?: (areaCode: string, areaName: string) => void;
  onAreaHover?: (areaCode: string, areaName: string) => void;
  config: MapConfiguration;
}

export const InteractiveMap: React.FC<InteractiveMapProps> = ({
  geoData,
  onAreaClick,
  onAreaHover,
  config,
}) => {
  const mapRef = useRef<L.Map>(null);
  const mapContainerRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (!mapContainerRef.current) return;

    const map = L.map(mapContainerRef.current, {
      center: [35.6762, 139.6503],
      zoom: 6,
      zoomControl: true,
    });

    // ベースマップの追加
    L.tileLayer(config.basemap.url, {
      attribution: config.basemap.attribution,
    }).addTo(map);

    // 地理データの追加
    const geoJsonLayer = L.geoJSON(geoData, {
      style: {
        fillColor: "#3388ff",
        weight: 2,
        opacity: 1,
        color: "white",
        dashArray: "3",
        fillOpacity: 0.2,
      },
      onEachFeature: (feature, layer) => {
        // クリックイベント
        layer.on("click", () => {
          const areaCode = feature.properties.areaCode;
          const areaName = feature.properties.areaName;
          onAreaClick?.(areaCode, areaName);
        });

        // ホバーイベント
        layer.on("mouseover", () => {
          const areaCode = feature.properties.areaCode;
          const areaName = feature.properties.areaName;
          onAreaHover?.(areaCode, areaName);

          layer.setStyle({
            fillOpacity: 0.5,
          });
        });

        layer.on("mouseout", () => {
          layer.setStyle({
            fillOpacity: 0.2,
          });
        });
      },
    }).addTo(map);

    map.fitBounds(geoJsonLayer.getBounds());
    mapRef.current = map;

    return () => {
      map.remove();
    };
  }, [geoData, config, onAreaClick, onAreaHover]);

  return (
    <div
      ref={mapContainerRef}
      className="w-full h-96 rounded-lg border border-gray-200"
    />
  );
};
```

## データフロー

### TopoJSON 直接使用のデータフロー

```
1. GeoshapeService.fetchPrefectureTopology()
   ↓
2. TopoJSONデータを取得（Mock/R2/External API）
   ↓
3. D3.js側でtopojson.feature()を使用してGeoJSONに変換
   ↓
4. 都道府県コード・名前の正規化
   ↓
5. D3.jsで地図を描画
```

## TopoJSON 直接使用のベストプラクティス

### 1. パフォーマンス最適化

- **ファイルサイズの削減**: TopoJSON は GeoJSON より約 80%小さく、ネットワーク転送が高速
- **メモリ効率**: 変換処理を D3.js 側に移譲することで、サーバー側のメモリ使用量を削減
- **キャッシュ効率**: TopoJSON データをそのままキャッシュすることで、ストレージ効率が向上

### 2. エラーハンドリング

```typescript
try {
  const topology = await fetchPrefectureTopology({
    useCache: true,
  });

  // データの妥当性チェック
  if (!topology || topology.type !== "Topology") {
    throw new Error("Invalid TopoJSON format");
  }

  // オブジェクトの存在チェック
  const objectName = Object.keys(topology.objects)[0];
  if (!objectName) {
    throw new Error("TopoJSON objects is empty");
  }

  // 変換と描画処理
  const geojson = topojson.feature(
    topology,
    topology.objects[objectName]
  ) as GeoJSON.FeatureCollection;
} catch (error) {
  console.error("地図データの取得に失敗:", error);
  // エラー状態の表示
}
```

### 3. 都道府県コード・名前の正規化

```typescript
// 都道府県コードと名前を正規化
const normalizedFeatures = geojson.features.map((feature) => {
  const properties = feature.properties || {};

  // 都道府県コードを抽出（5桁形式に正規化）
  const code = properties.N03_007 || properties.prefCode || properties.code;
  const prefCode = code ? `${String(code).padStart(2, "0")}000` : "00000";

  // 都道府県名を抽出
  const prefName =
    properties.N03_001 || properties.prefName || properties.name || "不明";

  return {
    ...feature,
    properties: {
      ...properties,
      prefCode,
      prefName,
    },
  };
});
```

// 地理データの取得
export const fetchGeoData = async (
  date: string,
  resolution: Resolution
): Promise<TopoJSON> => {
  const baseUrl = "https://geoshape.ex.nii.ac.jp/city/topojson";
  const url = `${baseUrl}/${date}/jp_pref${resolution.fileExtension}`;

  const response = await fetch(url);
  if (!response.ok) {
    throw new Error(`Failed to fetch geo data: ${response.statusText}`);
  }

  return response.json();
};

## 色スケール生成

統計値に基づく色スケールの生成方法です。

```typescript
export const generateColorScale = (
  values: number[],
  config: MapConfiguration["colorScale"]
): ((value: number) => string) => {
  const { colors, domain, type } = config;
  const [min, max] = domain;

  if (type === "sequential") {
    return (value: number) => {
      const normalized = (value - min) / (max - min);
      const index = Math.floor(normalized * (colors.length - 1));
      return colors[Math.max(0, Math.min(index, colors.length - 1))];
    };
  }

  // diverging type
  const mid = (min + max) / 2;
  return (value: number) => {
    if (value < mid) {
      const normalized = (value - min) / (mid - min);
      const index = Math.floor(normalized * (colors.length / 2 - 1));
      return colors[Math.max(0, Math.min(index, colors.length / 2 - 1))];
    } else {
      const normalized = (value - mid) / (max - mid);
      const index =
        Math.floor(normalized * (colors.length / 2 - 1)) + colors.length / 2;
      return colors[
        Math.max(colors.length / 2, Math.min(index, colors.length - 1))
      ];
    }
  };
};

// 統計値から自動的に色スケールを生成
export const generateAutoColorScale = (
  values: number[],
  type: "sequential" | "diverging" = "sequential"
): MapConfiguration["colorScale"] => {
  const min = Math.min(...values);
  const max = Math.max(...values);

  const colors =
    type === "sequential"
      ? [
          "#f7fbff",
          "#deebf7",
          "#c6dbef",
          "#9ecae1",
          "#6baed6",
          "#4292c6",
          "#2171b5",
          "#08519c",
          "#08306b",
        ]
      : [
          "#67001f",
          "#b2182b",
          "#d6604d",
          "#f4a582",
          "#fddbc7",
          "#f7f7f7",
          "#d1e5f0",
          "#92c5de",
          "#4393c3",
          "#2166ac",
          "#053061",
        ];

  return {
    type,
    colors,
    domain: [min, max],
  };
};
```

import { useBreakpoint } from "@/hooks/useBreakpoint";

export const ResponsiveMap: React.FC<MapProps> = ({
  geoData,
  values,
  config,
}) => {
  const breakpoint = useBreakpoint();

  const getMapHeight = () => {
    if (breakpoint === "mobile") return 300;
    if (breakpoint === "tablet") return 400;
    return 500;
  };

  const getInitialZoom = () => {
    if (breakpoint === "mobile") return config.responsive.mobileZoom;
    return config.responsive.desktopZoom;
  };

  return (
    <div className="w-full">
      <ChoroplethMap
        geoData={geoData}
        values={values}
        config={{
          ...config,
          responsive: {
            ...config.responsive,
            enabled: true,
          },
        }}
        height={getMapHeight()}
        initialZoom={getInitialZoom()}
      />
    </div>
  );
};

## パフォーマンス最適化

大量の地理データを扱う際の最適化手法です。

```typescript
// 地理データのキャッシュ
export class GeoDataCache {
  private cache = new Map<string, TopoJSON>();
  private readonly maxSize = 10;

  async get(key: string): Promise<TopoJSON | null> {
    return this.cache.get(key) || null;
  }

  set(key: string, data: TopoJSON): void {
    if (this.cache.size >= this.maxSize) {
      const firstKey = this.cache.keys().next().value;
      this.cache.delete(firstKey);
    }
    this.cache.set(key, data);
  }

  generateKey(date: string, resolution: string): string {
    return `${date}-${resolution}`;
  }
}

// 遅延読み込み
export const LazyMap: React.FC<MapProps> = ({ geoData, values, config }) => {
  const [isLoaded, setIsLoaded] = useState(false);

  useEffect(() => {
    const timer = setTimeout(() => {
      setIsLoaded(true);
    }, 100);

    return () => clearTimeout(timer);
  }, []);

  if (!isLoaded) {
    return (
      <div className="w-full h-96 bg-gray-100 rounded-lg flex items-center justify-center">
        <div className="text-gray-500">地図を読み込み中...</div>
      </div>
    );
  }

  return <ChoroplethMap geoData={geoData} values={values} config={config} />;
};
```
