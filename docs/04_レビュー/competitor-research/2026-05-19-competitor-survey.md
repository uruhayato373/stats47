## todo-ran 詳細 (https://todo-ran.com/)

### カテゴリツリー

- **国土・インフラ**
  - 気候: 年間晴れ日数、降水量、雪日数
  - インフラ: 道路延長、上下水道普及率
- **社会・政治**
  - 選挙: 2025 参院比例代表投票率、衆院比例得票率
  - 行政: 公務員数、議員数
- **産業・経済**
  - 物価: レギュラーガソリン価格、灯油価格
  - 産業: 製造業出荷額、農業産出額
- **文化・くらし・健康**
  - 教育: 小学生数、中学校数、高校進学率
  - 住宅: 持ち家率、空き家率
  - 貧困: 相対的貧困率、子育て世帯相対的貧困率
  - 未婚率: 20-65 歳以上 性別別
  - 健康: 狭心症・心筋梗塞死亡者数、食器洗い機普及率
- **娯楽・スポーツ**
  - スポーツ選手輩出: J リーガー出身地、プロ野球選手出身地
  - 高校野球: 甲子園勝利数・勝率
  - 施設: 野球場数、映画館数
- **店舗分布**
  - コンビニ、飲食店、小売店舗数 (店種別)
- **その他**
  - ランキング切り口分類 (東西対立型 / 都市地方型 / 東京突出型 / ダントツ型 / 全国軒数 / 全国人数)

## uub 詳細 (https://uub.jp/pdr/)

### カテゴリツリー (17 大カテゴリ)

- **人口系**: 人口密度、人口変遷、将来人口予測、出生・死亡・婚姻・離婚、高齢者人口、子ども人口、未婚率、平均寿命、100 歳以上人口、在留外国人
- **地理・気象系**: 東西南北端、面積、森林面積、可住地面積、気象データ、台風上陸回数、竜巻発生回数、海岸線、島の数
- **事故・災害系**: 交通事故 (件数・死者数)、山岳遭難、水難、火災発生件数、土砂災害危険箇所、震度別地震回数
- **交通系**: 鉄道路線・駅数、自動車保有台数、道の駅、空港、港湾、高齢者ドライバー
- **観光系**: ホテル・旅館、温泉、展示施設、外国人宿泊者数
- **健康・生活系**: 医療施設、病床数、医師数、持ち家比率、家賃、喫煙率、がん死亡、美肌ランキング、白物家電普及率
- **社会系**: 幸福度、魅力度、ジェンダーギャップ指数、地価、最低賃金、生活保護受給率、刑法犯、自殺率
- **教育系**: 幼稚園～大学、学力調査成績、英語力、暴力行為、不登校児童数、公立学校エアコン設置率
- **商業・経済系**: 郵便局、大規模小売店舗、キャッシュレス支払割合、通販利用額
- **スポーツ・娯楽系**: 駅伝順位、J リーグ、プロ棋士、映画スクリーン数
- **工業系 (20 業種)**: 製造業、食料品、繊維工業、化学工業、鉄鋼業、電気機械器具 ほか
- **農産業系**: 米、麦、馬鈴薯、野菜類、果実類、花き類
- **水産業系**: 漁港数、まぐろ類、かつお、サケ、かに類、貝類、海藻類
- **資源・エネルギー系**: 発電所、電力消費量、原子力発電所、上水道、都市ガス、ごみリサイクル率
- **通信・IT 系**: 通信機器保有率、インターネット利用率、NHK 受信料支払率
- **政治・行政系**: 投票率、衆参議員、都道府県議会議員、女性首長・議員比率、マイナカード普及率、ふるさと納税
- **番外編**: 落書き帳メンバー居住地分布

## stats47 拡充候補抽出のためのメモ

### 競合に多いが stats47 で薄そうなカテゴリ
1. **工業細目** (uub 20 業種): 製造業出荷額の業種別 (食料品 / 繊維 / 化学 / 鉄鋼 / 電気機械 等) → 工業統計調査 / 経済センサスから取得可
2. **水産魚種別** (uub): まぐろ / かつお / サケ / かに / 貝 / 海藻 → 海面漁業生産統計から取得可
3. **農業品目別** (todo-ran / uub): 米 / 麦 / 馬鈴薯 / 野菜類 / 果実類 / 花き → 作物統計から取得可
4. **災害・事故** (uub): 山岳遭難、水難、火災発生件数、震度別地震回数 → 警察庁・消防庁・気象庁
5. **歴史変遷** (uub): 人口変遷 (長期)、市制施行年、合併履歴 → e-Stat 長期時系列
6. **店舗分布** (todo-ran): コンビニ・チェーン店舗数 → 経済センサス、企業 IR データ
7. **教育詳細**: 公立学校エアコン設置率、不登校児童数、英語力 → 学校保健統計、文科省調査
8. **政治参加詳細**: 衆参議員数、女性首長・議員比率、マイナカード普及率、ふるさと納税受入額 → 総務省

### e-Stat で取得可能性が高そうな指標タイプ

- 経済センサス活動調査の産業中分類別事業所数・従業者数
- 国勢調査の世帯類型別・年齢階級別人口

- 患者調査 (疾患別受療率)
- 社会生活基本統計 (生活時間・趣味活動)
