# タイポグラフィシステムガイド

## 1. 概要

### 1.1 フォントファミリー

stats47 プロジェクトでは、**Geist** フォントファミリーをメインフォントとして使用しています。

**フォントスタック:**

```css
font-family: var(--font-geist-sans), "Hiragino Sans",
  "Hiragino Kaku Gothic ProN", "Yu Gothic", "YuGothic", "Meiryo", sans-serif;
```

**モノスペースフォント:**

```css
font-family: var(--font-geist-mono), "SF Mono", "Monaco", "Inconsolata",
  "Roboto Mono", monospace;
```

### 1.2 フォント選択の理由

**Geist フォントの特徴:**

- **モダンなデザイン**: 現代的で洗練された外観
- **優れた可読性**: 小さいサイズでも読みやすい
- **多言語対応**: 日本語を含む多言語サポート
- **Web 最適化**: Web フォントとして最適化されている
- **軽量**: ファイルサイズが小さく、パフォーマンスに優しい

**日本語フォントフォールバック:**

- **Hiragino Sans**: macOS の標準日本語フォント
- **Yu Gothic**: Windows の標準日本語フォント
- **Meiryo**: 古い Windows システム用

### 1.3 日本語フォント対応

**日本語表示の最適化:**

- システムフォントを優先して使用
- Web フォントの読み込み時間を最小化
- 環境に応じた最適なフォントを自動選択

## 2. フォント設定

### 2.1 Geist フォントの使用

**Next.js での設定:**

```tsx
// src/app/layout.tsx
import { GeistSans } from "geist/font/sans";
import { GeistMono } from "geist/font/mono";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ja" className={`${GeistSans.variable} ${GeistMono.variable}`}>
      <body className={GeistSans.className}>{children}</body>
    </html>
  );
}
```

**CSS 変数の定義:**

```css
/* globals.css */
@theme inline {
  --font-sans: var(--font-geist-sans);
  --font-mono: var(--font-geist-mono);
}
```

### 2.2 システムフォントフォールバック

**フォールバック戦略:**

1. **Geist Sans** - メインフォント（Web フォント）
2. **Hiragino Sans** - macOS システムフォント
3. **Yu Gothic** - Windows システムフォント
4. **Meiryo** - 古い Windows システムフォント
5. **sans-serif** - 汎用サンセリフフォント

### 2.3 Web フォントの読み込み

**パフォーマンス最適化:**

- **自動最適化**: Next.js が自動的にフォントを最適化
- **プリロード**: 重要なフォントを事前読み込み
- **フォールバック**: 読み込み中の表示を最適化

## 3. タイポグラフィスケール

### 3.1 フォントサイズ階層

**Tailwind CSS のフォントサイズ:**

```tsx
// キャプション、補助テキスト
<span className="text-xs">12px - キャプション</span>

// 小さなボタン、ラベル
<span className="text-sm">14px - ラベル</span>

// 本文テキスト（デフォルト）
<span className="text-base">16px - 本文</span>

// 強調テキスト
<span className="text-lg">18px - 強調</span>

// 小見出し
<span className="text-xl">20px - 小見出し</span>

// サブヘッディング
<span className="text-2xl">24px - サブヘッディング</span>

// ページタイトル
<span className="text-3xl">30px - ページタイトル</span>

// 大見出し
<span className="text-4xl">36px - 大見出し</span>
```

**カスタムサイズ:**

```tsx
// 任意のサイズ
<span className="text-[13px]">13px - カスタムサイズ</span>
<span className="text-[22px]">22px - カスタムサイズ</span>
```

### 3.2 行間（Line Height）

**行間の設定:**

```tsx
// 見出し用（密）
<h1 className="text-3xl leading-tight">密な行間 (1.25)</h1>

// 小見出し用
<h2 className="text-xl leading-snug">適度な行間 (1.375)</h2>

// 本文用（デフォルト）
<p className="text-base leading-normal">標準行間 (1.5)</p>

// 読みやすい本文用
<p className="text-base leading-relaxed">広い行間 (1.625)</p>
```

**行間の使い分け:**

- **leading-tight**: 見出し、短いテキスト
- **leading-snug**: 小見出し、サブタイトル
- **leading-normal**: 本文、一般的なテキスト
- **leading-relaxed**: 長文、読みやすいテキスト

### 3.3 フォントウェイト

**ウェイトの階層:**

```tsx
// 本文
<span className="font-normal">400 - 本文</span>

// 強調テキスト
<span className="font-medium">500 - 強調</span>

// 小見出し
<span className="font-semibold">600 - 小見出し</span>

// 見出し
<span className="font-bold">700 - 見出し</span>
```

**使用ガイドライン:**

- **font-normal**: 一般的なテキスト、本文
- **font-medium**: 重要な情報、強調したい部分
- **font-semibold**: セクションタイトル、小見出し
- **font-bold**: ページタイトル、大見出し

### 3.4 文字間隔（Letter Spacing）

**文字間隔の調整:**

```tsx
// 標準
<span className="tracking-normal">標準間隔</span>

// 広い間隔（見出し用）
<h1 className="text-3xl tracking-wide">広い間隔</h1>

// 狭い間隔（コンパクト表示用）
<span className="tracking-tight">狭い間隔</span>
```

## 5. 本文テキスト

### 5.1 段落スタイル

**基本の段落:**

```tsx
<p className="text-base leading-normal text-gray-700 dark:text-neutral-300">
  これは基本的な段落テキストです。読みやすさを重視した行間と文字サイズを使用しています。
</p>
```

**読みやすい段落:**

```tsx
<p className="text-base leading-relaxed text-gray-700 dark:text-neutral-300">
  長文の場合は、leading-relaxed を使用して読みやすさを向上させます。
  特に日本語の場合は、適切な行間が重要です。
</p>
```

### 5.2 リストスタイル

**順序なしリスト:**

```tsx

    リストアイテム 1
  </li>
  <li className="flex items-start">
    <span className="mr-2 text-indigo-600 dark:text-indigo-400">•</span>
    リストアイテム 2
```

**順序ありリスト:**

```tsx

      1.
    </span>
```

## 8. アクセシビリティ

### 8.1 最小フォントサイズ

**推奨最小サイズ:**

- **本文**: 16px (`text-base`) 以上
- **キャプション**: 12px (`text-xs`) 以上
- **UI 要素**: 14px (`text-sm`) 以上

**実装例:**

```tsx
// ✅ 良い例: 十分なサイズ
<span className="text-base">読みやすいテキスト</span>

// ❌ 悪い例: 小さすぎる
<span className="text-xs">読みにくいテキスト</span>
```

### 8.2 コントラスト比

**テキストと背景のコントラスト:**

- **通常テキスト**: 4.5:1 以上のコントラスト比
- **大きなテキスト**: 3:1 以上のコントラスト比

**実装例:**

```tsx
// ✅ 良い例: 十分なコントラスト
<p className="text-gray-900 dark:text-neutral-100">高コントラストテキスト</p>

// ❌ 悪い例: コントラスト不足
<p className="text-gray-400 dark:text-neutral-600">低コントラストテキスト</p>
```

### 8.3 読みやすさの確保

**行間の最適化:**

```tsx
// ✅ 良い例: 適切な行間
<p className="text-base leading-normal">読みやすいテキスト</p>

// ❌ 悪い例: 行間が狭すぎる
<p className="text-base leading-tight">読みにくいテキスト</p>
```

**文字間隔の調整:**

```tsx
// 大文字の場合は文字間隔を広げる
<h1 className="text-3xl font-bold tracking-wide">HEADING</h1>
```

## 9. 実装例

### 9.1 見出しコンポーネント

**Heading コンポーネント:**

```tsx
// src/components/atoms/Heading/Heading.tsx
interface HeadingProps {
  level: 1 | 2 | 3 | 4 | 5 | 6;
  children: React.ReactNode;
  className?: string;
}

const headingStyles = {
  1: "text-3xl font-bold leading-tight text-gray-900 dark:text-neutral-100",
  2: "text-2xl font-semibold leading-snug text-gray-800 dark:text-neutral-200",
  3: "text-xl font-medium leading-snug text-gray-700 dark:text-neutral-300",
  4: "text-lg font-medium text-gray-700 dark:text-neutral-300",
  5: "text-base font-medium text-gray-600 dark:text-neutral-400",
  6: "text-sm font-medium text-gray-600 dark:text-neutral-400",
};

export const Heading: React.FC<HeadingProps> = ({
  level,
  children,
  className = "",
}) => {
  const Component = `h${level}` as keyof JSX.IntrinsicElements;

  return (
    <Component className={`${headingStyles[level]} ${className}`}>
      {children}
    </Component>
  );
};
```

### 9.2 本文コンポーネント

**Text コンポーネント:**

```tsx
// src/components/atoms/Text/Text.tsx
interface TextProps {
  variant?: "body" | "caption" | "small";
  children: React.ReactNode;
  className?: string;
}

const textStyles = {
  body: "text-base leading-normal text-gray-700 dark:text-neutral-300",
  caption: "text-sm text-gray-600 dark:text-neutral-400",
  small: "text-xs text-gray-500 dark:text-neutral-500",
};

export const Text: React.FC<TextProps> = ({
  variant = "body",
  children,
  className = "",
}) => {
  return <p className={`${textStyles[variant]} ${className}`}>{children}</p>;
};
```

### 9.3 カスタムテキストコンポーネント

**Typography コンポーネント:**

```tsx
// src/components/atoms/Typography/Typography.tsx
interface TypographyProps {
  variant?: "h1" | "h2" | "h3" | "h4" | "h5" | "h6" | "body" | "caption";
  color?: "primary" | "secondary" | "tertiary" | "muted";
  children: React.ReactNode;
  className?: string;
}

const variantStyles = {
  h1: "text-3xl font-bold leading-tight",
  h2: "text-2xl font-semibold leading-snug",
  h3: "text-xl font-medium leading-snug",
  h4: "text-lg font-medium",
  h5: "text-base font-medium",
  h6: "text-sm font-medium",
  body: "text-base leading-normal",
  caption: "text-sm",
};

const colorStyles = {
  primary: "text-gray-900 dark:text-neutral-100",
  secondary: "text-gray-700 dark:text-neutral-300",
  tertiary: "text-gray-600 dark:text-neutral-400",
  muted: "text-gray-500 dark:text-neutral-500",
};

export const Typography: React.FC<TypographyProps> = ({
  variant = "body",
  color = "secondary",
  children,
  className = "",
}) => {
  const Component = variant.startsWith("h")
    ? (variant as keyof JSX.IntrinsicElements)
    : "p";

  return (
    <Component
      className={`${variantStyles[variant]} ${colorStyles[color]} ${className}`}
    >
      {children}
    </Component>
  );
};
```

## 10. トラブルシューティング

### 10.1 よくある問題

**問題 1: フォントが表示されない**

**原因:** Geist フォントの読み込みエラー

**解決策:**

```tsx
// layout.tsx でフォントを正しくインポート
import { GeistSans } from "geist/font/sans";
import { GeistMono } from "geist/font/mono";

// HTML にクラスを適用
<html className={`${GeistSans.variable} ${GeistMono.variable}`}>
  <body className={GeistSans.className}>
```

**問題 2: 日本語が正しく表示されない**

**原因:** 日本語フォントのフォールバックが不適切

**解決策:**

```css
/* globals.css で日本語フォントを追加 */
font-family: var(--font-geist-sans), "Hiragino Sans", "Yu Gothic", "Meiryo",
  sans-serif;
```

**問題 3: フォントサイズが期待と異なる**

**原因:** Tailwind の設定や CSS の競合

**解決策:**

```tsx
// 明示的にサイズを指定
<span className="text-[16px]">明示的サイズ</span>

// または、より具体的なクラス
<span className="!text-base">強制適用</span>
```

**問題 4: 行間が適切でない**

**原因:** leading クラスの不適切な使用

**解決策:**

```tsx
// 適切な行間を選択
<p className="text-base leading-normal">標準行間</p>
<p className="text-base leading-relaxed">読みやすい行間</p>
```

### 10.2 デバッグ方法

**フォントの確認:**
