# コンポーネント実装ガイド

## 1. ページコンポーネントの実装

### 1.1 カテゴリーページ実装

#### コンポーネント構造

カテゴリーページは以下の構造で実装します：

```tsx
// src/components/pages/CategoryPageClient.tsx
"use client";

import React from "react";
import Link from "next/link";
import { CategoryIcon } from "@/components/atoms/CategoryIcon";
import { getCategoriesForSidebar } from "@/lib/category";

interface CategoryPageClientProps {
  categoryId: string;
}

export default function CategoryPageClient({
  categoryId,
}: CategoryPageClientProps) {
  const categories = getCategoriesForSidebar();
  const category = categories.find((cat) => cat.id === categoryId);

  if (!category) {
    return <div>カテゴリーが見つかりません</div>;
  }

  return (
    <div className="min-h-screen bg-gray-50 dark:bg-neutral-900">
      {/* ページヘッダー */}
      <div className="bg-white dark:bg-neutral-800 border-b border-gray-200 dark:border-neutral-700">
        <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-6">
          {/* パンくずナビゲーション */}
          <nav className="flex items-center space-x-2 text-sm text-gray-500 dark:text-neutral-400 mb-4">
            <Link
              href="/"
              className="hover:text-gray-700 dark:hover:text-neutral-300"
            >
              統計データ
            </Link>
            <span>/</span>
            <span className="text-gray-900 dark:text-neutral-200">
              {category.name}
            </span>
          </nav>

          {/* タイトルとアイコン */}
          <div className="flex items-center space-x-4">
            <CategoryIcon
              categoryId={categoryId}
              className="w-12 h-12 text-gray-600 dark:text-neutral-400"
            />
            <h1 className="text-3xl font-bold text-gray-900 dark:text-neutral-100">
              {category.name}
            </h1>
          </div>
        </div>
      </div>

      {/* サブカテゴリーグリッド */}
      <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-8">
        {category.subcategories && category.subcategories.length > 0 ? (
          <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6">
            {category.subcategories.map((subcategory) => (
              <Link
                key={subcategory.id}
                href={`/${categoryId}/${subcategory.id}`}
                className="group block"
              >
                <div className="bg-white dark:bg-neutral-800 rounded-lg border border-gray-200 dark:border-neutral-700 p-6 hover:shadow-md transition-shadow">
                  <div className="flex items-center space-x-3">
                    <CategoryIcon
                      categoryId={categoryId}
                      className="w-8 h-8 text-gray-600 dark:text-neutral-400"
                    />
                    <h3 className="text-lg font-medium text-gray-900 dark:text-neutral-200 group-hover:text-indigo-600 dark:group-hover:text-indigo-400">
                      {subcategory.name}
                    </h3>
                  </div>
                </div>
              </Link>
            ))}
          </div>
        ) : (
          /* 空状態表示 */
          <div className="text-center py-12">
            <CategoryIcon
              categoryId={categoryId}
              className="w-16 h-16 text-gray-400 dark:text-neutral-500 mx-auto mb-4"
            />
            <p className="text-gray-500 dark:text-neutral-400">
              このカテゴリーにはサブカテゴリーがありません
            </p>
          </div>
        )}
      </div>
    </div>
  );
}
```

#### UI 要素の実装

**ページヘッダー:**

- カテゴリーアイコン: `CategoryIcon` コンポーネントを使用
- パンくずナビゲーション: `Link` コンポーネントで実装
- タイトル: カテゴリー名を大きく表示

**サブカテゴリーグリッド:**

- レスポンシブグリッド: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`
- カードデザイン: ホバー効果付き
- リンク: Next.js の `Link` コンポーネント

**空状態表示:**

- 大きなアイコン
- 説明メッセージ

```tsx
// src/components/subcategories/index.tsx
export const getDashboardComponentByArea = (
  subcategoryId: string,
  areaCode: string,
  categoryId?: string
): React.ComponentType<SubcategoryDashboardPageProps> => {
  const subcategory = getSubcategoryInfo(subcategoryId, categoryId);

  if (!subcategory) {
    return DefaultDashboardPage;
  }

  const ComponentMap = subcategory.components;

  if (areaCode === "00000") {
    return ComponentMap.national || DefaultDashboardPage;
  } else {
    return ComponentMap.prefecture || DefaultDashboardPage;
  }
};
```

```tsx
// src/components/atoms/Button/Button.tsx
interface ButtonProps {
  variant?: "primary" | "secondary" | "outline" | "ghost";
  size?: "sm" | "md" | "lg";
  disabled?: boolean;
  loading?: boolean;
  children: React.ReactNode;
  onClick?: () => void;
  className?: string;
}

export const Button: React.FC<ButtonProps> = ({
  variant = "primary",
  size = "md",
  disabled = false,
  loading = false,
  children,
  onClick,
  className = "",
}) => {
  const baseClasses =
    "inline-flex items-center justify-center font-medium rounded-lg transition-colors focus:outline-none focus:ring-2 focus:ring-offset-2";

  const variantClasses = {
    primary:
      "bg-indigo-600 text-white hover:bg-indigo-700 focus:ring-indigo-500 disabled:bg-indigo-300",
    secondary:
      "bg-gray-600 text-white hover:bg-gray-700 focus:ring-gray-500 disabled:bg-gray-300",
    outline:
      "border border-gray-300 bg-white text-gray-700 hover:bg-gray-50 focus:ring-indigo-500 dark:border-neutral-600 dark:bg-neutral-800 dark:text-neutral-300 dark:hover:bg-neutral-700",
    ghost:
      "text-gray-700 hover:bg-gray-100 focus:ring-indigo-500 dark:text-neutral-300 dark:hover:bg-neutral-700",
  };

  const sizeClasses = {
    sm: "px-3 py-1.5 text-sm",
    md: "px-4 py-2 text-sm",
    lg: "px-6 py-3 text-base",
  };

  return (
    <button
      className={`${baseClasses} ${variantClasses[variant]} ${sizeClasses[size]} ${className}`}
      disabled={disabled || loading}
      onClick={onClick}
    >
      {loading && (
        <svg
          className="animate-spin -ml-1 mr-2 h-4 w-4"
          fill="none"
          viewBox="0 0 24 24"
        >
          <circle
            className="opacity-25"

            cy="12"

            stroke="currentColor"

          />
        </svg>
```

```tsx
// src/components/organisms/visualization/ChoroplethMap.tsx
interface ChoroplethMapProps {
  data: ChoroplethData[];
  title?: string;
  height?: number;
}

export const ChoroplethMap: React.FC<ChoroplethMapProps> = ({
  data,
  title,
  height = 400,
}) => {
  const svgRef = useRef<SVGSVGElement>(null);

  useEffect(() => {
    if (!svgRef.current || !data.length) return;

    const svg = select(svgRef.current);
    svg.selectAll("*").remove();

    // 地図の描画処理
    const width = svgRef.current.clientWidth;
    const projection = geoMercator().fitSize([width, height], japanTopology);

    const path = geoPath().projection(projection);

    // 地図の描画
    svg
      .selectAll("path")
      .data(
        topojson.feature(japanTopology, japanTopology.objects.japan).features
      )
      .enter()
      .append("path")
      .attr("d", path)
      .attr("fill", (d) => {
        const areaData = data.find(
          (item) => item.areaCode === d.properties.code
        );
        return areaData ? areaData.color : "#e5e7eb";
      })
      .attr("stroke", "#ffffff")
      .attr("stroke-width", 1)
      .on("mouseover", (event, d) => {
        const areaData = data.find(
          (item) => item.areaCode === d.properties.code
        );
        if (areaData) {
          // ツールチップの表示
          showTooltip(event, areaData);
        }
      })
      .on("mouseout", () => {
        hideTooltip();
      });
  }, [data, height]);

  return (
    <div className="bg-white dark:bg-neutral-800 rounded-lg border border-gray-200 dark:border-neutral-700 p-4">
      {title && (
        <h3 className="text-lg font-medium text-gray-900 dark:text-neutral-200 mb-4">
```

### 6.7 アクセシビリティのベストプラクティス

**装飾的なアイコン:**

```tsx
// 装飾目的のみのアイコン
<Search className="w-4 h-4" aria-hidden="true" />
```

**意味のあるアイコン:**

```tsx
// 情報を伝えるアイコン
<AlertCircle className="w-4 h-4" role="img" aria-label="エラーが発生しました" />
```

**ボタン内のアイコン:**

```tsx
// ボタンにaria-labelを設定
<button aria-label="検索">
  <Search className="w-4 h-4" aria-hidden="true" />
</button>

// または、アイコンとテキストの両方がある場合
<button>
  <Search className="w-4 h-4" aria-hidden="true" />
  検索
</button>
```

### 6.8 パフォーマンス最適化

**Tree-shaking の活用:**

```tsx
// ✅ 良い例: 必要なアイコンのみインポート
import { Search, RefreshCw, Download } from "lucide-react";

// ❌ 悪い例: 全てをインポート
import * as Icons from "lucide-react";
```

**動的インポート:**

```tsx
// 大量のアイコンを使用する場合
const IconComponent = lazy(() =>
  import("lucide-react").then((module) => ({
    default: module[iconName],
  }))
);
```
