# チャート実装ガイド

## 概要

Rechartsライブラリを使用したチャートコンポーネントの実装方法について説明します。統計データの可視化に必要な基本的なパターンとベストプラクティスを提供します。

## 基本コンポーネント

### 1. LineChart（折れ線グラフ）

時系列データの表示に使用します。

```typescript
import { LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, Legend, ResponsiveContainer } from 'recharts';

interface LineChartProps {
  data: Array<{ name: string; value: number; [key: string]: any }>;
  dataKey: string;
  color?: string;
  height?: number;
}

export const CustomLineChart: React.FC<LineChartProps> = ({
  data,
  dataKey,
  color = '#8884d8',
  height = 400
}) => {
  return (
    <ResponsiveContainer width="100%" height={height}>
      <LineChart data={data}>
        <CartesianGrid strokeDasharray="3 3" />
        <XAxis dataKey="name" />
        <YAxis />
        <Tooltip />
        <Legend />
        <Line 
          type="monotone" 
          dataKey={dataKey} 
          stroke={color}
          strokeWidth={2}
          dot={{ fill: color, strokeWidth: 2, r: 4 }}
        />
      </LineChart>
    </ResponsiveContainer>
  );
};
```

### 3. PieChart（円グラフ）

構成比の表示に使用します。

```typescript
import { PieChart, Pie, Cell, ResponsiveContainer, Tooltip, Legend } from 'recharts';

interface PieChartProps {
  data: Array<{ name: string; value: number; color?: string }>;
  height?: number;
}

const COLORS = ['#0088FE', '#00C49F', '#FFBB28', '#FF8042', '#8884d8'];

export const CustomPieChart: React.FC<PieChartProps> = ({
  data,
  height = 400
}) => {
  return (
    <ResponsiveContainer width="100%" height={height}>
      <PieChart>
        <Pie
          data={data}
          cx="50%"
          cy="50%"
          labelLine={false}
          label={({ name, percent }) => `${name} ${(percent * 100).toFixed(0)}%`}
          outerRadius={80}
          fill="#8884d8"
          dataKey="value"
        >
          {data.map((entry, index) => (
            <Cell key={`cell-${index}`} fill={entry.color || COLORS[index % COLORS.length]} />
          ))}
        </Pie>
        <Tooltip />
        <Legend />
      </PieChart>
    </ResponsiveContainer>
  );
};
```

## レスポンシブ対応

デバイスサイズに応じたチャート表示の実装方法です。

```typescript
import { useBreakpoint } from '@/hooks/useBreakpoint';

export const ResponsiveChart: React.FC<ChartProps> = ({ data, config }) => {
  const breakpoint = useBreakpoint();
  
  const getChartHeight = () => {
    if (breakpoint === 'mobile') return 250;
    if (breakpoint === 'tablet') return 350;
    return config.displayOptions.height || 400;
  };

  const getFontSize = () => {
    if (breakpoint === 'mobile') return 12;
    if (breakpoint === 'tablet') return 14;
    return 16;
  };

  return (
    <div className="w-full">
      <ResponsiveContainer width="100%" height={getChartHeight()}>
        <LineChart data={data}>
          <XAxis 
            dataKey="name" 
            fontSize={getFontSize()}
            tick={{ fontSize: getFontSize() }}
          />
          <YAxis 
            fontSize={getFontSize()}
            tick={{ fontSize: getFontSize() }}
          />
          <Tooltip 
            contentStyle={{ 
              fontSize: getFontSize(),
              borderRadius: '8px'
            }}
          />
          <Legend 
            wrapperStyle={{ fontSize: getFontSize() }}
          />
          <Line type="monotone" dataKey="value" stroke="#8884d8" />
        </LineChart>
      </ResponsiveContainer>
    </div>
  );
};
```

## 色スケールとテーマ設定

プロジェクト全体で統一された色スケールの実装です。

```typescript
export const colorSchemes = {
  sequential: {
    blue: ['#f7fbff', '#deebf7', '#c6dbef', '#9ecae1', '#6baed6', '#4292c6', '#2171b5', '#08519c', '#08306b'],
    green: ['#f7fcf5', '#e5f5e0', '#c7e9c0', '#a1d99b', '#74c476', '#41ab5d', '#238b45', '#006d2c', '#00441b'],
    red: ['#fff5f0', '#fee0d2', '#fcbba1', '#fc9272', '#fb6a4a', '#ef3b2c', '#cb181d', '#a50f15', '#67000d']
  },
  diverging: {
    redBlue: ['#67001f', '#b2182b', '#d6604d', '#f4a582', '#fddbc7', '#f7f7f7', '#d1e5f0', '#92c5de', '#4393c3', '#2166ac', '#053061'],
    redGreen: ['#8e0152', '#c51b7d', '#de77ae', '#f1b6da', '#fde0ef', '#f7f7f7', '#e6f5d0', '#b8e186', '#7fbc41', '#4d9221', '#276419']
  }
};

export const getColorScale = (values: number[], scheme: keyof typeof colorSchemes.sequential = 'blue') => {
  const colors = colorSchemes.sequential[scheme];
  const min = Math.min(...values);
  const max = Math.max(...values);
  
  return (value: number) => {
    const normalized = (value - min) / (max - min);
    const index = Math.floor(normalized * (colors.length - 1));
    return colors[index];
  };
};
```

## データ変換ユーティリティ

e-Stat APIのデータをRecharts用フォーマットに変換するユーティリティです。

```typescript
export interface EstatDataPoint {
  areaCode: string;
  areaName: string;
  value: number;
  year?: number;
}

export const transformEstatDataForChart = (
  data: EstatDataPoint[],
  type: 'timeSeries' | 'ranking' = 'timeSeries'
): Array<{ name: string; value: number; [key: string]: any }> => {
  if (type === 'timeSeries') {
    return data.map(point => ({
      name: point.year?.toString() || point.areaName,
      value: point.value,
      areaCode: point.areaCode,
      areaName: point.areaName
    }));
  }
  
  // ranking type
  return data.map(point => ({
    name: point.areaName,
    value: point.value,
    areaCode: point.areaCode
  }));
};

export const groupDataByArea = (
  data: EstatDataPoint[]
): Record<string, Array<{ name: string; value: number }>> => {
  const grouped: Record<string, Array<{ name: string; value: number }>> = {};
  
  data.forEach(point => {
    if (!grouped[point.areaCode]) {
      grouped[point.areaCode] = [];
    }
    grouped[point.areaCode].push({
      name: point.year?.toString() || point.areaName,
      value: point.value
    });
  });
  
  return grouped;
};
```

## エラーハンドリング

チャート表示時のエラー処理パターンです。

```typescript
interface ChartWithErrorHandlingProps extends ChartProps {
  error?: Error | null;
  loading?: boolean;
}

export const ChartWithErrorHandling: React.FC<ChartWithErrorHandlingProps> = ({
  data,
  config,
  error,
  loading
}) => {
  if (loading) {
    return (
      <div className="flex items-center justify-center h-96">
        <div className="animate-spin rounded-full h-8 w-8 border-b-2 border-blue-600"></div>
        <span className="ml-2 text-gray-600">データを読み込み中...</span>
      </div>
    );
  }

  if (error) {
    return (
      <div className="flex items-center justify-center h-96 bg-red-50 rounded-lg">
        <div className="text-center">
          <div className="text-red-600 text-lg font-semibold mb-2">
            データの読み込みに失敗しました
          </div>
          <div className="text-red-500 text-sm">
            {error.message}
          </div>
        </div>
      </div>
    );
  }

  if (!data || data.length === 0) {
    return (
      <div className="flex items-center justify-center h-96 bg-gray-50 rounded-lg">
        <div className="text-center text-gray-600">
          表示するデータがありません
        </div>
      </div>
    );
  }

  return <CustomLineChart data={data} config={config} />;
};
```

## パフォーマンス最適化

大量データのチャート表示時の最適化手法です。

```typescript
import { useMemo } from 'react';

export const OptimizedChart: React.FC<ChartProps> = ({ data, config }) => {
  // データのメモ化
  const processedData = useMemo(() => {
    if (!data || data.length === 0) return [];
    
    // データサンプリング（1000件を超える場合）
    if (data.length > 1000) {
      const step = Math.ceil(data.length / 1000);
      return data.filter((_, index) => index % step === 0);
    }
    
    return data;
  }, [data]);

  // チャート設定のメモ化
  const chartConfig = useMemo(() => ({
    ...config,
    displayOptions: {
      ...config.displayOptions,
      // 大量データ時はアニメーションを無効化
      animation: processedData.length < 100
    }
  }), [config, processedData.length]);

  return <CustomLineChart data={processedData} config={chartConfig} />;
};
```
