# Export（エクスポートドメイン）

## 概要

データのエクスポート機能を担当する汎用ドメイン。統計データや分析結果を様々な形式で出力します。

## 責務

- データのエクスポート
- エクスポート形式の管理
- エクスポート設定の管理
- エクスポート履歴の管理
- エクスポート権限の管理

## ユビキタス言語

- **エクスポート**: データの外部出力
- **エクスポート形式**: CSV、Excel、PDF 等の出力形式
- **エクスポート設定**: 出力時の設定項目
- **エクスポート履歴**: 過去のエクスポート記録

## 主要エンティティ

- ExportJob（エクスポートジョブ）
- ExportFormat（エクスポート形式）
- ExportHistory（エクスポート履歴）
