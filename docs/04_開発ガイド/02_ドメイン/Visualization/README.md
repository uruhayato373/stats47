# Visualization（可視化ドメイン）

## 概要

統計データの可視化を担当するコアドメイン。ダッシュボード、グラフ、地図などの視覚的表現を提供します。

## 責務

- 統計データの視覚的表現
- インタラクティブな可視化
- レスポンシブな表示
- アクセシビリティ対応
- データソース表記

## ユビキタス言語

- **ダッシュボード**: 統計データの総合的な表示画面
- **グラフ**: 統計データの図表表現
- **地図**: 地理的な統計データの可視化
- **コロプレス地図**: 地域別の統計値を色分けで表示
- **チャート**: 各種グラフの総称

## サブドメイン

### ダッシュボード

統計データの総合的な表示画面

### グラフ

統計データの図表表現

- 棒グラフ
- 折れ線グラフ
- 円グラフ
- 散布図
- ヒートマップ

### 地図

地理的な統計データの可視化

- コロプレス地図
- ベースマップ
- インタラクティブ地図

## 主要エンティティ

- Dashboard（ダッシュボード）
- Chart（チャート）
- Map（地図）
- VisualizationConfig（可視化設定）
