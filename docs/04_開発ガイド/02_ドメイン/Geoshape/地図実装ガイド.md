# Geoshape ドメイン - 地図実装ガイド

## 概要

このガイドでは、Geoshape ドメインを使用して地図機能を実装する方法を説明します。D3.js を使用した都道府県地図の実装例を中心に、カスタマイズ方法やパフォーマンス最適化についても解説します。

## 基本的な使用方法

### 1. 基本的な地図表示

```tsx

```

## データの取得と管理

### 1. 直接データを取得

```tsx
import { GeoshapeService } from "@/features/gis/geoshape/services/geoshape-service";

async function loadMapData() {
  try {
    // 都道府県データを取得
    const features = await GeoshapeService.getPrefectureFeatures({
      resolution: "low",
      useCache: true,
    });

    console.log("取得した都道府県数:", features.features.length);

    // 特定の都道府県を取得
    const tokyo = await GeoshapeService.getPrefectureFeature("13000");
    if (tokyo) {
      console.log("東京都:", tokyo.properties.prefName);
    }
  } catch (error) {
    console.error("データ取得エラー:", error);
  }
}
```

### 2. データソースの可用性チェック

```tsx
import { GeoshapeService } from "@/features/gis/geoshape/services/geoshape-service";

async function checkDataSources() {
  const sources = await GeoshapeService.checkDataSources();

  console.log("Mock利用可能:", sources.mock);
  console.log("R2利用可能:", sources.r2);
  console.log("外部API利用可能:", sources.external);
}
```

## カスタマイズ例

### 1. ランキングデータとの連携

```tsx
import { PrefectureMap } from "@/features/visualization/map/common/PrefectureMap";

function RankingMap({ rankingData }) {
  const getPrefectureColor = (prefCode) => {
    const data = rankingData.find((d) => d.prefCode === prefCode);
    if (!data) return "#e0e0e0";

    // ランキング値に基づいて色を決定
    const normalized =
      data.value / Math.max(...rankingData.map((d) => d.value));
    return d3.interpolateBlues(normalized);
  };

  return (
    <PrefectureMap
      width={800}
      height={600}
      fillColor={(feature) => getPrefectureColor(feature.properties.prefCode)}
    />
  );
}
```

### 2. カスタムツールチップ

```tsx
import { useState } from "react";
import { PrefectureMap } from "@/features/visualization/map/common/PrefectureMap";

function MapWithTooltip() {
  const [tooltip, setTooltip] = useState(null);

  return (
    <div className="relative">
      <PrefectureMap
        width={800}
        height={600}
        onPrefectureHover={(feature) => {
          if (feature) {
            setTooltip({
              x: event.clientX,
              y: event.clientY,
              content: feature.properties.prefName,
            });
          } else {
            setTooltip(null);
          }
        }}
      />

      {tooltip && (
        <div
          className="absolute bg-black text-white px-2 py-1 rounded text-sm pointer-events-none z-10"
          style={{
            left: tooltip.x + 10,
            top: tooltip.y - 10,
          }}
        >
          {tooltip.content}
        </div>
      )}
    </div>
  );
}
```

## パフォーマンス最適化

### 1. メモリキャッシュの活用

```tsx
// 同じデータを複数回取得する場合、キャッシュが効く
const features1 = await GeoshapeService.getPrefectureFeatures({
  useCache: true,
});
const features2 = await GeoshapeService.getPrefectureFeatures({
  useCache: true,
});
// features2はキャッシュから取得される
```

### 2. 必要に応じたデータ取得

```tsx
// 特定の都道府県のみ必要な場合
const tokyo = await GeoshapeService.getPrefectureFeature("13000", {
  resolution: "low", // 低解像度で十分な場合
  useCache: true,
});
```

### 3. キャッシュのクリア

```tsx
import { GeoshapeService } from "@/features/gis/geoshape/services/geoshape-service";

// メモリキャッシュをクリア（必要に応じて）
GeoshapeService.clearCache();
```

## トラブルシューティング

### 1. 地図が表示されない

**原因**: データの取得に失敗している可能性があります。

**解決方法**:

```tsx
// エラーハンドリングを追加
<PrefectureMap
  width={800}
  height={600}
  onError={(error) => {
    console.error("地図エラー:", error);
    // エラー表示やフォールバック処理
  }}
/>
```

### 2. パフォーマンスが悪い

**原因**: 高解像度データの使用やキャッシュの未活用。

**解決方法**:

- 低解像度データの使用
- メモリキャッシュの活用
- 不要なアニメーションの無効化

```tsx
<PrefectureMap
  width={800}
  height={600}
  enableAnimation={false} // アニメーション無効
  // 低解像度データは自動的に選択される
/>
```

### 3. R2 ストレージエラー

**原因**: R2 ストレージの設定が不完全。

**解決方法**:

- 環境変数の確認
- R2 ストレージの可用性チェック

```tsx
const sources = await GeoshapeService.checkDataSources();
if (!sources.r2) {
  console.warn("R2ストレージが利用できません");
}
```

## ベストプラクティス

### 1. エラーハンドリング

```tsx
function SafeMapComponent() {
  const [error, setError] = useState(null);
  const [isLoading, setIsLoading] = useState(true);

  const handleError = (error) => {
    setError(error.message);
    setIsLoading(false);
  };

  if (error) {
    return <div>エラー: {error}</div>;
  }

  return (
    <PrefectureMap
      width={800}
      height={600}
      onError={handleError}
      onLoad={() => setIsLoading(false)}
    />
  );
}
```

### 2. レスポンシブデザイン

```tsx
function ResponsiveMap() {
  const [dimensions, setDimensions] = useState({ width: 800, height: 600 });

  useEffect(() => {
    const updateDimensions = () => {
      const container = document.getElementById("map-container");
      if (container) {
        setDimensions({
          width: container.offsetWidth,
          height: Math.min(container.offsetWidth * 0.75, 600),
        });
      }
    };

    updateDimensions();
    window.addEventListener("resize", updateDimensions);
    return () => window.removeEventListener("resize", updateDimensions);
  }, []);

  return (
    <div id="map-container" className="w-full">
      <PrefectureMap width={dimensions.width} height={dimensions.height} />
    </div>
  );
}
```
