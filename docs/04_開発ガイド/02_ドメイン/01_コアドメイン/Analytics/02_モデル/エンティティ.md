# Analytics ドメイン - エンティティ

## 主要エンティティ

### RankingItem（ランキング項目）

ランキングの対象となる統計指標を表すエンティティ。

```typescript
class RankingItem {
  private constructor(
    private readonly rankingKey: RankingKey,
    private readonly label: string,
    private readonly unit: string,
    private readonly dataSource: DataSource,
    private readonly category: Category,
    private readonly subcategory: Subcategory
  ) {}

  getRankingKey(): RankingKey {
    return this.rankingKey;
  }

  getLabel(): string {
    return this.label;
  }

  getUnit(): string {
    return this.unit;
  }

  getDataSource(): DataSource {
    return this.dataSource;
  }

  getCategory(): Category {
    return this.category;
  }

  getSubcategory(): Subcategory {
    return this.subcategory;
  }
}
```

### RankingValue（ランキング値）

特定地域の特定ランキング項目の値を表すエンティティ。


### Comparison（比較データ）

複数地域間の比較分析結果を表すエンティティ。

```typescript
class Comparison {
  private constructor(
    private readonly comparisonId: ComparisonId,
    private readonly baseAreaCode: AreaCode,
    private readonly comparisonAreaCodes: AreaCode[],
    private readonly rankingKey: RankingKey,
    private readonly year: Year,
    private readonly differences: Map<AreaCode, number>,
    private readonly ratios: Map<AreaCode, number>
  ) {}

  getComparisonId(): ComparisonId {
    return this.comparisonId;
  }

  getBaseAreaCode(): AreaCode {
    return this.baseAreaCode;
  }

  getComparisonAreaCodes(): AreaCode[] {
    return this.comparisonAreaCodes;
  }

  getRankingKey(): RankingKey {
    return this.rankingKey;
  }

  getYear(): Year {
    return this.year;
  }

  getDifference(areaCode: AreaCode): number {
    return this.differences.get(areaCode) || 0;
  }

  getRatio(areaCode: AreaCode): number {
    return this.ratios.get(areaCode) || 0;
  }
}
```

## エンティティの関係性

- **RankingItem** は **RankingValue** の親エンティティ
- **RankingValue** は **StatisticalAnalysis** の構成要素
- **RankingValue** は **Comparison** の構成要素
- **RankingValue** は **TimeSeriesAnalysis** の構成要素
- **RankingValue** は **CorrelationAnalysis** の構成要素

## ビジネスルール

1. **RankingValue** の値は負の値も許可される
2. **RankingValue** の順位は 1 から始まる
3. **StatisticalAnalysis** は最低 3 つのデータポイントが必要
4. **Comparison** は最低 2 つの地域が必要
5. **TimeSeriesAnalysis** は最低 2 年のデータが必要
6. **CorrelationAnalysis** は最低 10 のデータポイントが必要
