# Analytics（統計分析ドメイン）

## 概要

統計データの分析・計算を担当するコアドメイン。ランキング生成、比較分析、時系列分析、相関分析などの統計的な処理を提供します。

## 責務

- ランキング計算
- 比較分析
- 傾向分析
- 統計サマリー生成
- データ品質評価

## ユビキタス言語

- **ランキング**: 統計値に基づく地域の順位付け
- **比較分析**: 複数地域間の統計値の比較
- **時系列分析**: 時間軸での統計値の変化分析
- **相関分析**: 統計値間の相関関係の分析

## サブドメイン

### ランキング

都道府県・市区町村のランキング生成機能

### 比較分析

地域間の統計値の比較分析機能（Phase 2）

### 時系列分析

統計値の時系列データ分析機能（Phase 2）

- CAGR（年平均成長率）計算
- トレンドライン生成
- 前年比・前期比計算

### 相関分析

統計値間の相関関係分析機能（Phase 2）

- ピアソン相関係数
- 散布図生成
- 回帰分析

## 主要エンティティ

- RankingItem（ランキング項目）
- RankingValue（ランキング値）
- StatisticalAnalysis（統計分析結果）
- Comparison（比較データ）
