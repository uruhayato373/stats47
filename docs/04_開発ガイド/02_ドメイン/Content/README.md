# Content（コンテンツ管理ドメイン）

## 概要

コンテンツの管理を担当する汎用ドメイン。ブログ記事やその他のコンテンツを管理します。

## 責務

- コンテンツの作成・編集
- コンテンツの公開・非公開
- コンテンツの分類・タグ付け
- コンテンツの検索
- コンテンツのバージョン管理

## ユビキタス言語

- **コンテンツ**: ブログ記事などの情報
- **記事**: ブログの個別記事
- **タグ**: コンテンツの分類ラベル
- **カテゴリ**: コンテンツの大分類

## サブドメイン

### Blog

ブログ記事の管理機能

## 主要エンティティ

- Content（コンテンツ）
- Article（記事）
- Tag（タグ）
