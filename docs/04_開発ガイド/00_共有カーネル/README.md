# 共有カーネル

## 概要

複数のドメインで共通して使用される要素を管理する共有カーネル。ドメイン間の重複を避け、一貫性を保ちます。

## 責務

- 共通の値オブジェクト定義
- 共通の型定義
- 共通のユーティリティ関数
- 共通の定数定義
- 共通の例外定義

## ユビキタス言語

- **値オブジェクト**: 不変の値表現
- **型定義**: 共通の型情報
- **ユーティリティ**: 共通のヘルパー関数
- **定数**: 共通の定数値

## 主要な要素

### 値オブジェクト

- ID（識別子）
- Date（日付）
- Money（金額）
- Email（メールアドレス）

### 型定義

- Result（結果型）
- Option（オプション型）
- Either（Either 型）

### ユーティリティ

- Validation（検証）
- Formatting（フォーマット）
- Conversion（変換）
