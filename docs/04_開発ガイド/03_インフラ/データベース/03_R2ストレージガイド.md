# R2 ストレージガイド

## 概要

stats47 プロジェクトにおける Cloudflare R2 ストレージの設計と実装について説明します。

### R2 ストレージとは

Cloudflare R2 は、AWS S3 互換のオブジェクトストレージサービスです。

- **S3 互換 API**: AWS SDK を使用してアクセス可能
- **グローバル CDN**: 世界中で高速アクセス
- **コスト効率**: エグレス料金なし
- **セキュリティ**: エンタープライズグレードのセキュリティ

### D1 との使い分け

| 用途             | D1                  | R2                  |
| ---------------- | ------------------- | ------------------- |
| **データベース** | ✅ 構造化データ     | ❌                  |
| **メタデータ**   | ✅ 検索・フィルタ用 | ✅ 完全なレスポンス |
| **キャッシュ**   | ✅ 高速アクセス     | ✅ 大容量・長期保存 |
| **ファイル**     | ❌                  | ✅ 画像・JSON・CSV  |
| **バックアップ** | ❌                  | ✅ データアーカイブ |

### ユースケース

1. **e-Stat API レスポンスキャッシュ**: 完全な API レスポンスを JSON 形式で保存
2. **統計データアーカイブ**: 過去の統計データを長期保存
3. **画像・ファイル**: ダッシュボード用の画像やエクスポートファイル
4. **バックアップ**: D1 データベースのバックアップファイル

## セットアップ

#### 環境別バケット構成

```bash

wrangler r2 bucket create stats47-development

# ステージング用バケット
wrangler r2 bucket create stats47-staging

# 本番用バケット
wrangler r2 bucket create stats47
```

```toml

binding = "METAINFO_BUCKET"
bucket_name = "stats47"
preview_bucket_name = "stats47-development"

# ステージング環境
[env.staging]
NODE_ENV = "production"

[[env.staging.r2_buckets]]
binding = "METAINFO_BUCKET"
bucket_name = "stats47-staging"
```

### アクセス許可の設定

#### Cloudflare R2 トークンの作成

1. **Cloudflare ダッシュボードにログイン**
2. **R2 Object Storage** → **Manage R2 API tokens**
3. **Create API token**をクリック
4. 以下の設定でトークンを作成：
   - **Token name**: `stats47-development`
   - **Permissions**: `Object Read & Write`
   - **Bucket**: `stats47-development`（開発用バケット）

#### 環境変数の設定

**`.env.development`**:

```bash
# R2 S3互換API設定
CLOUDFLARE_ACCOUNT_ID=your_account_id
CLOUDFLARE_R2_ACCESS_KEY_ID=your_access_key_id
CLOUDFLARE_R2_SECRET_ACCESS_KEY=your_secret_access_key
CLOUDFLARE_R2_BUCKET_NAME=stats47-development
```

**本番環境**:


## 実装パターン

### R2 への保存実装

#### 基本的な保存パターン

```typescript
export class EstatMetaInfoR2Repository {
  static async saveMetaInfo(
    env: { METAINFO_BUCKET: R2Bucket },
    statsDataId: string,
    metaInfo: EstatMetaInfoResponse
  ): Promise<{ key: string; size: number }> {
    // サマリー情報を抽出
    const tableInf = metaInfo.GET_META_INFO?.METADATA_INF?.TABLE_INF;
    if (!tableInf) {
      throw new Error("統計表情報が見つかりません");
    }

    // R2保存用データ構造に変換
    const r2Data: MetaInfoCacheDataR2 = {
      version: "1.0",
      stats_data_id: statsDataId,
      saved_at: new Date().toISOString(),
      meta_info_response: metaInfo,
      summary: {
        table_title: tableInf.TITLE?.$ || "",
        stat_name: tableInf.STAT_NAME?.$ || "",
        organization: tableInf.GOV_ORG?.$ || "",
        survey_date: tableInf.SURVEY_DATE || "",
        updated_date: tableInf.UPDATED_DATE || "",
      },
    };

    // R2オブジェクトキー生成
    const key = `estat_metainfo/${statsDataId}/meta.json`;

    // JSONに変換
    const jsonString = JSON.stringify(r2Data, null, 2);
    const jsonBuffer = new TextEncoder().encode(jsonString);

    // R2に保存
    await env.METAINFO_BUCKET.put(key, jsonBuffer, {
      httpMetadata: {
        contentType: "application/json",
      },
      customMetadata: {
        "stats-data-id": statsDataId,
        "saved-at": r2Data.saved_at,
        "table-title": r2Data.summary.table_title,
      },
    });

    return {
      key,
      size: jsonBuffer.byteLength,
    };
  }
}
```

#### バッチ保存パターン

```typescript
export class BatchR2Repository {
  static async batchSave(
    env: { METAINFO_BUCKET: R2Bucket },
    items: Array<{ key: string; data: any }>
  ): Promise<Array<{ key: string; success: boolean; error?: string }>> {
    const results = await Promise.allSettled(
      items.map(async (item) => {
        const jsonString = JSON.stringify(item.data, null, 2);
        const jsonBuffer = new TextEncoder().encode(jsonString);

        await env.METAINFO_BUCKET.put(item.key, jsonBuffer, {
          httpMetadata: {
            contentType: "application/json",
          },
        });

        return { key: item.key, success: true };
      })
    );

    return results.map((result, index) => {
      if (result.status === "fulfilled") {
        return result.value;
      } else {
        return {
          key: items[index].key,
          success: false,
          error: result.reason.message,
        };
      }
    });
  }
}
```

### R2 からの読み込み

#### 基本的な読み込みパターン

```typescript
export class EstatMetaInfoR2Repository {
  static async getMetaInfo(
    env: { METAINFO_BUCKET: R2Bucket },
    statsDataId: string
  ): Promise<EstatMetaInfoResponse | null> {
    try {
      const key = `estat_metainfo/${statsDataId}/meta.json`;
      const object = await env.METAINFO_BUCKET.get(key);

      if (!object) {
        return null;
      }

      const jsonText = await object.text();
      const cacheData: MetaInfoCacheDataR2 = JSON.parse(jsonText);

      return cacheData.meta_info_response as EstatMetaInfoResponse;
    } catch (error) {
      console.error("R2メタ情報キャッシュ取得エラー:", error);
      return null;
    }
  }
}
```

#### 条件付き読み込み

```typescript
export class ConditionalR2Repository {
  static async getWithCondition(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string,
    condition: (metadata: any) => boolean
  ): Promise<any | null> {
    try {
      const object = await env.METAINFO_BUCKET.get(key);

      if (!object) {
        return null;
      }

      // メタデータをチェック
      if (condition(object.customMetadata)) {
        const data = await object.json();
        return data;
      }

      return null;
    } catch (error) {
      console.error("R2条件付き読み込みエラー:", error);
      return null;
    }
  }
}
```

### キャッシュ戦略

#### TTL 付きキャッシュ

```typescript
export class TTLR2Repository {
  static async saveWithTTL(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string,
    data: any,
    ttlHours: number = 24
  ): Promise<void> {
    const expiresAt = new Date();
    expiresAt.setHours(expiresAt.getHours() + ttlHours);

    const cacheData = {
      data,
      expires_at: expiresAt.toISOString(),
    };

    const jsonString = JSON.stringify(cacheData, null, 2);
    const jsonBuffer = new TextEncoder().encode(jsonString);

    await env.METAINFO_BUCKET.put(key, jsonBuffer, {
      httpMetadata: {
        contentType: "application/json",
      },
      customMetadata: {
        "expires-at": expiresAt.toISOString(),
      },
    });
  }

  static async getWithTTL(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string
  ): Promise<any | null> {
    try {
      const object = await env.METAINFO_BUCKET.get(key);

      if (!object) {
        return null;
      }

      const cacheData = await object.json();
      const expiresAt = new Date(cacheData.expires_at);

      if (new Date() > expiresAt) {
        // TTL期限切れ
        await env.METAINFO_BUCKET.delete(key);
        return null;
      }

      return cacheData.data;
    } catch (error) {
      console.error("R2 TTL読み込みエラー:", error);
      return null;
    }
  }
}
```

#### キャッシュヒット・ミス戦略

```typescript
export class CacheStrategyR2Repository {
  static async getWithFallback<T>(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string,
    fallbackFn: () => Promise<T>,
    ttlHours: number = 24
  ): Promise<T> {
    // キャッシュから取得を試行
    const cached = await this.getWithTTL(env, key);

    if (cached) {
      console.log(`R2キャッシュヒット: ${key}`);
      return cached;
    }

    console.log(`R2キャッシュミス: ${key}`);

    // フォールバック関数を実行
    const data = await fallbackFn();

    // キャッシュに保存
    await this.saveWithTTL(env, key, data, ttlHours);

    return data;
  }
}
```

### バッチ処理

#### 大量データの分割処理

```typescript
export class BatchProcessorR2Repository {
  static async processLargeDataset(
    env: { METAINFO_BUCKET: R2Bucket },
    data: any[],
    batchSize: number = 100
  ): Promise<Array<{ key: string; success: boolean; error?: string }>> {
    const results: Array<{ key: string; success: boolean; error?: string }> =
      [];

    for (let i = 0; i < data.length; i += batchSize) {
      const batch = data.slice(i, i + batchSize);

      const batchResults = await Promise.allSettled(
        batch.map(async (item, index) => {
          const key = `batch/${i + index}/${item.id}.json`;
          const jsonString = JSON.stringify(item, null, 2);
          const jsonBuffer = new TextEncoder().encode(jsonString);

          await env.METAINFO_BUCKET.put(key, jsonBuffer, {
            httpMetadata: {
              contentType: "application/json",
            },
          });

          return { key, success: true };
        })
      );

      results.push(
        ...batchResults.map((result, index) => {
          if (result.status === "fulfilled") {
            return result.value;
          } else {
            return {
              key: `batch/${i + index}/${batch[index].id}.json`,
              success: false,
              error: result.reason.message,
            };
          }
        })
      );
    }

    return results;
  }
}
```

### トラブルシューティング

#### よくある問題

1. **"R2 バケットが設定されていません"エラー**

   - 原因: 環境変数`METAINFO_BUCKET`が設定されていない
   - 解決: Cloudflare 環境で R2 バケットの環境変数を設定

2. **権限エラー**

   - 原因: R2 バケットへの書き込み権限がない
   - 解決: Cloudflare アカウントで R2 バケットの権限を確認

3. **開発環境で R2 保存されない**
   - 原因: 正常な動作（開発環境ではスキップされる）
   - 解決: 本番環境でテストする

```typescript
export class MetadataSearchR2Repository {
  static async searchByMetadata(
    env: { METAINFO_BUCKET: R2Bucket },
    filters: Record<string, string>
  ): Promise<Array<{ key: string; metadata: any }>> {
    const results: Array<{ key: string; metadata: any }> = [];

    // プレフィックスで絞り込み
    const prefix = filters.prefix || "";
    const list = await env.METAINFO_BUCKET.list({ prefix });

    for (const object of list.objects) {
      const metadata = object.customMetadata || {};

      // フィルタ条件をチェック
      const matches = Object.entries(filters).every(([key, value]) => {
        if (key === "prefix") return true;
        return metadata[key] === value;
      });

      if (matches) {
        results.push({
          key: object.key,
          metadata,
        });
      }
    }

    return results;
  }
}
```

```typescript
export class ParallelR2Repository {
  static async parallelGet(
    env: { METAINFO_BUCKET: R2Bucket },
    keys: string[]
  ): Promise<Array<{ key: string; data: any | null; error?: string }>> {
    const results = await Promise.allSettled(
      keys.map(async (key) => {
        try {
          const object = await env.METAINFO_BUCKET.get(key);
          const data = object ? await object.json() : null;
          return { key, data };
        } catch (error) {
          return { key, data: null, error: error.message };
        }
      })
    );

    return results.map((result, index) => {
      if (result.status === "fulfilled") {
        return result.value;
      } else {
        return {
          key: keys[index],
          data: null,
          error: result.reason.message,
        };
      }
    });
  }
}
```

```typescript
export class LifecycleR2Repository {
  static async setLifecyclePolicy(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string,
    daysToExpire: number
  ): Promise<void> {
    const expiresAt = new Date();
    expiresAt.setDate(expiresAt.getDate() + daysToExpire);

    await env.METAINFO_BUCKET.put(key, data, {
      customMetadata: {
        "expires-at": expiresAt.toISOString(),
      },
    });
  }

  static async cleanupExpired(
    env: { METAINFO_BUCKET: R2Bucket },
    prefix: string = ""
  ): Promise<number> {
    const list = await env.METAINFO_BUCKET.list({ prefix });
    const now = new Date();
    let deletedCount = 0;

    for (const object of list.objects) {
      const expiresAt = object.customMetadata?.["expires-at"];

      if (expiresAt && new Date(expiresAt) < now) {
        await env.METAINFO_BUCKET.delete(object.key);
        deletedCount++;
      }
    }

    return deletedCount;
  }
}
```

#### 圧縮の活用

```typescript
export class CompressionR2Repository {
  static async saveCompressed(
    env: { METAINFO_BUCKET: R2Bucket },
    key: string,
    data: any
  ): Promise<{ key: string; size: number; compressedSize: number }> {
    const jsonString = JSON.stringify(data, null, 2);
    const originalSize = new TextEncoder().encode(jsonString).byteLength;

    // Gzip圧縮
    const compressed = await this.compress(jsonString);

    await env.METAINFO_BUCKET.put(key, compressed, {
      httpMetadata: {
        contentType: "application/json",
        contentEncoding: "gzip",
      },
    });

    return {
      key,
      size: originalSize,
      compressedSize: compressed.byteLength,
    };
  }

  private static async compress(text: string): Promise<Uint8Array> {
    const stream = new CompressionStream("gzip");
    const writer = stream.writable.getWriter();
    const reader = stream.readable.getReader();

    writer.write(new TextEncoder().encode(text));
    writer.close();

    const chunks: Uint8Array[] = [];
    let done = false;

    while (!done) {
      const { value, done: readerDone } = await reader.read();
      done = readerDone;
      if (value) {
        chunks.push(value);
      }
    }

    const totalLength = chunks.reduce((sum, chunk) => sum + chunk.length, 0);
    const result = new Uint8Array(totalLength);
    let offset = 0;

    for (const chunk of chunks) {
      result.set(chunk, offset);
      offset += chunk.length;
    }

    return result;
  }
}
```
