# D-15 女性労働力率が高い県は出生率が高いのか

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 女性労働力率が高い県は本当に出生率が高いのか ── 「M字カーブ解消」と合計特殊出生率の県別相関を検証
- **切り口**: 「女性が働く県ほど出生率が高い」通説を 47 県データで検証。M字カーブ解消度と保育所充足率・所得・産業構造の重回帰
- **必要データ**: 国勢調査（年齢別労働力率）/ 人口動態調査（合計特殊出生率）/ 厚労省保育所等の状況 / 賃金構造基本統計
- **関連 stats47 ブログ**: women-labor
- **アフィ枠**: 育児・働き方関連（A8.net）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 「女性労働力率高 = 出生率高」通説をデータで検証
2. 47県の散布図 ── 強い相関はあるか
3. 第3要因の影響 ── 保育所充足率・3世代同居率・所得
4. M字カーブ解消が進んだ県 ── 福井・島根・鳥取の構造
5. 「働くほど産めない」構造 ── 都市部の罠
6. 政策示唆 ── どの順序で何をすべきか
