# D-4 高齢者独居率が示す「見えない危機」

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 高齢者独居率20%超の県が示す「見えない危機」 ── 孤独死・買い物難民・医療アクセスのデータ分析
- **切り口**: 高齢者独居率は単なる人口指標ではない。孤独死・買い物難民・医療アクセス・介護崩壊の予兆指標として読む
- **必要データ**: 国勢調査（世帯類型別）/ 消防庁孤独死統計 / 厚労省介護保険事業状況報告
- **関連 stats47 ブログ**: aging-solo-living-crisis
- **アフィ枠**: 高齢者見守りサービス（A8.net）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 高齢者独居率ランキング ── 上位5県の共通点
2. 孤独死データとの相関 ── 独居率が高い県で何が起きているか
3. 「買い物難民」の地理 ── スーパー撤退と独居高齢者
4. 医療アクセスの限界 ── 通院困難者と訪問医療の県別差
5. 元県庁職員のリアル ── 民生委員・包括支援センターが見える地獄
6. 2040年予測 ── 独居率がさらに上がる5県
