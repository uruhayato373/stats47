# D-13 介護保険料が県財政を食う日

## 概要

note 有料化候補。

## 企画詳細

- **想定タイトル**: 介護保険料が県財政を食う日 ── 2040年「保険料月8,000円超」自治体ランキングと持続不可能な構造
- **切り口**: 第1号被保険者保険料の県別差。2024年実績→2040年推計で「月8,000円超」となる県・市区町村
- **必要データ**: 介護保険事業状況報告 / 国立社人研将来推計 / 厚労省介護保険料水準集計
- **関連 stats47 ブログ**: care-insurance-burden
- **アフィ枠**: 介護保険商品（A8.net）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 介護保険料2024年 県別ランキング
2. 2040年推計 ── 「月8,000円超」自治体の地図
3. なぜ保険料が高くなるのか ── 3つの要因（高齢化率・サービス利用率・施設密度）
4. 「保険料は安いが介護が受けられない」県の構造
5. 県財政への波及 ── 法定負担分が圧迫する一般会計
6. 制度改革の方向性 ── 全国一律化・自己負担引き上げの議論
