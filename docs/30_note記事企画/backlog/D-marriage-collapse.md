# D-2 「未婚化」の正体 結婚しない/できない県

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 47都道府県「未婚率」の正体 ── 結婚しない県と結婚できない県の違いをデータで分解する
- **切り口**: 「未婚化」は単一現象ではない。所得・性比・産業構造・通勤圏で原因が異なる
- **必要データ**: 国勢調査（年齢別未婚率）/ 人口動態調査（婚姻率）/ 賃金構造基本統計調査（県別所得）/ 住民基本台帳（性比）
- **関連 stats47 ブログ**: marriage-unmarried-crisis
- **アフィ枠**: 婚活サービス（A8.net）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 都道府県別「30代未婚率」上位下位ランキング
2. 「結婚しない」3パターン ── 高所得都市型 / 低所得地方型 / 性比偏重型
3. 男女別に分解 ── 男余り県と女余り県の地図
4. 産業構造との相関 ── 製造業県と未婚率
5. 政策効果はあるか ── 婚活支援自治体の成果検証
