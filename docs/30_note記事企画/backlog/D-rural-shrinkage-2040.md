# D-7 2040 過疎崩壊フロンティア

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 2040過疎崩壊フロンティア ── 自治体半分が消滅可能性都市になる県と「持ちこたえる」県の境界線
- **切り口**: 増田レポートの「消滅可能性都市」を実データで再検証。2014年予測から10年経った結果と2040年再予測
- **必要データ**: 国立社会保障・人口問題研究所 将来推計人口（市区町村別）/ 国勢調査 / 住民基本台帳人口移動報告
- **関連 stats47 ブログ**: population-future
- **アフィ枠**: なし（教養記事）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 「消滅可能性都市」2014→2024の通信簿
2. 2040年再予測 ── 新たな消滅可能性自治体
3. 県別「自治体存続率」 ── 半分以上が消える県
4. 「持ちこたえる」自治体の3条件 ── 産業・若年女性・通勤圏
5. 自治体合併の限界 ── 平成の大合併が解決しなかったこと
6. 元県庁職員のリアル ── 「閉じる」自治体の現場
