# 掛け合わせ相関記事 note企画 (型B)

## 概要

掛け合わせコンテンツ戦略 (型B = 統計×統計相関) の note 記事企画。ブログ企画
(`docs/20_ブログ記事企画/backlog/gis-cross-correlation-articles.md`) と同じ 8 本を、
note 読者向けに企画する。

**note 読者の特性** (アクセス実測 2026-04〜05 より): 都道府県ランキング記事を好む。
1 位は「年間日照時間ランキング」728 ビュー。→ note 版は **ランキングを前面**に出し、
「2 つのランキングを並べると意外な関係が見える」という切り口で書く。#2 日照×太陽光
note 記事 (`docs/31_note記事原稿/sunshine-solar-housing/`) と同じ続編フォーマット。

- 文字数: 3,500〜4,500字、無料
- ブログ (SEO・解説) と差別化: note は「ランキング読み比べ」のエンタメ性を強める
- frontmatter は既存の一般ランキング note 形式 (`title` / `description`) に合わせる
  (執筆時に `docs/31_note記事原稿/sunshine-solar-housing/note.md` を参照)

## 共通の執筆規律 (全記事必須)

- **相関 ≠ 因果**: 「○○だから××」と断定しない。「○○の県は××も多い」と観測事実で
- **集計してから筋を決める**: 県別データを集計し外れ値を見てから物語を確定
- **r を正確に**: 相関係数を誇張しない
- **第三の変数**: 都市度・所得・年齢構成など背景要因に必ず触れる

## 各記事の企画 (note 版の切り口)

各記事は対応するブログ企画 (A〜H) と同じデータ・同じ規律。note 版の差分のみ記す。

- **A 大学卒×海外旅行**: 「海外旅行ランキング」を主役に。日照時間ランキングの読者は
  ランキングが好き → 海外旅行行動者率ランキングを見せ、そこに学歴ランキングを重ねる
- **B テレワーク×学び**: 「テレワーク実施率ランキング」起点。在宅で働ける県は何が違うか
- **C 将来人口×スポーツ**: 「将来人口が増える県ランキング」起点。若い県の活気
- **D 音楽×旅行**: 2 つの「行動者率ランキング」読み比べ。アクティブな県の人物像
- **E 通勤圏×海外旅行**: 「他市区町村通勤者比率ランキング」起点。都市圏近郊の暮らし
- **F 有価証券×学歴**: 「有価証券保有割合ランキング」起点。資産形成と学歴
- **G 持ち家×単独世帯**: 持ち家比率は既存 note 記事あり (アクセス上位)。その続編として
  「持ち家の県とひとり暮らしの県は逆」を見せる
- **H 共同住宅×高齢世帯**: 都市/地方の住まいの地図。#2 日照×太陽光 note と構造が近い
  ため、切り口を「高齢世帯」側に寄せて重複を避ける

## 展開順 (案)

note は #2 (日照×太陽光) の続編フォーマットが実証済み。G (持ち家) は既存の人気記事
「持ち家比率ランキング」の続編として書けるため反応が読みやすい → G を先行候補に。
A・B も話題性が高い。最終的な順序は note アクセス実測を見て調整。

## ブログとの関係

同じ 8 テーマをブログ (SEO・解説) と note (ランキング読み物) の両方で展開する。
ブログ → 検索流入を stats47 に。note → ランキング読者の獲得。
両者でデータは共通、文体と主役の見せ方を変える。
