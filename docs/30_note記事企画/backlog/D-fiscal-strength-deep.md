# D-3 財政力指数1.0超は東京だけ 自治体家計簿

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 財政力指数1.0超は東京都だけ ── 元県庁職員が読み解く47都道府県「自治体家計簿」の通信簿
- **切り口**: 財政力指数・実質公債費比率・将来負担比率の3指標で47県を採点。「健全」と「危機」の境界線
- **必要データ**: 地方財政状況調査（健全化判断比率4指標）/ 地方交付税交付金 / 県民経済計算
- **関連 stats47 ブログ**: fiscal-strength-ranking
- **アフィ枠**: なし（教養記事）
- **優先度**: S

## 構成案（H2 見出しレベルで）
1. 「財政力指数」とは何か ── 1.0超が意味するもの
2. 47県の財政力指数ランキング ── 東京1.18 vs 島根0.27
3. 健全化判断比率4指標で見る ── 北海道・夕張ショック以降の制度設計
4. 「交付税依存度」が高い県の構造 ── 自前財源と国費の比率
5. 元県庁職員のリアル ── 年度末の駆け込み予算執行の実態
6. 2030年予測 ── 高齢化で財政が破綻する県はあるか
