# D-10 公務員の仕事が「価値を生まない」と言われる構造

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 公務員の仕事は本当に「価値を生まない」のか ── 元県庁職員が解剖する47都道府県庁の業務構造とアウトプット
- **切り口**: 「公務員バッシング」の根拠と反論をデータで分解。県職員1人あたり住民数・人件費比率・業務アウトカム指標
- **必要データ**: 地方公共団体定員管理調査 / 地方財政状況調査 / 県民経済計算 / 行政評価レポート
- **関連 stats47 ブログ**: civil-servant
- **アフィ枠**: 公務員試験対策（A8.net）
- **優先度**: S

## 構成案（H2 見出しレベルで）
1. 都道府県職員1人あたり住民数ランキング
2. 人件費が県財政に占める比率 ── 「人件費高すぎ」論の検証
3. 「価値を生む」の定義問題 ── アウトカム指標で公務員を評価できるか
4. 元県庁職員のリアル ── 8割の業務は「制度の運用」
5. 民間委託・指定管理の限界 ── どこまで外注できるのか
6. 「住民1人あたり行政サービスコスト」県別ランキング
