# D-1 東京一極集中30年データ深掘り

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 東京一極集中30年 ── データで見る「地方創生」が止められなかった人口の重力
- **切り口**: 1995→2025の30年で、人口・所得・企業本社・大学進学先の集中度はどう変わったか。地方創生政策（2014〜）後も加速したか
- **必要データ**: 住民基本台帳人口移動報告（県別転入超過数）/ 国勢調査人口 / 県民経済計算 / 経済センサス（本社所在地）/ 学校基本調査（進学先）
- **関連 stats47 ブログ**: population-migration / fiscal-strength-ranking
- **アフィ枠**: なし（教養記事）
- **優先度**: S

## 構成案（H2 見出しレベルで）
1. 1995年と2025年 ── 東京圏のシェアはどう変わったか
2. 「転入超過」の正体 ── 18〜22歳の若年層と30代の二山構造
3. 地方創生10年（2014〜2024）の通信簿 ── 数値で見る政策評価
4. 経済集中の指標 ── 県民総生産・上場企業本社・大学進学先
5. 元県庁職員のリアル ── 「地方の声」がどう中央に届かないか
6. 2040年予測 ── このまま行くと地方はどうなるか
