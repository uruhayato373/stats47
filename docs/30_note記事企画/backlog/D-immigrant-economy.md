# D-9 外国人住民が支える地方経済

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 外国人住民が支える地方経済 ── 製造業・農業・介護現場のデータと「2030年なしでは回らない」県
- **切り口**: 外国人住民比率の高い県（群馬・愛知・静岡・三重）の産業構造分析。製造業・農業・介護で「すでに不可欠」な実態
- **必要データ**: 在留外国人統計 / 国勢調査 / 事業所統計 / 厚労省外国人雇用状況届出
- **関連 stats47 ブログ**: foreign-resident
- **アフィ枠**: なし（教養記事）
- **優先度**: A

## 構成案（H2 見出しレベルで）
1. 外国人住民比率ランキング ── 群馬・愛知・静岡の構造
2. 業種別 ── 製造業・農業・介護で何%を占めるか
3. 「事実上の移民国家」の現状 ── 技能実習生・特定技能・永住者
4. 出身国別の地域分布 ── ブラジル/ベトナム/中国/フィリピン
5. 自治体の支援体制差 ── 多文化共生に積極的な県
6. 2030年予測 ── 外国人住民なしでは産業が成立しない県
