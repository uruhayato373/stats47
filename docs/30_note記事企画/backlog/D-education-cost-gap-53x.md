# D-6 教育格差53倍 家計調査の真実

## 企画詳細

- **想定タイトル**: 教育格差53倍 ── 家計調査が暴く「補習教育」の世帯所得別支出が示す日本の階層化
- **切り口**: 家計調査 265 品目の中で「補習教育」だけが Q1/Q5 で53倍の格差。食料は1.6倍、光熱は1.2倍なのに教育だけ突出する構造
- **必要データ**: 家計調査（年収五分位×品目別）statsDataId=0002070005 / 学校基本調査（進学率県別）
- **関連 stats47 ブログ**: educationsports
- **アフィ枠**: 学習塾・教材（A8.net）
- **優先度**: S

## 構成案（H2 見出しレベルで）
1. 家計調査 265 品目で「最大格差」は何か
2. 補習教育53倍 ── 塾・家庭教師費の Q1/Q5 比較
3. 「教育以外は格差が小さい」事実 ── 食料1.6倍・光熱1.2倍
4. 進学率の県別差に現れる結果 ── 東京・神奈川 vs 沖縄・青森
5. 時系列で見る格差拡大 ── 2014→2024の10年
6. 政策の限界 ── 高校無償化後も補習教育格差は縮まらなかった
