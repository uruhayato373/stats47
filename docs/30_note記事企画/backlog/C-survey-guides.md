# C シリーズ — 統計調査ガイド集約

## 概要
日本の主要統計調査（国勢調査・家計調査・賃金構造基本統計調査等）の活用ガイド。
2,000〜5,000字。元県庁職員視点で「データの落とし穴」「実務での使い方」を解説。
C-1（家計調査）は完了済み（[note記事URL]参照）。
