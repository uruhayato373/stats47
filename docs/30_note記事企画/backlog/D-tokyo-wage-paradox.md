# D-5 東京の年収が高くても貯まらない構造

## 概要
3,000〜5,000字の深掘り分析。元県庁職員視点 + データの読み方注釈。
note 有料化候補。

## 企画詳細

- **想定タイトル**: 東京の年収はなぜ「貯まらない」のか ── 47都道府県「実質可処分所得」ランキングの罠
- **切り口**: 賃金額面ランキングと貯蓄ランキングは一致しない。住居費・通勤コスト・物価で「実質」を見ると別の県が浮上
- **必要データ**: 賃金構造基本統計調査 / 家計調査（県別貯蓄現在高）/ 消費者物価地域差指数 / 住宅・土地統計（家賃）
- **関連 stats47 ブログ**: savings-balance-ranking
- **アフィ枠**: 資産運用・iDeCo（A8.net）
- **優先度**: S

## 構成案（H2 見出しレベルで）
1. 「年収日本一」と「貯蓄日本一」が違う理由
2. 東京の家計簿を分解 ── 住居費・通勤・教育費の重さ
3. 「実質可処分所得」ランキング ── 物価補正後の本当の豊かさ
4. 「貯蓄ができる県」TOP5 の家計構造
5. ライフステージ別の最適県 ── 独身/子育て/老後で異なる答え
