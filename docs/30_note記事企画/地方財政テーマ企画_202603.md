### ラスパイレス指数 データ取得手順（年次更新用）

#### 概要

- ranking_key: `laspeyres-index-prefecture`
- data_source: `soumu`（総務省報道発表、adapter_type: manual）
- 登録済み: 14年分（H24/2012 〜 R7/2025、658行）

#### データソース

| 年度 | ソース | URL |
|---|---|---|
| 最新年度（R7〜） | 総務省報道発表 PDF | https://www.soumu.go.jp/iken/kyuyo.html |
| 過去年度（〜R6） | 地方公務員給与実態調査 Excel | https://www.soumu.go.jp/main_sosiki/jichi_gyousei/c-gyousei/teiin-kyuuyo02.html |

#### Excel ファイル構造

- シート名: `都道府県(47)`（または類似名）
- 列構成: C列 = 団体コード、E列 = ラスパイレス指数
- H24（2012）は `.xls` 形式、H25（2013）以降は `.xlsx` 形式
- 団体コード → area_code 変換: 先頭2桁を取得（例: `010006` → `01`）

#### データ注意点

- **H24（2012）・H25（2013）の値は異常値ではない**: 国家公務員給与が東日本大震災復興のため臨時削減（2012-2013）された結果、地方公務員の指数が相対的に100〜112と高く出ている。これは制度上の基準変動であり、実際に地方公務員給与が急上昇したわけではない。

#### 年次更新手順

1. 総務省報道発表ページから最新年度の PDF/Excel を取得
2. Python で Excel をパース（`.xlsx` は `openpyxl`、`.xls` は `xlrd`）
3. 47都道府県分の JSON を生成
4. Node.js（`better-sqlite3`）でローカル D1 に INSERT
5. `ranking_items` の `available_years`・`latest_year` を UPDATE
6. `/sync-remote-d1` でリモート D1 に反映
