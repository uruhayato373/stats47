# note 戦略（note.com）

> 更新: 2026-03-11
> **このファイルが note 関連の SSOT（Single Source of Truth）**。他ドキュメントからはリンク参照のみ。

## 全体戦略

### コンセプト: 「統計をもっと身近に」の実践

note は stats47 の **認知獲得・被リンク獲得・フォロワー基盤構築** の場として運用する。note.com のドメインパワーを「自サイトの敵」ではなく「自サイトへ送客する最強の味方」として活用する。

**方針（2026-03-11 確定）**: stats47.jp のドメインパワーが成長途上である現段階では、**noteのドメインパワーを活用して検索上位を押さえ、stats47.jp への動線を確保する方が全体最適**。ランキング記事の量産（A シリーズ）でビューを稼ぎ、考察記事（B/C/D シリーズ）で信頼とフォロワーを構築する2層戦略を取る。

### ファネル構造

```
量産層（A シリーズ: ランキング記事）
  Google検索・noteレコメンドで大量流入
  → 記事内 stats47.jp リンク（UTM付き）で送客
  → 関連する B/C/D 記事への内部リンクで回遊

考察層（B/C/D シリーズ: 読み物・ガイド・分析）
  noteフォロワー・検索で流入
  → stats47.jp リンク（UTM付き）で送客
  → 信頼構築 → フォロワー獲得
    ↓
収益化（stats47.jp）
  「実際のランキング・グラフ・地図を見てみよう」← AdSense + アフィリエイト収益
```

### コンテンツ棲み分けルール

stats47.jp と note で**同じテーマを扱ってもよい**。それぞれの強みで差別化する。

| | note（A シリーズ） | note（B/C/D シリーズ） | stats47.jp |
|---|---|---|---|
| **内容** | ランキング要約 + 簡単な考察 | ストーリー・考察・調査ガイド | 全47都道府県データ + インタラクティブ |
| **文字数** | 800〜1,200字 | 1,500〜5,000字 | — |
| **CTA** | 「全県版は stats47 で →」 | 「詳細データは stats47 で →」 | 本体 |
| **制作** | `/post-note-ranking` で自動生成 | 4ステップワークフロー | — |

> **canonical URL**: note.com ではcanonicalタグを自由に設定できないため、noteの記事は「独立コンテンツ」として設計する。

### 投稿ルール

- 全記事に stats47.jp へのリンクを UTM パラメータ付きで **5〜10本** 設置
- 記事末尾に**固定CTA**（stats47.jp への誘導）を設置
- A シリーズは `/post-note-ranking` スキルで自動生成
- B/C/D シリーズは `/validate-note-idea` → `/investigate-note-data` → `/design-note-structure` → `/write-note-section` → `/edit-note-draft` のフローで作成

## シリーズ A: ランキング記事（量産型）

**目的**: DB の `ranking_items` からランキング記事を自動生成し、note のドメインパワーでビューを獲得する。全記事に stats47.jp リンクを設置し、送客の主力チャネルとする。

**制作フロー**: `/post-note-ranking` スキルで一括生成（4ステップワークフロー不要）

**共通構成**（800〜1,200字）:
1. フック（1位と最下位の差を提示し、驚きを与える）
2. ランキング結果（上位5 + 下位5）
3. 簡単な考察（地域パターン・背景を2〜3段落）
4. stats47.jp リンク集 + 固定CTA

**タイトル形式**: 「【20XX年版】都道府県〇〇ランキング｜ポイント要約」
- 既存の実績あるタイトル形式を踏襲
- stats47.jp のブログ記事とはタイトルが異なるため共存可能

**stats47.jp との棲み分け**:
- note A シリーズ: 上位5+下位5の要約 + 簡単な考察（テキストベース）
- stats47.jp: 全47都道府県 + インタラクティブチャート + ソート + フィルタ + CSV ダウンロード

**量産ルール**:
- ビューが見込めるテーマから優先（人口・経済・生活・教育 > ニッチ指標）
- 1記事あたり stats47.jp リンクを **5本以上** 設置（UTM付き）
- 関連する B/C/D 記事があればリンクで誘導

---

## シリーズ B: ランキング読み物（考察型）

**目的**: A シリーズのランキングデータを「読み物」として再構成し、深い考察で差別化する。A シリーズで反応が良かったテーマを B シリーズで深掘りする流れが理想。

**制作フロー**: `/validate-note-idea` → `/investigate-note-data` → `/design-note-structure` → `/write-note-section` → `/edit-note-draft`

**共通構成**（1,500〜2,500字）:
1. フック（意外な事実・読者の常識を揺さぶる問い）
2. ランキング結果の要約（上位5〜10県を提示）
3. なぜこの結果になるのか（背景・歴史・地理・産業構造を考察）
4. 元行政職員の視点（データの定義の落とし穴、調査方法の注意点）
5. 「もっと詳しく知りたい方へ」→ stats47.jp リンク集

**タイトル形式**: 「なぜ〜なのか」「意外な〜」「〜の正体」等の読み物型

**A シリーズとの連携**:
- A シリーズ記事の末尾に「考察記事はこちら →」で B シリーズへリンク
- B シリーズ記事の冒頭に「ランキング全体はこちら →」で A シリーズへリンク

**目的**: 「〇〇調査でわかること・活用方法」を解説し、stats47.jp への被リンクとフォロワー獲得を両立する。note のブランディング基盤。

**制作フロー**: `/validate-note-idea` → `/investigate-note-data` → `/design-note-structure` → `/write-note-section` → `/edit-note-draft`

**共通構成**（2,000〜5,000字）:
1. この調査とは（概要・実施機関・周期）
2. どんなデータが取れるか（主要指標の一覧）
3. 都道府県別に見るとここが面白い（3〜5 個の具体例 + stats47.jp リンク）
4. 元行政職員の視点（調査の裏側・データの注意点・現場での活用例）
5. 活用シーン（ビジネス・研究・日常の意思決定）

**目的**: 複数指標・時系列変化・社会構造を横断的に論じるエッセイ。フォロワー獲得・エンゲージメント向上が主目的。

**B シリーズとの違い**:
- B: 1つのランキング指標を取り上げてライトに考察（1,500〜2,500字）

**共通構成**:
1. 問題提起（社会課題としてのフレーミング）
