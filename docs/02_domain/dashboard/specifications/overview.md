# ダッシュボードドメイン概要

## 目的

ダッシュボードドメインは、e-Stat APIから取得した統計データを、全国・都道府県・市区町村の3階層で可視化するためのシステムです。カテゴリ・サブカテゴリ別に統計カード、各種グラフ、コロプレス地図を組み合わせて、ユーザーが直感的に統計情報を理解できるインターフェースを提供します。

## 3階層構造

### 1. 全国ダッシュボード（areaCode: "00000"）
- **対象**: 日本全国の統計データ
- **特徴**: 都道府県ランキング、全国分布地図、推移グラフ
- **用途**: 国全体の傾向把握、都道府県間比較

### 2. 都道府県ダッシュボード（areaCode: "01000"-"47000"）
- **対象**: 特定の都道府県の統計データ
- **特徴**: 全国比較、市区町村ランキング、都道府県内分布
- **用途**: 都道府県の特徴分析、市区町村間比較

### 3. 市区町村ダッシュボード（areaCode: "01101"-"47999"）
- **対象**: 特定の市区町村の統計データ
- **特徴**: 都道府県内順位、周辺比較、詳細分析
- **用途**: 地域の詳細分析、近隣自治体との比較

## 主要機能

### 統計カード
- 単一指標の数値表示
- 前年比・前回比の表示
- カラーコーディングによる視覚的強調
- ローディング・エラー状態の適切な表示

### 時系列グラフ
- **折れ線グラフ**: 年度別推移の表示
- **複数折れ線グラフ**: 複数指標の同時比較
- **棒グラフ**: 年度別数値比較
- **積み上げ棒グラフ**: 内訳の可視化

### 構成比グラフ
- **円グラフ**: 男女比率、年齢構成など
- **人口ピラミッド**: 年齢別人口構成
- **ドーナツチャート**: カテゴリ別構成比

### 地図可視化
- **コロプレス地図（都道府県レベル）**: 全国の都道府県別分布
- **コロプレス地図（市区町村レベル）**: 都道府県内の市区町村別分布
- インタラクティブ機能（hover、click、ズーム）

### 比較・ランキング
- 全国ランキング（都道府県別）
- 都道府県内ランキング（市区町村別）
- 周辺地域との比較
- 上位・下位の表示

### 可視化ライブラリ
- **Recharts**: 基本的なグラフ（折れ線、棒、円グラフ）
- **D3.js**: 高度な可視化（人口ピラミッド、カスタムグラフ）
- **TopoJSON**: 地図データの管理

### スタイリング
- **Tailwind CSS 4**: ユーティリティファーストCSS
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応
- **ダークモード**: システム設定に応じた自動切り替え

### データ取得
- **e-Stat API**: 政府統計データの取得
- **Cloudflare D1**: メタデータのキャッシュ
- **Cloudflare R2**: ランキングデータのキャッシュ

## カテゴリ・サブカテゴリ対応

### 人口統計
- **基本人口**: 総人口、男女別人口、年齢別人口
- **世帯**: 世帯数、世帯構成
- **出生・死亡**: 出生率、死亡率、自然増減率
- **人口移動**: 転入・転出、社会増減
- **人口構成**: 年齢構成、就業構造

### 経済統計
- **経済指標**: GDP、県民所得
- **産業構造**: 就業者数、事業所数
- **家計**: 家計収支、消費支出
- **労働**: 就業率、失業率、賃金

### 社会統計
- **教育**: 学校数、児童・生徒数、進学率
- **医療**: 病院数、医師数、健康指標
- **福祉**: 社会保障給付、介護施設
- **安全**: 犯罪発生率、交通事故

### 環境・インフラ
- **環境**: 大気汚染、水質、廃棄物
- **インフラ**: 道路、上下水道、エネルギー
- **土地利用**: 森林率、農地率、市街地率

## レイアウトシステム

### 固定順序レイアウト
ユーザーエクスペリエンス最適化のため、ダッシュボードの表示順序は固定されています。

#### 全国ダッシュボード
1. 統計カード（主要指標）
2. 都道府県ランキング
3. 地域分布地図
4. 推移グラフ
5. 詳細分析

#### 都道府県ダッシュボード
1. 統計カード（都道府県指標）
2. 全国比較グラフ
3. 市区町村ランキング
4. 推移グラフ
5. 周辺都道府県比較

#### 市区町村ダッシュボード
1. 統計カード（市区町村指標）
2. 都道府県内順位
3. 周辺比較グラフ
4. 推移グラフ
5. 詳細分析

### レスポンシブデザイン
- **モバイル**: 1列レイアウト、縦スクロール
- **タブレット**: 2列レイアウト、グラフサイズ最適化
- **デスクトップ**: 3列レイアウト、フル機能表示

## データフロー

```
ユーザーリクエスト
    ↓
ページコンポーネント（SSR）
    ↓
カテゴリ・サブカテゴリ検証
    ↓
地域コード分析（国/都道府県/市区町村判定）
    ↓
ダッシュボードコンポーネント選択
    ↓
可視化ウィジェット
    ↓
e-Stat API データ取得
    ↓
データ整形・表示
```

## まとめ

ダッシュボードドメインは、日本の統計データを3階層で可視化する包括的なシステムです。全国・都道府県・市区町村の各レベルで最適化された表示を提供し、ユーザーが直感的に統計情報を理解できるインターフェースを実現します。

技術的には、Next.js、React、TypeScriptを基盤とし、RechartsとD3.jsを組み合わせた可視化システム、e-Stat APIとの統合、レスポンシブデザイン、アクセシビリティ対応など、現代的なWebアプリケーションのベストプラクティスを採用しています。

このシステムにより、政府統計データがより身近で理解しやすい形で提供され、データドリブンな意思決定を支援することができます。
