# ブログドメイン概要

## ドメインの目的

ブログドメインは、stats47プロジェクトにおけるMDXベースのブログ機能を管理するドメインです。統計データの可視化を含む豊かなコンテンツを提供し、ユーザーにデータドリブンな分析記事を提供します。

## 主要機能

### 1. MDXコンテンツ管理
- **MDXファイル**: Markdown + JSXの組み合わせで記事を作成
- **フロントマター**: メタデータ、SEO設定、可視化設定を管理
- **コンポーネント埋め込み**: Reactコンポーネントを記事内で直接利用

### 2. 可視化コンポーネント統合
- **D3.jsコンポーネント**: コロプレス地図、棒グラフ、折れ線グラフ
- **Rechartsコンポーネント**: インタラクティブなチャート
- **カスタムコンポーネント**: Alert、Callout、LinkCardなど

### 3. データ連携
- **統計データ統合**: R2ハイブリッドアーキテクチャとの連携
- **ランキングデータ**: 都道府県別ランキングの表示
- **時系列データ**: 過去データの推移表示

### 4. SEO・パフォーマンス
- **静的生成**: SSGによる高速表示
- **構造化データ**: 検索エンジン最適化
- **キャッシュ戦略**: Cloudflare KV/R2を活用

## アーキテクチャ概要

```
MDXファイル → フロントマター解析 → MDX変換 → コンポーネントマッピング → レンダリング
     ↓
統計データAPI ← データ連携 ← 可視化コンポーネント
```

## 技術スタック

- **MDX処理**: next-mdx-remote, gray-matter
- **変換プラグイン**: rehype, remark
- **シンタックスハイライト**: shiki
- **検索**: FlexSearch
- **キャッシュ**: Cloudflare KV/R2

## ドメイン境界

### 内部責任
- MDXファイルの読み込み・解析
- フロントマターの処理
- コンポーネントマッピング
- 記事のレンダリング
- 検索機能
- SEO最適化

### 外部依存
- **Visualizationドメイン**: 可視化コンポーネント
- **Rankingドメイン**: 統計データ取得
- **Databaseドメイン**: メタデータ保存（任意）

## データフロー

1. **記事作成**: MDXファイル作成 → フロントマター設定
2. **ビルド時**: MDX解析 → 静的生成 → キャッシュ保存
3. **表示時**: キャッシュ取得 → コンポーネントレンダリング
4. **検索時**: FlexSearchインデックス → 結果表示

## 品質要件

### パフォーマンス
- **ページ読み込み**: < 2秒
- **検索応答**: < 500ms
- **キャッシュヒット率**: > 90%

### アクセシビリティ
- **WCAG 2.1 AA準拠**
- **キーボードナビゲーション対応**
- **スクリーンリーダー対応**

### SEO
- **構造化データ**: Article schema
- **メタタグ**: 適切なOGP設定
- **サイトマップ**: 自動生成
