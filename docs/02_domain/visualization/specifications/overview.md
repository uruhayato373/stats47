# 可視化ドメイン概要

**作成日**: 2025-10-16  
**バージョン**: 1.0  
**対象**: 全可視化機能の設計・実装・管理

---

## 目的

stats47プロジェクトにおける全ての可視化機能を管理するドメイン。D3.jsとRechartsの両方を使用して、様々なチャートとインタラクティブな可視化を提供します。

## 対象範囲

### 地図可視化
- コロプレスマップ（都道府県・市区町村レベル）
- インタラクティブな地図操作（ズーム・パン）
- 地域データの色分け表示

### チャート可視化
- 棒グラフ（ランキング表示）
- 折れ線グラフ（時系列データ）
- 散布図（相関分析）
- ヒートマップ（多変量データ）
- その他の統計チャート

### インタラクティブ機能
- ホバー時の詳細表示
- クリック時の詳細ページ遷移
- フィルタリング・ソート機能
- アニメーション・トランジション

### レスポンシブデザイン
- モバイルからデスクトップまで対応
- 画面サイズに応じた最適化
- タッチ操作対応

## 使用ライブラリ

### D3.js
**用途**: 高度なカスタマイズが必要な可視化

**特徴**:
- SVGの完全なコントロール
- 複雑なアニメーション・トランジション
- 独自のインタラクション実装
- データドリブンなDOM操作

**適用例**:
- コロプレスマップ
- カスタムツールチップ付き地図
- 複雑なネットワーク図
- 高度なインタラクション付きチャート

### Recharts
**用途**: 標準的なチャート表示

**特徴**:
- 宣言的なAPI
- React統合が容易
- 迅速な開発が可能
- レスポンシブ対応が組み込み済み

**適用例**:
- ランキングの棒グラフ
- 時系列の折れ線グラフ
- ダッシュボードの集計チャート
- 標準的な統計表示

## 設計原則

### 1. 一貫性
- 全てのチャートで共通のデザインシステム
- 統一されたカラーパレット
- 共通のインタラクションパターン
- 一貫したアニメーション

### 2. パフォーマンス
- 大量データでも高速レンダリング
- メモリ効率の良い実装
- 仮想スクロール対応
- 適切なキャッシュ戦略

### 3. アクセシビリティ
- WCAG 2.1 AA準拠
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 高コントラスト表示

### 4. レスポンシブ
- モバイルファーストデザイン
- 画面サイズに応じた最適化
- タッチ操作対応
- 適切なフォントサイズ

## アーキテクチャ

### コンポーネント構造
```
src/components/charts/
├── d3js/
│   ├── ChoroplethMap.tsx
│   ├── BarChart.tsx
│   └── LineChart.tsx
```

```
src/lib/visualization/
├── d3js/
│   ├── choropleth/
│   ├── bar-chart/
│   └── common/
└── recharts/
    ├── bar-chart/
    ├── line-chart/

```

## データフロー

1. **データ取得**: ランキングデータサービスから取得
2. **データ変換**: 可視化用フォーマットに変換
3. **レンダリング**: 選択されたライブラリで描画
4. **インタラクション**: ユーザー操作に応じた更新
5. **状態管理**: React状態と同期
