# YouTube 動画候補選定

**選定日**: 2026-03-20
**選定基準**: `youtube_retention_analysis_20260319.md` の分析結果に基づく

## 選定基準

離脱分析から判明した成功パターン:

1. **尺は28-35秒が最適**（維持率63-64%、90秒超は37%）
2. **身近で自分ごと化しやすいテーマ**（体重96.1%、退職金129.9%、月収57.0%）
3. **格差が大きいとインパクトあり**（1位と47位の倍率が大きい）
4. **煽れるタイトル**が冒頭維持に効く（「あなたの県は何位？」パターン）

### 除外済み

- 消費支出額/消費量ランキング（大量にあるが動画映えしにくい）
- 行財政の内部指標（一般財源割合等、関心層が限定的）
- 既存46動画と重複するテーマ

## 推奨する制作優先順位

### 第1弾（即制作 × 高確度）

1. **公務員の平均給与** — 退職手当129.9%の再現狙い
2. **公務員のボーナス** — セットで投稿
3. **手取り収入ランキング** — 月収57%の発展版
4. **がん受療率** — 脳血管疾患83.1%の再現狙い
5. **人口密度2024** — 100倍差のインパクト、28秒に短縮再制作

### 第2弾（時事連動）

6. **家賃ランキング2024** — 家賃高騰ニュースと連動
7. **外国人人口** — 移民議論の時事性
8. **投票率** — 次の選挙前に投稿

### 第3弾（季節連動）

9. **アイス消費額** — 夏前に投稿
10. **ビール消費額** — 夏に投稿

## 制作時の注意点

- **尺は28秒を厳守**（35秒以下）
- **冒頭3秒で結論**を出す（「1位は○○県！」）
- **テロップで数値を大きく表示**（スマホ視聴前提）
- **「あなたの県は何位？」**のフックをタイトル・冒頭に入れる
- **113秒の長尺動画は廃止**。バズった交通事故・県民所得のみ例外検討
