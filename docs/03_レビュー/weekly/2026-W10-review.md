## 学び・ナレッジ

- **IPSS データは e-Stat API にない**: 市区町村別将来推計人口は IPSS サイトから Excel で直接取得する必要がある。e-Stat 検索では見つからない
  - `/knowledge` に記録推奨: はい

- **TopoJSON の `_city.topojson` vs `_city_dc.topojson`**: `_dc`（dissolve city）版は政令指定都市を1ポリゴンにまとめる。区レベルで表示するには `_city.topojson` を使う
  - `/knowledge` に記録推奨: はい

- **IPSS Excel の都道府県コードは先頭ゼロなし**: 01〜09 の県で `zfill(5)` が必要（例: `1100` → `01100`）
  - `/knowledge` に記録推奨: はい

- **Next.js のサーバーサイドキャッシュ**: ファイルが存在しなかった時の結果がキャッシュされ、ファイル追加後も null が返り続ける。`.next/cache` の削除が必要
  - `/knowledge` に記録推奨: いいえ（一般的な知識）
