# stats47 North Star Metric

生成日: 2026-03-09

## ビジネスゲーム

- **分類**: Attention ゲーム
- **理由**: 収益モデル（AdSense + アフィリエイト）がページビュー・滞在時間に依存。ユーザーが統計コンテンツを閲覧する時間と深さが、直接的に広告収益を決定する。データ提供（Productivity）の要素もあるが、現段階では Attention が主軸。

---

## North Star Metric

### 週間エンゲージドセッション数

ユーザーが stats47.jp で **2ページ以上閲覧したセッション数**（週次）。

- **定義**: GA4 の `engaged_sessions`（10秒以上滞在 or 2ページ以上閲覧 or コンバージョン発生）の週間合計
- **現在値**: 未計測（GA4 定期計測が未実施）
- **目標値（3ヶ月後）**: 週 2,000 セッション（≒ 月間 8,000 → PV 3万相当）

### なぜこの指標か

1. **「見た」だけでなく「探索した」を測る**: 1ページで離脱したユーザーは広告もアフィリエイトもほぼ見ない。2ページ以上 = ユーザーが価値を感じて深掘りした証拠
2. **収益モデルとの直接相関**: ページ/セッションが増えるほど広告インプレッション・アフィリエイト接触が増加
3. **ビジョンとの整合**: 「ランキングで笑った人が原データを読めるようになる」のファネル進行は、ページ遷移として計測可能
4. **施策の効果が即座に反映**: 記事公開、内部リンク改善、SNS 投稿のどれも、エンゲージドセッション数に数日で影響する

### 7 基準チェック

| # | 基準 | 評価 | 根拠 |
|---|---|---|---|
| 1 | 理解しやすい | ○ | 「今週、2ページ以上見てくれた人が何人いたか」で誰でも分かる |
| 2 | 顧客中心 | ○ | ユーザーの探索行動（=価値体験）を直接反映。売上指標ではない |
| 3 | 持続的価値 | ○ | エンゲージしたユーザーはリピーターになりやすく、長期的な習慣を示す先行指標 |
| 4 | ビジョン整合 | ○ | 「統計を知る→理解する→活用する」のファネルは2ページ以上の探索として発現する |
| 5 | 定量的 | ○ | GA4 で自動計測。明確な数値 |
| 6 | アクション可能 | ○ | 記事公開（流入増）、内部リンク改善（ページ/セッション増）、SNS投稿（新規流入）で直接影響 |
| 7 | 先行指標 | ○ | エンゲージドセッション ↑ → PV ↑ → AdSense収益 ↑ の因果チェーン |

### 却下した候補

| 候補 | 却下理由 |
|---|---|
| 月間PV | 遅行指標であり「質」を反映しない。PV が増えても直帰率90%なら収益に貢献しない |
| 月間ユニークユーザー数 | 「来た」だけで「価値を感じた」かが不明。1ページ離脱を含む |
| 月間オーガニック検索流入数 | SEO のみの指標。SNS・note 経由の流入が反映されない。チャネル偏重 |
| 月間データ引用数 | 重要だが計測が困難（被リンク検知はリアルタイム性が低い）。週次の意思決定に使えない |
| 週間リピーター数 | 重要だが現段階でリピーター数が少なすぎてノイズが大きい。トラフィック規模が出てから再検討 |
| AdSense 収益 | NSM は売上そのものではない。顧客視点の指標であるべき |

## Input Metrics

NSM（週間エンゲージドセッション数）を駆動する 4 つの Input Metrics。

### 1. 週間オーガニック検索流入数

- **定義**: GSC のクリック数（週次合計）
- **NSM との関係**: 検索流入はエンゲージドセッションの最大のソース。検索意図が明確なためページ遷移率が高い
- **現在値**: 未計測
- **目標値**: 週 1,500 クリック（3ヶ月後）
- **施策例**:
  - ブログ記事の継続公開（現69本 → 100本）
  - GSC データで順位 11-30 位の記事をリライト
  - FAQ 構造化データの追加

### 2. 週間SNS流入セッション数

- **定義**: GA4 の SNS 経由セッション数（X + Instagram + YouTube + note）の週次合計
- **NSM との関係**: SNS はオーガニック検索を補完する流入チャネル。特に新規ユーザー獲得に有効
- **現在値**: ほぼ 0（投稿開始直後）
- **目標値**: 週 300 セッション（3ヶ月後）
- **施策例**:
  - X 週3投稿の定着
  - Instagram 週2投稿
  - YouTube Shorts の説明欄からの誘導

### 3. ページ/セッション

- **定義**: GA4 の `screen_page_views / sessions`（週平均）
- **NSM との関係**: 流入数が同じでも、ページ/セッションが高ければエンゲージドセッション数が増える。コンテンツの深さと内部リンクの質を反映
- **現在値**: 未計測
- **目標値**: 2.5 以上（3ヶ月後）
- **施策例**:
  - ランキングページ → 関連ブログ記事への CTA
  - ブログ記事 → 関連ランキングへのリンク
  - カテゴリページの回遊導線強化

### 4. 公開コンテンツ数

- **定義**: 公開済み記事数 + アクティブランキングページ数
- **NSM との関係**: 検索流入のエントリポイントが増え、回遊先の選択肢も増える。コンテンツの「供給」側の指標
- **現在値**: 69記事 + 1,511ランキング = 1,580ページ
- **目標値**: 100記事 + 1,511ランキング = 1,611ページ（3ヶ月後）
- **施策例**:
  - `/plan-blog-articles` で記事企画を量産
  - トレンド連動記事の速報体制
  - 既存記事の品質向上（リライト）

## メトリクスツリー

```
週間エンゲージドセッション数（NSM）
├── 週間オーガニック検索流入数
│   ├── ブログ記事の公開・リライト
│   ├── 構造化データ（FAQ, Dataset）の追加
│   └── 被リンク獲得（note, UGC）
├── 週間SNS流入セッション数
│   ├── X 週3投稿
│   ├── Instagram 週2投稿
│   └── YouTube 説明欄・ピン留めコメントの誘導
├── ページ/セッション
│   ├── 内部リンクの最適化
│   ├── 関連コンテンツ CTA の配置
│   └── カテゴリ・サブカテゴリの回遊導線
└── 公開コンテンツ数
    ├── 新規記事の公開
    ├── トレンド連動記事
    └── 既存記事のリライト
```

## 計測方法

| 指標 | データソース | 更新頻度 | スキル |
|---|---|---|---|
| NSM（エンゲージドセッション） | GA4 | 週次（月曜） | `/fetch-ga4-data` |
| オーガニック検索流入 | GSC | 週次（月曜） | `/fetch-gsc-data` |
| SNS流入セッション | GA4 | 週次（月曜） | `/fetch-ga4-data` |
| ページ/セッション | GA4 | 週次（月曜） | `/fetch-ga4-data` |
| 公開コンテンツ数 | DB (articles + ranking_items) | `/weekly-plan` 内で自動計測 | — |

## `/weekly-plan` への統合

`/weekly-plan` の「現状サマリー」テーブルに以下の行を追加する:

```markdown
## 現状サマリー

| 指標 | 現在値 | 目標 | 前週比 |
|---|---|---|---|
| **NSM: 週間エンゲージドセッション** | N | 2,000 | +N% |
| Input: オーガニック検索流入（週） | N | 1,500 | +N% |
| Input: SNS流入セッション（週） | N | 300 | +N% |
| Input: ページ/セッション | N.N | 2.5 | +N% |
| Input: 公開コンテンツ数 | N | 1,611 | +N |
```

これにより、毎週の計画策定時に NSM と Input Metrics の推移が自動的に評価され、タスクの優先度判断に使われる。
