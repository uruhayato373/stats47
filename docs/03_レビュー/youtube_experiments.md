### 仮説の詳細

- **なぜ人口か**: 競合「ランクの森」が同テーマで180万再生。最も自分ごと化しやすいテーマで、フォーマットの効果を純粋に測定できる
- **なぜ6秒か**: 6秒だとスワイプ前に動画が終わり、維持率が構造的に高くなる。ループ再生もカウントされる
- **成功基準**: 初動48h再生 > 100（ショートフィード露出成功の閾値）
- **失敗時の撤退判断**: 48h再生 < 50 かつ維持率 < 80% → テーマ問題かフォーマット問題かを切り分け

### 競合分析メモ

- **ランクの森**: 人口ランキング TOP15 / 6秒 / 180万再生 / いいね率 0.09%
- タグが内容と不一致（プロ野球年俸タグ → 人口ランキング動画）= 意図的な関連動画ハイジャック
- エンゲージメント率が極めて低い → アルゴリズムに乗って受動的に消費された
