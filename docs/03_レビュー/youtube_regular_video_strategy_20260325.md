## 8. stats47の通常動画が伸びなかった理由と改善策

| 問題 | 現状 | 改善策 |
|---|---|---|
| **尺が短すぎる** | 3:13〜3:17 | **4:50〜5:00に延長** |
| **テーマが地味** | 社会増減率、退職金 | **犯罪・外国人・年収など生活密着テーマ** |
| **タイトルに感情フックがない** | 「社会増減率ランキング」 | **「1位は○○じゃない！？」「ワースト県は？」** |
| **「都道府県別」の位置** | タイトルに含まれていない | **先頭に「都道府県別」を配置**（4倍の効果） |
| **投稿本数が少なすぎる** | 2本 | **週2〜3本ペースで量産** |
| **初動のエンゲージメントゼロ** | いいね・コメント0 | Shortsからの誘導、コメント誘導CTA |

## 9. 推奨アクションプラン

### Phase 1（最初の10本）

1. **テンプレート化**: 4:50〜5:00の動画テンプレートを確立
2. **Sランクから着手**: 犯罪→外国人→生活保護→年収→離婚率の順
3. **タイトル最適化**: 「都道府県別 ○○ランキング｜意外性のある副題」の型
4. **公開時間固定**: JST 20時
5. **投稿頻度**: 週2本以上

### Phase 2（11本目以降）

1. Aランクテーマに展開
2. 初期10本の再生数データを元にテーマ・タイトルを最適化
3. 時事ネタとの組合せ（選挙、災害、季節イベント等）
