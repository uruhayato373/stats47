

# データで見る日本の農業地図：就業人口と生産性から読み取れる地域格差と農業の現実

## はじめに

日本の農業を語る上で欠かせないのが、実際に農業に従事する人々の分布と、その生産性です。2014年度の農業就業人口（販売農家）と2018年度の就業者1人当たり農業産出額のデータを組み合わせて分析してみると、単なる数字以上に、日本の地域特性や農業政策の課題、そして各地域の農業戦略の違いが浮かび上がってきます。

## 就業人口と生産性：異なる農業戦略が見えるトップランキング

### 就業人口トップ3の実態

**1位：北海道（96,557人）**
圧倒的な1位は北海道。約9万7千人という数字は、偏差値73.9という高い数値からも分かる通り、他県を大きく引き離しています。しかし興味深いのは生産性データです。2018年度の就業者1人当たり農業産出額は1,304.2万円で、これも全国1位。つまり北海道は「人数も多く、生産性も高い」という理想的な農業大国の姿を示しています。

**2位：茨城県（89,594人）**
意外かもしれませんが、茨城県が2位。首都圏に近い立地でありながら、約9万人の農業就業者を抱えています。生産性は503.2万円（全国6位）と高水準で、多様な農産物の生産基地として効率的な農業を展開していることが分かります。

**3位：長野県（82,922人）**
山間地が多い長野県が3位にランクイン。しかし生産性データを見ると315.5万円（全国25位）と中位レベル。これは標高差を活かした多品目栽培や果樹栽培など、手間をかける農業の特徴を反映していると考えられます。

### 生産性ランキングで見える新たな視点

就業人口ランキングとは異なる顔を見せるのが生産性ランキングです。

**生産性2位：鹿児島県（840.2万円）**
就業人口では13位だった鹿児島県が、生産性では2位に躍進。畜産業を中心とした高付加価値農業の成果が数字に表れています。

**生産性3位：宮崎県（762.0万円）**
同じく畜産県として知られる宮崎県も、就業人口21位から生産性3位へと大きくジャンプアップ。九州地方の畜産業の競争力の高さを物語っています。

## 地域格差の現実：最下位グループが物語る課題と可能性

### 就業人口最下位グループの実態

**47位：東京都（10,986人）**
東京都の農業就業者数は約1万1千人。偏差値34.5という数値は、都市化の進展を如実に示しています。しかし生産性は218.5万円（全国46位）と、都市農業特有の小規模・高コスト構造が見て取れます。

**46位：大阪府（14,796人）**
大阪府も同様に、就業者数の少なさ（14,796人）に加えて、生産性も224.4万円（全国45位）と低位に留まっています。

### 生産性最下位が示す構造的課題

興味深いのは、生産性最下位が奈良県（209.6万円）であることです。就業人口も43位と少なく、小規模農業の限界を示しているとも言えるでしょう。一方で、石川県は就業人口では45位と少ないものの、生産性では311.3万円（27位）と健闘しており、効率的な農業経営への取り組みが見えます。

## 中間層に見る多様な農業戦略

20位台から30位台を見ると、各地域の特色と戦略の違いが明確に表れています。

### 高付加価値戦略で勝負する地域

**群馬県**：就業人口22位（44,006人）→ 生産性5位（557.7万円）
人数は少なくても高い生産性を実現。施設園芸や畜産での成功例と考えられます。

**千葉県**：就業人口6位（73,410人）→ 生産性4位（580.2万円）
首都圏への供給基地として、人数・生産性ともに高水準を維持。

### 従来型農業の健闘組

**岡山県（20位・48,469人）**：生産性は289.1万円（33位）と中位ながら、瀬戸内の温暖な気候を活かした安定した農業基盤を維持。

**愛媛県（23位・41,104人）**：生産性は300.0万円（32位）。柑橘類中心の農業構造が数字に反映されています。

### 意外な躍進組

**沖縄県**：就業人口42位（19,916人）と少ないものの、生産性は496.1万円（8位）と高順位。亜熱帯農業の特殊性と高付加価値作物への特化が功を奏しています。

**佐賀県**：就業人口35位（26,244人）から生産性10位（486.6万円）へ大幅上昇。効率的な農業経営への転換が成功していることが窺えます。

## データから読み取れる5つの重要ポイント

### 1. 「規模」と「効率性」は必ずしも比例しない
北海道のように両方を兼ね備える理想型もあれば、群馬県や千葉県のように比較的少ない人数で高い生産性を実現する「効率特化型」、長野県のように多くの人が従事しながらも生産性は中位の「雇用重視型」など、多様な農業モデルが存在します。

### 2. 畜産業の競争力が際立つ
鹿児島県、宮崎県の生産性の高さは、畜産業の付加価値の高さを示しています。これは飼料効率の改善や品種改良、6次産業化の成果と考えられます。

### 3. 都市近郊農業の二極化
首都圏では、茨城県・千葉県が高い生産性を維持する一方で、東京都・神奈川県は低迷。同じ都市近郊でも明暗が分かれています。

### 4. 中山間地域の構造的課題
長野県、新潟県など、就業人口は多いものの生産性が中位に留まる地域は、地形的制約による効率化の限界を示している可能性があります。

### 5. 地方小県の戦略的転換
沖縄県、佐賀県のように、就業人口では下位でも高い生産性を実現している地域は、特色ある農業への特化が成功していることを示しています。

## 2014年から2018年への変化と現在への示唆

就業人口データが2014年、生産性データが2018年という4年間の時差は、農業構造の変化を読み取る上で興味深い視点を提供します。

この間に日本の農業界では以下のような変化が起きました：

- **TPP11発効に向けた競争力強化の取り組み**
- **AI・IoTを活用したスマート農業の本格的導入**
- **担い手の大規模化・法人化の加速**
- **6次産業化の推進**

これらの変化が、各地域の生産性向上にどのような影響を与えたかを考察すると、現在（2025年）の農業状況をより深く理解できるはずです。特に、生産性で上位にランクインした地域は、これらの変化にいち早く対応できた地域と言えるかもしれません。

## まとめ：数字が語る日本農業の戦略地図

このデータから見えてくるのは、日本の農業の多様性と、それぞれの地域が取る戦略の違いです。

**北海道型：大規模・高効率モデル**
広大な土地を活かした大規模農業で、人数・生産性ともにトップを実現。

**九州畜産県型：高付加価値特化モデル**
鹿児島県、宮崎県に代表される、畜産業を中心とした高収益農業。

**首都圏近郊型：市場密着モデル**
茨城県、千葉県のような、大消費地に近い立地を活かした効率的農業。

**地方特化型：ニッチ戦略モデル**
沖縄県、佐賀県のような、地域特性を活かしたユニークな農業展開。

**従来型維持モデル**
長野県、新潟県のような、多くの人が従事する従来型農業の維持。
