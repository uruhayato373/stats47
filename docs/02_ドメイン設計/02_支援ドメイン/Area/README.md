# Area（地域管理ドメイン）

## 概要

地域情報の管理を担当する支援ドメイン。都道府県・市区町村の情報を管理し、他のドメインに地域データを提供します。

## 責務

- 地域情報の管理
- 地域コードの管理
- 地域階層の管理
- 地域名の管理
- 地域の代表点管理

## ユビキタス言語

- **地域**: 都道府県・市区町村などの行政区域
- **地域コード**: 地域を一意に識別するコード
- **地域階層**: 国 → 地方 → 都道府県 → 市区町村の階層構造
- **代表点**: 地域の代表的な座標点

## 主要エンティティ

- Area（地域）
- AreaHierarchy（地域階層）
- AreaCode（地域コード）
