# Category（カテゴリ管理ドメイン）

## 概要

統計カテゴリの管理を担当する支援ドメイン。統計データの分類体系を管理し、他のドメインにカテゴリ情報を提供します。

## 責務

- 統計カテゴリの管理
- カテゴリ階層の管理
- サブカテゴリの管理
- カテゴリの表示名管理
- カテゴリの並び順管理

## ユビキタス言語

- **カテゴリ**: 統計データの大分類
- **サブカテゴリ**: カテゴリの細分類
- **カテゴリコード**: カテゴリを一意に識別するコード
- **表示名**: ユーザーに表示されるカテゴリ名

## 主要エンティティ

- Category（カテゴリ）
- Subcategory（サブカテゴリ）
- CategoryHierarchy（カテゴリ階層）
