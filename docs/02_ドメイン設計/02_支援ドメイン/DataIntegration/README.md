# DataIntegration（データ統合ドメイン）

## 概要

外部データソースとの統合を担当する支援ドメイン。e-Stat API やその他のデータソースとの連携を管理します。

## 責務

- 外部データソースとの連携
- データの取得・変換
- データの正規化
- データの品質管理
- データのキャッシュ管理

## ユビキタス言語

- **データソース**: 統計データの出典
- **API**: 外部システムとの連携インターフェース
- **データ変換**: 外部データ形式の内部形式への変換
- **データ品質**: データの信頼性・完全性

## サブドメイン

### e-Stat-API

e-Stat API との連携機能

## 主要エンティティ

- DataSource（データソース）
- ApiEndpoint（API エンドポイント）
- DataTransformation（データ変換）
