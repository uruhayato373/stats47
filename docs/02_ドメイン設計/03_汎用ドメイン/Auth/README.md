# Auth（認証ドメイン）

## 概要

ユーザー認証・認可を担当する汎用ドメイン。ユーザーの身元確認とアクセス制御を管理します。

## 責務

- ユーザー認証
- ユーザー認可
- セッション管理
- パスワード管理
- アクセス制御

## ユビキタス言語

- **認証**: ユーザーの身元確認
- **認可**: ユーザーのアクセス権限確認
- **セッション**: ユーザーのログイン状態
- **トークン**: 認証情報の表現

## 主要エンティティ

- User（ユーザー）
- Session（セッション）
- Permission（権限）
