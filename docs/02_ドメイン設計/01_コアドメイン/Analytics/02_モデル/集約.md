# Analytics ドメイン - 集約

## 集約の定義

集約は、関連するエンティティと値オブジェクトを一つの単位として扱い、データの整合性を保つための境界を定義します。

## 主要な集約

### RankingAggregate（ランキング集約）

ランキング項目とその値の集約。

```typescript
class RankingAggregate {
  private constructor(
    private readonly rankingItem: RankingItem,
    private readonly rankingValues: RankingValue[]
  ) {}

  static create(
    rankingItem: RankingItem,
    rankingValues: RankingValue[]
  ): RankingAggregate {
    return new RankingAggregate(rankingItem, rankingValues);
  }

  getRankingItem(): RankingItem {
    return this.rankingItem;
  }

  getRankingValues(): RankingValue[] {
    return [...this.rankingValues]; // 防御的コピー
  }

  getRankingValuesForYear(year: Year): RankingValue[] {
    return this.rankingValues.filter((value) => value.getYear().equals(year));
  }

  getTopN(n: number): RankingValue[] {
    return this.rankingValues
      .sort((a, b) => a.getRank() - b.getRank())
      .slice(0, n);
  }

  getBottomN(n: number): RankingValue[] {
    return this.rankingValues
      .sort((a, b) => b.getRank() - a.getRank())
      .slice(0, n);
  }

  getRankingValueByArea(areaCode: AreaCode): RankingValue | null {
    return (
      this.rankingValues.find((value) =>
        value.getAreaCode().equals(areaCode)
      ) || null
    );
  }

  addRankingValue(rankingValue: RankingValue): void {
    // ビジネスルール: 同じ地域・年度の値は重複できない
    const existingIndex = this.rankingValues.findIndex(
      (value) =>
        value.getAreaCode().equals(rankingValue.getAreaCode()) &&
        value.getYear().equals(rankingValue.getYear())
    );

    if (existingIndex >= 0) {
      this.rankingValues[existingIndex] = rankingValue;
    } else {
      this.rankingValues.push(rankingValue);
    }
  }

  removeRankingValue(areaCode: AreaCode, year: Year): void {
    this.rankingValues = this.rankingValues.filter(
      (value) =>
        !value.getAreaCode().equals(areaCode) || !value.getYear().equals(year)
    );
  }
}
```

### StatisticalAnalysisAggregate（統計分析集約）

統計分析結果と関連データの集約。

```typescript
class StatisticalAnalysisAggregate {
  private constructor(
    private readonly analysis: StatisticalAnalysis,
    private readonly rankingValues: RankingValue[]
  ) {}

  static create(
    analysis: StatisticalAnalysis,
    rankingValues: RankingValue[]
  ): StatisticalAnalysisAggregate {
    return new StatisticalAnalysisAggregate(analysis, rankingValues);
  }

  getAnalysis(): StatisticalAnalysis {
    return this.analysis;
  }

  getRankingValues(): RankingValue[] {
    return [...this.rankingValues]; // 防御的コピー
  }

  getOutliers(): RankingValue[] {
    const quartiles = this.analysis.getQuartiles();
    const iqr = quartiles.getIQR();
    const lowerBound = quartiles.getQ1() - 1.5 * iqr;
    const upperBound = quartiles.getQ3() + 1.5 * iqr;

    return this.rankingValues.filter(
      (value) => value.getValue() < lowerBound || value.getValue() > upperBound
    );
  }

  getNormalRange(): { min: number; max: number } {
    const quartiles = this.analysis.getQuartiles();
    const iqr = quartiles.getIQR();
    return {
      min: quartiles.getQ1() - 1.5 * iqr,
      max: quartiles.getQ3() + 1.5 * iqr,
    };
  }

  isValueOutlier(value: number): boolean {
    const normalRange = this.getNormalRange();
    return value < normalRange.min || value > normalRange.max;
  }
}
```

### ComparisonAggregate（比較分析集約）

比較分析結果と関連データの集約。

```typescript
class ComparisonAggregate {
  private constructor(
    private readonly comparison: Comparison,
    private readonly baseRankingValue: RankingValue,
    private readonly comparisonRankingValues: RankingValue[]
  ) {}

  static create(
    comparison: Comparison,
    baseRankingValue: RankingValue,
    comparisonRankingValues: RankingValue[]
  ): ComparisonAggregate {
    return new ComparisonAggregate(
      comparison,
      baseRankingValue,
      comparisonRankingValues
    );
  }

  getComparison(): Comparison {
    return this.comparison;
  }

  getBaseRankingValue(): RankingValue {
    return this.baseRankingValue;
  }

  getComparisonRankingValues(): RankingValue[] {
    return [...this.comparisonRankingValues]; // 防御的コピー
  }

  getLargestDifference(): { areaCode: AreaCode; difference: number } {
    let maxDifference = 0;
    let maxAreaCode = this.comparison.getBaseAreaCode();

    for (const areaCode of this.comparison.getComparisonAreaCodes()) {
      const difference = Math.abs(this.comparison.getDifference(areaCode));
      if (difference > maxDifference) {
        maxDifference = difference;
        maxAreaCode = areaCode;
      }
    }

    return { areaCode: maxAreaCode, difference: maxDifference };
  }

  getSmallestDifference(): { areaCode: AreaCode; difference: number } {
    let minDifference = Infinity;
    let minAreaCode = this.comparison.getBaseAreaCode();

    for (const areaCode of this.comparison.getComparisonAreaCodes()) {
      const difference = Math.abs(this.comparison.getDifference(areaCode));
      if (difference < minDifference) {
        minDifference = difference;
        minAreaCode = areaCode;
      }
    }

    return { areaCode: minAreaCode, difference: minDifference };
  }

  getRankingByDifference(): Array<{ areaCode: AreaCode; difference: number }> {
    return this.comparison
      .getComparisonAreaCodes()
      .map((areaCode) => ({
        areaCode,
        difference: this.comparison.getDifference(areaCode),
      }))
      .sort((a, b) => Math.abs(b.difference) - Math.abs(a.difference));
  }
}
```

### TimeSeriesAnalysisAggregate（時系列分析集約）

時系列分析結果と関連データの集約。

```typescript
class TimeSeriesAnalysisAggregate {
  private constructor(
    private readonly analysis: TimeSeriesAnalysis,
    private readonly rankingValues: RankingValue[]
  ) {}

  static create(
    analysis: TimeSeriesAnalysis,
    rankingValues: RankingValue[]
  ): TimeSeriesAnalysisAggregate {
    return new TimeSeriesAnalysisAggregate(analysis, rankingValues);
  }

  getAnalysis(): TimeSeriesAnalysis {
    return this.analysis;
  }

  getRankingValues(): RankingValue[] {
    return [...this.rankingValues]; // 防御的コピー
  }

  getRankingValuesSortedByYear(): RankingValue[] {
    return this.rankingValues.sort(
      (a, b) => a.getYear().getValue() - b.getYear().getValue()
    );
  }

  getYearOverYearChange(year: Year): number {
    return this.analysis.getYearOverYearChange(year);
  }

  getPeakYear(): Year {
    const sortedValues = this.getRankingValuesSortedByYear();
    let peakValue = -Infinity;
    let peakYear = sortedValues[0].getYear();

    for (const value of sortedValues) {
      if (value.getValue() > peakValue) {
        peakValue = value.getValue();
        peakYear = value.getYear();
      }
    }

    return peakYear;
  }

  getTroughYear(): Year {
    const sortedValues = this.getRankingValuesSortedByYear();
    let troughValue = Infinity;
    let troughYear = sortedValues[0].getYear();

    for (const value of sortedValues) {
      if (value.getValue() < troughValue) {
        troughValue = value.getValue();
        troughYear = value.getYear();
      }
    }

    return troughYear;
  }

  getVolatility(): number {
    const values = this.rankingValues.map((v) => v.getValue());
    const mean = values.reduce((sum, val) => sum + val, 0) / values.length;
    const variance =
      values.reduce((sum, val) => sum + Math.pow(val - mean, 2), 0) /
      values.length;
    return Math.sqrt(variance);
  }
}
```

## 集約の設計原則

1. **集約ルート**: 各集約には 1 つの集約ルートがある
2. **境界**: 集約の境界は明確に定義される
3. **整合性**: 集約内のデータ整合性は集約ルートが保証する
4. **不変条件**: 集約の不変条件は常に満たされる
5. **参照**: 集約間の参照は ID のみで行う

## ビジネスルール

1. **RankingAggregate**: 同じランキング項目の値のみを含む
2. **StatisticalAnalysisAggregate**: 分析対象のランキング値のみを含む
3. **ComparisonAggregate**: 比較対象のランキング値のみを含む
4. **TimeSeriesAnalysisAggregate**: 時系列分析対象のランキング値のみを含む
