# Analytics ドメイン - 値オブジェクト

## 主要な値オブジェクト

### RankingKey（ランキングキー）

ランキング項目を一意に識別する値オブジェクト。

```typescript
class RankingKey {
  private constructor(private readonly value: string) {
    if (!value || value.trim().length === 0) {
      throw new Error("RankingKey cannot be empty");
    }
  }

  static create(value: string): RankingKey {
    return new RankingKey(value);
  }

  getValue(): string {
    return this.value;
  }

  equals(other: RankingKey): boolean {
    return this.value === other.value;
  }

  toString(): string {
    return this.value;
  }
}
```

### Year（年度）

年度を表す値オブジェクト。

```typescript
class Year {
  private constructor(private readonly value: number) {
    if (value < 1900 || value > 2100) {
      throw new Error("Year must be between 1900 and 2100");
    }
  }

  static create(value: number): Year {
    return new Year(value);
  }

  getValue(): number {
    return this.value;
  }

  equals(other: Year): boolean {
    return this.value === other.value;
  }

  isAfter(other: Year): boolean {
    return this.value > other.value;
  }

  isBefore(other: Year): boolean {
    return this.value < other.value;
  }

  toString(): string {
    return this.value.toString();
  }
}
```

### DataSource（データソース）

データの出典を表す値オブジェクト。

```typescript
class DataSource {
  private constructor(
    private readonly source: string,
    private readonly version: string,
    private readonly lastUpdated: Date
  ) {
    if (!source || source.trim().length === 0) {
      throw new Error("DataSource source cannot be empty");
    }
  }

  static create(
    source: string,
    version: string,
    lastUpdated: Date
  ): DataSource {
    return new DataSource(source, version, lastUpdated);
  }

  getSource(): string {
    return this.source;
  }

  getVersion(): string {
    return this.version;
  }

  getLastUpdated(): Date {
    return this.lastUpdated;
  }

  equals(other: DataSource): boolean {
    return this.source === other.source && this.version === other.version;
  }

  toString(): string {
    return `${this.source} v${this.version}`;
  }
}
```

### Quartiles（四分位数）

統計データの四分位数を表す値オブジェクト。

```typescript
class Quartiles {
  private constructor(
    private readonly q1: number,
    private readonly q2: number,
    private readonly q3: number
  ) {
    if (q1 > q2 || q2 > q3) {
      throw new Error("Quartiles must be in ascending order");
    }
  }

  static create(q1: number, q2: number, q3: number): Quartiles {
    return new Quartiles(q1, q2, q3);
  }

  getQ1(): number {
    return this.q1;
  }

  getQ2(): number {
    return this.q2;
  }

  getQ3(): number {
    return this.q3;
  }

  getIQR(): number {
    return this.q3 - this.q1;
  }

  equals(other: Quartiles): boolean {
    return this.q1 === other.q1 && this.q2 === other.q2 && this.q3 === other.q3;
  }
}
```

### Trend（トレンド）

時系列データの傾向を表す値オブジェクト。

```typescript
enum TrendType {
  INCREASING = "increasing",
  DECREASING = "decreasing",
  STABLE = "stable",
  VOLATILE = "volatile",
}

class Trend {
  private constructor(
    private readonly type: TrendType,
    private readonly strength: number,
    private readonly confidence: number
  ) {
    if (strength < 0 || strength > 1) {
      throw new Error("Trend strength must be between 0 and 1");
    }
    if (confidence < 0 || confidence > 1) {
      throw new Error("Trend confidence must be between 0 and 1");
    }
  }

  static create(type: TrendType, strength: number, confidence: number): Trend {
    return new Trend(type, strength, confidence);
  }

  getType(): TrendType {
    return this.type;
  }

  getStrength(): number {
    return this.strength;
  }

  getConfidence(): number {
    return this.confidence;
  }

  isSignificant(): boolean {
    return this.confidence > 0.8;
  }

  equals(other: Trend): boolean {
    return (
      this.type === other.type &&
      this.strength === other.strength &&
      this.confidence === other.confidence
    );
  }
}
```

### Significance（有意性）

統計的有意性を表す値オブジェクト。

```typescript
enum SignificanceLevel {
  NOT_SIGNIFICANT = "not_significant",
  MARGINALLY_SIGNIFICANT = "marginally_significant",
  SIGNIFICANT = "significant",
  HIGHLY_SIGNIFICANT = "highly_significant",
}

class Significance {
  private constructor(
    private readonly level: SignificanceLevel,
    private readonly pValue: number
  ) {
    if (pValue < 0 || pValue > 1) {
      throw new Error("P-value must be between 0 and 1");
    }
  }

  static create(pValue: number): Significance {
    let level: SignificanceLevel;
    if (pValue > 0.1) {
      level = SignificanceLevel.NOT_SIGNIFICANT;
    } else if (pValue > 0.05) {
      level = SignificanceLevel.MARGINALLY_SIGNIFICANT;
    } else if (pValue > 0.01) {
      level = SignificanceLevel.SIGNIFICANT;
    } else {
      level = SignificanceLevel.HIGHLY_SIGNIFICANT;
    }
    return new Significance(level, pValue);
  }

  getLevel(): SignificanceLevel {
    return this.level;
  }

  getPValue(): number {
    return this.pValue;
  }

  isSignificant(): boolean {
    return (
      this.level === SignificanceLevel.SIGNIFICANT ||
      this.level === SignificanceLevel.HIGHLY_SIGNIFICANT
    );
  }

  equals(other: Significance): boolean {
    return this.level === other.level && this.pValue === other.pValue;
  }
}
```

## 値オブジェクトの特徴

1. **不変性**: 作成後に変更できない
2. **等価性**: 値が同じなら等しいとみなす
3. **自己完結性**: 必要な情報を全て含む
4. **検証**: 作成時に有効性を検証
5. **表現**: 適切な文字列表現を提供
