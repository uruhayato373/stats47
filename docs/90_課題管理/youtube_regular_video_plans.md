## 企画1: 犯罪・治安ランキング（優先度: 最高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | 未投稿（完全新規） |
| 競合実績 | 「データは語る」7.9万再生、「日本リスクマップ」41万再生 |
| DBキー数 | 18キー（窃盗犯・凶悪犯・覚醒剤・検挙率等） |
| データ年 | 2023年 |

### タイトル候補

1. **都道府県別「泥棒が多い県」ランキング 1位は大阪でも2位は意外すぎる県だった**
2. **都道府県別「治安が悪い県」ワースト10を発表します【犯罪データで判明】**
3. **都道府県別「犯罪が少ない安全な県」TOP10は？検挙率72%のあの県が最強**

### 推奨: 案1

「2位が茨城」という意外性が最も強い。窃盗犯フォーカスで生活密着感も高い。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「窃盗犯が最も多い県、1位は大阪。でも2位、知ってますか？」 |
| 0:15-0:30 | 導入: 2023年の窃盗犯認知件数（人口千対）を使用 |
| 0:30-3:00 | ランキング発表: 47位→1位。コロプレスマップで地域パターン可視化 |
| 3:00-3:30 | 深掘り1: 北関東の闇（茨城2位・群馬3位・栃木5位 + 検挙率ワースト級） |
| 3:30-4:00 | 深掘り2: 検挙率と犯罪件数の逆相関（散布図） |
| 4:00-4:30 | 深掘り3: 犯罪は78%減った事実（2001年→2023年の全国推移） |
| 4:30-4:50 | まとめ + CTA |

### 見どころデータ

- 窃盗犯: 1位大阪(6.65) → 2位茨城(5.24) → 3位群馬(5.21)。東京は8位
- 検挙率: 1位島根(72.7%) vs 47位大阪(26.7%)。2.7倍差
- 覚醒剤: 4位に福井県(8.6)が急浮上（2014年は38位）
- 少年犯罪: 1位和歌山(4.7人)。1980年→2023年で88%減少
- 窃盗犯の20年推移: 全国平均15.35 → 3.29（78.6%減）

## 企画2: 外国人人口ランキング（優先度: 最高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | Shortsのみ（通常動画未投稿） |
| 競合実績 | 移民・外国人テーマは時事性が高く、10万再生超の事例多数 |
| DBキー数 | 14キー（総数 + 中国・韓国・アメリカ・国籍別） |
| データ年 | 2020年（人口比）、2024年（総数） |

### タイトル候補

1. **都道府県別「外国人が多い県」ランキング 国籍を変えると地図が激変する【中国・韓国・アメリカ】**
2. **4年で倍増した県も。在留外国人376万人時代の都道府県ランキング【2024年最新】**
3. **40年で16倍に増えた県。外国人人口の推移を全都道府県で見る【1980→2024】**

### 推奨: 案1

国籍別でコロプレスマップが激変する視覚的インパクトが最強。サムネも作りやすい。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「外国人が多い県、総数で見ると東京1位。でも国籍を変えると…」 |
| 0:15-0:30 | 導入: 在留外国人376万人、40年で5.6倍 |
| 0:30-1:30 | ランキング発表1: 総数TOP10（東京74万→茨城10万） |
| 1:30-2:00 | 転換: 「人口比で見ると地図が変わります」→ 群馬3位・三重4位・岐阜5位 |
| 2:00-2:30 | 国籍別1: 中国人マップ → 首都圏に極度集中 |
| 2:30-3:00 | 国籍別2: 韓国・朝鮮人マップ → 大阪1位(922)、近畿圏に集中 |
| 3:00-3:30 | 国籍別3: アメリカ人マップ → 沖縄1位(197)、米軍基地 |
| 3:30-4:00 | 深掘り: 群馬84%が「その他」→ブラジル人集住（入管法改正1990年） |
| 4:00-4:30 | 急増データ: 熊本+101%（TSMC効果）、北海道+97% |
| 4:30-4:50 | まとめ + CTA |

### 見どころデータ

- 総数: 東京74万 > 大阪33万 > 愛知33万
- 人口比: 群馬3位(2,756)、三重4位(2,526) — 製造業集住
- 中国人: 首都圏4県がTOP5独占
- 韓国人: 大阪1位(922)、東京は4位(565) — 歴史的背景
- アメリカ人: 沖縄1位(197)、2位東京の1.7倍 — 米軍基地
- 群馬の「その他」84% → ブラジル人（大泉町）
- 熊本が4年で倍増（TSMC関連）
- 40年で群馬15.9倍増

## 企画3: 生活保護率ランキング（優先度: 高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | BCR（推移動画）のみ。単年深堀り未投稿 |
| 競合実績 | 「データは語る」21.6万再生 |
| DBキー数 | 11キー（被保護人員・世帯・医療扶助・住宅扶助等） |
| データ年 | 2023年 |

### タイトル候補

1. **都道府県別 生活保護率ランキング 33人に1人の大阪と、237人に1人の富山【7.2倍差】**
2. **東京は所得1位なのに生活保護率9位。都道府県データで見る日本の格差の真実**
3. **都道府県別「生活保護が少ない県」TOP10 北陸3県が独占する意外な理由**

### 推奨: 案1

「33人に1人」と「237人に1人」の対比が衝撃的。具体的数字は再生数を伸ばす。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「大阪では33人に1人が生活保護。でも富山では237人に1人」 |
| 0:15-0:30 | 導入: 被保護実人員（人口千対）2023年データ |
| 0:30-2:30 | ランキング発表: 47位→1位。コロプレスマップ |
| 2:30-3:00 | 深掘り1: 大阪の推移（1996年11.68→2013年34.22ピーク→2023年30.42 低下中） |
| 3:00-3:30 | 深掘り2: 北陸3県の秘密（共働き率全国TOP3と完全一致） |
| 3:30-4:00 | 深掘り3: 東京の矛盾（所得1位なのに保護率9位 → 都市型格差） |
| 4:00-4:30 | クロス分析: 県民所得 vs 生活保護率の散布図（相関-0.17の弱さ） |
| 4:30-4:50 | まとめ: 「豊かさ」は所得だけで決まらない + CTA |

### 見どころデータ

- 1位大阪(30.42) vs 47位富山(4.23) = 7.19倍差
- 大阪: 約33人に1人が被保護者
- 北陸3県: 富山47位・福井45位・石川43位 ← 共働き率TOP3と完全一致
- 東京: 所得1位なのに保護率9位(19.61) — 都市型格差の証拠
- 県民所得 vs 生活保護率の相関: r=-0.17（予想外に弱い）
- 医療扶助: 大阪25.97（ほぼ全受給者に適用 → 医療費問題）

## 企画4: 県民所得・年収ランキング（優先度: 高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | Shorts投稿済み（県民所得）。通常動画未投稿 |
| 競合実績 | 年収・所得系は安定して数万再生 |
| DBキー数 | 25キー（県民所得・可処分所得・世帯年収等） |
| データ年 | 県民所得2020年、可処分所得2024年 |

### タイトル候補

1. **都道府県別 県民所得ランキング 3位は福井県。大阪は22位【誰も知らない年収格差】**
2. **都道府県別「本当に豊かな県」ランキング 東京と沖縄で2.4倍差の衝撃**
3. **都道府県別 可処分所得ランキング 手取りで見ると"意外な県"が上位に来る**

### 推奨: 案1

「3位福井」「大阪22位」の意外性が最強フック。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「1人当たり県民所得3位、東京でも大阪でもない。福井県です」 |
| 0:15-0:30 | 導入: 1人当たり県民所得（2020年）の概要 |
| 0:30-2:30 | ランキング発表: 47位沖縄(217万)→1位東京(521万)。コロプレスマップ |
| 2:30-3:00 | 深掘り1: 福井3位・栃木4位・富山5位 — なぜ北陸・北関東が強い？（製造業+共働き） |
| 3:00-3:30 | 深掘り2: 大阪22位(283万) — 第二の都市のイメージと現実のギャップ |
| 3:30-4:00 | 転換: 可処分所得で見ると？→ 埼玉2位・千葉3位・奈良4位（ベッドタウン効果） |
| 4:00-4:30 | 深掘り3: 最下位は沖縄でなく愛媛(42万/月) — 可処分所得の意外な最下位 |
| 4:30-4:50 | まとめ + CTA |

### 見どころデータ

- 県民所得: 1位東京(521万) vs 47位沖縄(217万) = 2.41倍差
- 3位福井(318万) > 22位大阪(283万) — 「第二の都市」の実態
- 北陸・北関東が上位独占: 製造業 × 共働き × 持ち家
- 可処分所得: 2位埼玉(62万) > 3位千葉(61万) — ベッドタウン効果
- 可処分所得の最下位は愛媛(42万) — 沖縄より低い
- 奈良4位(60万) — 大阪通勤圏の手取り力

## 企画5: 離婚率ランキング（優先度: 高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | Shortsのみ。通常動画未投稿 |
| 競合実績 | 離婚・結婚テーマは生活密着で安定 |
| DBキー数 | 6キー（離婚率・婚姻率・初婚年齢等） |
| データ年 | 2022年 |

### タイトル候補

1. **都道府県別 離婚率ランキング 沖縄が47年間ずっと1位…北陸が最下位の意外な理由**
2. **結婚した約半分が離婚する県がある。離婚/婚姻比率で見る都道府県ランキング**
3. **東京は婚姻率1位なのに離婚が少ない理由。都道府県・離婚ランキングの闇と光**

### 推奨: 案1

「47年間ずっと1位」のインパクトと「北陸が最下位の理由」のセットが強い。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「離婚率1位の県、47年間ずっと同じ県です。沖縄。」 |
| 0:15-0:30 | 導入: 離婚率（人口千対）2022年データ |
| 0:30-2:30 | ランキング発表: 47位富山(1.06)→1位沖縄(2.10)。コロプレスマップ |
| 2:30-3:00 | 深掘り1: 沖縄は婚姻率も4位 → 「結婚も離婚も多い」構造 |
| 3:00-3:30 | 深掘り2: 離婚/婚姻比率 → 高知48.8%が実は1位。約2組に1組 |
| 3:30-4:00 | 深掘り3: 北陸3県の秘密（富山47位・石川46位・福井44位 → 三世代同居+共働き+持ち家） |
| 4:00-4:20 | 東京の逆転: 婚姻率1位なのに離婚/婚姻比率は47位(25.6%)。初婚年齢32.3歳 |
| 4:20-4:50 | 経年変化: 全国的に2007-2009がピーク → 低下傾向 + CTA |

### 見どころデータ

- 沖縄: 離婚率1位(2.10)を47年間維持。2位宮崎(1.67)を大きく引き離す
- 離婚/婚姻比率: 1位高知(48.8%)、2位沖縄(47.1%) — 約2組に1組
- 北陸3県: 離婚率も離婚/婚姻比率も全国最低レベル
- 東京: 婚姻率1位(5.36) × 離婚/婚姻比率47位(25.6%) — 「慎重に結婚」
- 相関: 生活保護高齢者率 × 離婚率 r=0.730（貧困と離婚の強い関連）
- 経年: 全国的に2007-2009年ピーク → 低下傾向

## 企画6: 自殺率ランキング（優先度: 中高）

### 基本情報

| 項目 | 内容 |
|---|---|
| 投稿状況 | BCR（推移動画）のみ。単年深堀り未投稿 |
| 競合実績 | センシティブだがデータ系では需要あり |
| DBキー数 | 4キー（自殺率・自殺者数・65歳以上自殺者数） |
| データ年 | 2023年 |

### タイトル候補

1. **都道府県別 自殺率ランキング "ワースト1位"から脱出した秋田県の奇跡【42.6→19.4】**
2. **都道府県別 自殺率ランキング 2004年と2023年で顔ぶれがまったく違う衝撃の事実**
3. **福井県がまた最下位。自殺率が低い県は幸福度ランキングと一致するのか検証**

### 推奨: 案1

秋田の劇的改善ストーリーは希望があり、センシティブなテーマでもポジティブな着地ができる。

### 動画構成（5分）

| 時間 | 内容 |
|---|---|
| 0:00-0:15 | フック: 「かつて自殺率42.6で全国ワースト1位だった秋田県。2023年は19.4。何が起きたのか」 |
| 0:15-0:30 | 導入: 自殺率（人口10万対）2023年データ |
| 0:30-2:00 | ランキング発表: 47位福井(13.6)→1位和歌山(21.8)。コロプレスマップ |
| 2:00-2:30 | 比較: 2004年のランキング（秋田39.2が1位、東北独占）と2023年の違い |
| 2:30-3:30 | 秋田の奇跡: 2006年ピーク42.6→2023年19.4の54%減。藤里モデル等の対策 |
| 3:30-4:00 | 深掘り: 福井が最下位 → 離婚率・生活保護率でも最下位圏。「幸福県」の条件 |
| 4:00-4:30 | 全国トレンド: 全体的に低下傾向。相談窓口の情報 |
| 4:30-4:50 | まとめ + 相談窓口案内 + CTA |

### 見どころデータ

- 2023年1位: 和歌山(21.8)、2位宮崎(21.5)、3位岩手(20.2)
- 47位: 福井(13.6) — 離婚率44位・生活保護率45位でもある「幸福県」
- 秋田の軌跡: 2006年42.6(ワースト1位) → 2023年19.4(6位) = 54%減
- 秋田の自殺者数: 1999年486人 → 2023年176人 = 64%減
- 2004年TOP5は東北・日本海側が独占 → 2023年は分散化
- 実数では東京1位(2,196人)、うち24%が65歳以上
- 相関: 病院病床稼働率 × 自殺率 r=-0.465（医療インフラ充実で低い）

### 注意事項

- 相談窓口（いのちの電話等）を動画末尾と概要欄に必ず記載
- データに基づく客観的な報告に徹し、特定地域を貶める表現を避ける
- YouTubeのセンシティブコンテンツポリシーに準拠
