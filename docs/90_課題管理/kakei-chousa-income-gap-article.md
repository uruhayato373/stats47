# 年収階級別 消費支出格差の記事企画

## コンセプト

家計調査の年収五分位データを使い、「年収によって何にお金を使うかがどれだけ違うか」を品目別に可視化する記事。

## データソース

- **statsDataId**: `0002070005`（用途分類 × 年間収入五分位階級別、年次）
- **cdCat02**: `03`（二人以上の世帯 2000年～）
- **年収五分位**: Q1(低所得) 〜 Q5(高所得) の5段階 + 平均
- **地域**: 全国のみ
- **品目数**: 265品目（用途分類）

## 記事構成案

### タイトル案

- 「年収で53倍の差 ── 家計調査が暴く"教育格差"の実態」
- 「年収別の家計簿を丸裸にする ── 家計調査265品目の完全分析」

### 構成

1. **導入**: 消費支出全体は2.1倍だが、品目別に見ると53倍の格差がある
2. **教育格差**: 補習教育53倍、授業料19倍 → 教育機会の不平等
3. **生活必需品は格差が小さい**: 食料1.6倍、光熱1.2倍、魚介類1.0倍
4. **嗜好品・ぜいたく品で格差が開く**: 洋服5.3倍、自動車4.4倍、教養娯楽2.7倍
5. **時系列変化**: 過去10年で格差は拡大しているか？（同データで2014年と比較）
6. **stats47の都道府県ランキングとの連携**: 品目別消費の地域差リンク

## チャート案

1. **倍率ランキング棒グラフ**: 全品目の Q5/Q1 倍率を降順で表示
2. **大分類の年収階級別 積み上げ棒グラフ**: Q1〜Q5 の支出構成比
3. **教育費の年収階級別 棒グラフ**: 授業料・補習教育・教科書の内訳
4. **食料の内訳比較**: Q1 vs Q5 のレーダーチャート or 棒グラフ
5. **時系列折れ線**: 教育格差倍率の推移（2000年〜2024年）

## 実行手順

1. e-Stat API で全265品目 × 年収五分位 × 2024年データを JSON 取得
2. 時系列比較用に 2014年データも取得
3. `/generate-article-charts` でチャート SVG 生成
4. 記事執筆（ブログ or note）
