# X（Twitter）引用RT戦略

作成日: 2026-03-25
ステータス: 企画段階

## 目的

stats47に関連するバズツイートを引用RTし、データ画像を添付することでインプレッションとフォロワーを伸ばす。

## 運用ルール

| 項目 | 推奨 |
|---|---|
| 頻度 | 1日2〜3件まで（自然な範囲） |
| 対象ツイート | 1,000いいね以上のバズツイート |
| 添付 | stats47のコロプレスマップ画像やランキング画像 |
| テキスト | データの事実＋意外な発見を1行（URLは3回に1回程度） |
| 比率 | 引用RT 3 : オリジナル投稿 7 |

## 有効な理由

- データは引用RTの最強コンテンツ。事実の「裏付け」として引用されやすい
- 地図・ランキング画像は視覚的インパクトがあり、タイムラインで目を引く
- 引用RTは元ツイート主にも通知が行くため、フォロー・RTされる確率が高い

## 注意点

### やるべき形
- バズツイート（少子化・地方創生・物価・治安など）に対し、関連するstats47のデータ画像＋一言コメントで引用
- 「ちなみに都道府県別データで見るとこうなっています」のような補足・検証の立場

### 避けるべき形
- 機械的に大量引用 → スパム判定・アカウント凍結リスク
- 宣伝臭が強い（毎回サイトURL付き）→ ミュートされる
- 元ツイートと無関係なデータを貼る → 逆効果

## 実現方法

ブラウザCLIで「関連ツイートの検索→引用RT候補のリストアップ」まで自動化。投稿自体は手動で確認してから行う（セミオート運用）。
