# GA4 管理画面 設定ガイド

## 設定済み

- [x] データ保持期間: 14か月（2026-03-25）
- [x] 内部トラフィックの定義: 自宅IP `2001:ce8:143:6d31` プレフィックス

## 未設定

### 1. 拡張計測

**場所**: 管理 → データストリーム → ウェブストリーム（stats47.jp）→ 拡張計測（歯車アイコン）

以下をONにする:

| 項目 | 説明 |
|---|---|
| スクロール数 | ページ90%到達を自動計測 |
| 離脱クリック | 外部サイトへのリンククリック |
| サイト内検索 | クエリパラメータ `q` を指定 |
| フォームの操作 | フォーム送信の追跡 |

### 2. カスタムディメンション登録

**場所**: 管理 → データの表示 → カスタム定義 → カスタムディメンションを作成

以下5件を登録する:

| ディメンション名 | イベントパラメータ名 | 範囲 |
|---|---|---|
| ランキングキー | `ranking_key` | イベント |
| カテゴリキー | `category_key` | イベント |
| エリアタイプ | `area_type` | イベント |
| 年度コード | `year_code` | イベント |
| 検索語 | `search_term` | イベント |

※ 登録しないとGA4レポートでこれらのパラメータによるフィルタ・集計ができない

### 3. キーイベント（コンバージョン）登録

**場所**: 管理 → データの表示 → キーイベント → 新しいキーイベント

以下3件を登録する:

| イベント名 | 意味 |
|---|---|
| `file_download` | CSVダウンロード |
| `affiliate_click` | アフィリエイトリンククリック（収益直結） |
| `share` | シェアボタンクリック（拡散効果） |

### 4. 内部トラフィックフィルタの有効化

**場所**: 管理 → データの収集と修正 → データフィルタ

1. 「Internal Traffic」をクリック
2. フィルタの状態を **テスト → 有効** に変更
3. 保存

※ 定義は済んでいるが、フィルタ自体が有効になっていないと除外されない
