
# SNS画像サイズ一覧

## X（旧Twitter）

### プロフィール関連

- **プロフィール画像**: 400×400ピクセル（円形に表示）
- **ヘッダー画像**: 1500×500ピクセル

### 投稿画像

- **推奨サイズ（横長）**: 1200×675ピクセル（16:9比率）
- **正方形**: 1080×1080ピクセル（1:1比率）
- **縦長**: 2:3、4:5などの比率も投稿可能
    - タイムライン表示時は一部トリミングされる場合あり
    - クリックで全体表示可能
    - 重要な要素は中央付近に配置推奨

## Instagram

### プロフィール関連

- **プロフィール画像**: 320×320ピクセル（円形に表示）

### 投稿画像

- **フィード（正方形）**: 1080×1080ピクセル（1:1比率）
- **フィード（縦長）**: 1080×1350ピクセル（4:5比率）
- **フィード（横長）**: 1080×566ピクセル（1.91:1比率）

### ストーリーズ・リール

- **ストーリーズ**: 1080×1920ピクセル（9:16比率）
- **リール**: 1080×1920ピクセル（9:16比率）

## Facebook

### プロフィール関連

- **プロフィール写真**: 170×170ピクセル（デスクトップ表示時）
- **カバー写真**:
    - デスクトップ: 820×312ピクセル
    - モバイル: 640×360ピクセル

### 投稿画像

- **投稿画像**: 1200×630ピクセル
- **ストーリーズ**: 1080×1920ピクセル（9:16比率）
- **イベントカバー画像**: 1920×1080ピクセル

// src/actions/generateStatsImage.ts
"use server";

import { ImageGenerator } from "@/lib/image";
import { ColorScheme } from "@/types/features";
import { StatsSchema } from "@/types/stats";
import { SNSImageConfig, StatsImageConfig } from "@/types/image";

// シングルトンインスタンス
const imageGenerator = new ImageGenerator();

/**
 * 統計データから複数プラットフォーム用の画像を一括生成するServer Action
 */
export async function generateMultiPlatformStatsImages(
  config: StatsImageConfig
): Promise<{ success: boolean; images: Record<string, string>; message?: string }> {
  try {
    const results = await Promise.allSettled(
      config.snsConfigs.map(async (snsConfig) => {
        const result = await imageGenerator.generateSNSImage(
          config.data,
          config.title,
          config.subtitle,
          snsConfig,
          config.colorScheme,
          config.useMinValueForScale
        );
        
        return {
          platform: snsConfig.platform,
          ...result
        };
      })
    );

    const images: Record<string, string> = {};
    const errors: string[] = [];

    results.forEach((result, index) => {
      const platform = config.snsConfigs[index].platform;
      
      if (result.status === 'fulfilled' && result.value.success) {
        images[platform] = result.value.filePath!;
      } else {
        errors.push(`${platform}: ${result.status === 'rejected' ? result.reason : result.value.message}`);
      }
    });

    return {
      success: Object.keys(images).length > 0,
      images,
      message: errors.length > 0 ? `一部の画像生成に失敗: ${errors.join(', ')}` : undefined
    };
  } catch (error) {
    return {
      success: false,
      images: {},
      message: "画像生成中にエラーが発生しました"
    };
  }
}

/**
 * 従来の単一画像生成（後方互換性）
 */
export async function generateStatsImage(
  statsData: StatsSchema[],
  title: string,
  dataFile: string,
  colorScheme: ColorScheme,
  useMinValueForScale: boolean
): Promise<{ success: boolean; filePath?: string; message?: string }> {
  try {
    const result = await imageGenerator.generateStatsImages(
      statsData,
      title,
      dataFile,
      colorScheme,
      useMinValueForScale
    );

    return {
      success: result.success,
      message: result.message,
      filePath: result.filePath,
    };
  } catch {
    return {
      success: false,
      message: "画像生成中にエラーが発生しました",
    };
  }
}

### 3. **SNS特化画像生成クラスの実装**

```typescript
// src/lib/image/snsImageGenerator.ts
import { createCanvas, loadImage, registerFont } from 'canvas';
import { SNSImageConfig, StatsSchema } from '@/types/image';
import { ColorScheme } from '@/types/features';

export class SNSImageGenerator {
  private readonly defaultFonts = {
    title: 'Noto Sans JP',
    body: 'Noto Sans JP'
  };

  constructor() {
    this.initializeFonts();
  }

  private async initializeFonts() {
    // フォントの登録
    registerFont('./public/fonts/NotoSansJP-Bold.otf', { family: 'Noto Sans JP', weight: 'bold' });
    registerFont('./public/fonts/NotoSansJP-Regular.otf', { family: 'Noto Sans JP', weight: 'normal' });
  }

  async generateSNSImage(
    data: StatsSchema[],
    title: string,
    subtitle: string,
    snsConfig: SNSImageConfig,
    colorScheme: ColorScheme,
    useMinValueForScale: boolean
  ): Promise<{ success: boolean; filePath?: string; message?: string }> {
    try {
      const canvas = createCanvas(snsConfig.dimensions.width, snsConfig.dimensions.height);
      const ctx = canvas.getContext('2d');

      // 背景の描画
      this.drawBackground(ctx, snsConfig);
      
      // ロゴとブランディング
      if (snsConfig.elements.showLogo) {
        await this.drawLogo(ctx, snsConfig);
      }
      
      // タイトルとサブタイトル
      this.drawText(ctx, title, subtitle, snsConfig);
      
      // データ可視化
      if (snsConfig.elements.showDataVisualization) {
        this.drawDataVisualization(ctx, data, snsConfig, colorScheme);
      }
      
      // コールトゥアクション
      if (snsConfig.elements.showCallToAction) {
        this.drawCallToAction(ctx, snsConfig);
      }

      // 画像の保存
      const filePath = await this.saveImage(canvas, snsConfig.platform, title);
      
      return {
        success: true,
        filePath,
        message: `${snsConfig.platform}用画像が正常に生成されました`
      };
    } catch (error) {
      console.error('SNS画像生成エラー:', error);
      return {
        success: false,
        message: `${snsConfig.platform}用画像の生成に失敗しました`
      };
    }
  }

  private drawBackground(ctx: CanvasRenderingContext2D, config: SNSImageConfig) {
    // グラデーション背景
    const gradient = ctx.createLinearGradient(0, 0, 0, config.dimensions.height);
    gradient.addColorStop(0, config.style.backgroundColor);
    gradient.addColorStop(1, this.adjustBrightness(config.style.backgroundColor, -20));
    
    ctx.fillStyle = gradient;
    ctx.fillRect(0, 0, config.dimensions.width, config.dimensions.height);
  }

  private async drawLogo(ctx: CanvasRenderingContext2D, config: SNSImageConfig) {
    try {
      const logo = await loadImage('./public/images/logo.png');
      const logoSize = 60;
      const x = 30;
      const y = 30;
      
      ctx.drawImage(logo, x, y, logoSize, logoSize);
    } catch (error) {
      console.warn('ロゴの読み込みに失敗:', error);
    }
  }

  private drawText(ctx: CanvasRenderingContext2D, title: string, subtitle: string, config: SNSImageConfig) {
    ctx.font = `bold ${config.style.fontSize.title}px ${config.style.fontFamily}`;
    ctx.fillStyle = config.style.textColor;
    ctx.textAlign = 'center';
    
    // タイトルの描画
    const titleY = config.dimensions.height * 0.3;
    this.drawWrappedText(ctx, title, config.dimensions.width / 2, titleY, config.dimensions.width * 0.8);
    
    // サブタイトルの描画
    if (subtitle) {
      ctx.font = `${config.style.fontSize.subtitle}px ${config.style.fontFamily}`;
      ctx.fillStyle = this.adjustBrightness(config.style.textColor, -30);
      const subtitleY = titleY + config.style.fontSize.title + 20;
      this.drawWrappedText(ctx, subtitle, config.dimensions.width / 2, subtitleY, config.dimensions.width * 0.8);
    }
  }

  private drawDataVisualization(ctx: CanvasRenderingContext2D, data: StatsSchema[], config: SNSImageConfig, colorScheme: ColorScheme) {
    // シンプルな棒グラフまたは円グラフ
    if (data.length <= 5) {
      this.drawBarChart(ctx, data, config);
    } else {
      this.drawPieChart(ctx, data.slice(0, 5), config);
    }
  }

  private drawBarChart(ctx: CanvasRenderingContext2D, data: StatsSchema[], config: SNSImageConfig) {
    const chartArea = {
      x: config.dimensions.width * 0.1,
      y: config.dimensions.height * 0.5,
      width: config.dimensions.width * 0.8,
      height: config.dimensions.height * 0.3
    };

    const maxValue = Math.max(...data.map(d => d.value));
    const barWidth = chartArea.width / data.length * 0.8;
    const barSpacing = chartArea.width / data.length * 0.2;

    data.forEach((item, index) => {
      const barHeight = (item.value / maxValue) * chartArea.height;
      const x = chartArea.x + index * (barWidth + barSpacing);
      const y = chartArea.y + chartArea.height - barHeight;

      // バーの描画
      ctx.fillStyle = config.style.accentColor;
      ctx.fillRect(x, y, barWidth, barHeight);

      // ラベルの描画
      ctx.fillStyle = config.style.textColor;
      ctx.font = `${config.style.fontSize.body}px ${config.style.fontFamily}`;
      ctx.textAlign = 'center';
      ctx.fillText(item.areaName, x + barWidth / 2, chartArea.y + chartArea.height + 20);
    });
  }

  private drawCallToAction(ctx: CanvasRenderingContext2D, config: SNSImageConfig) {
    const ctaText = '詳細を見る';
    const ctaY = config.dimensions.height * 0.85;
    
    // CTAボタンの背景
    const buttonWidth = 200;
    const buttonHeight = 50;
    const buttonX = (config.dimensions.width - buttonWidth) / 2;
    const buttonY = ctaY - buttonHeight / 2;
    
    ctx.fillStyle = config.style.accentColor;
    ctx.fillRect(buttonX, buttonY, buttonWidth, buttonHeight);
    
    // CTAテキスト
    ctx.fillStyle = '#FFFFFF';
    ctx.font = `bold ${config.style.fontSize.body}px ${config.style.fontFamily}`;
    ctx.textAlign = 'center';
    ctx.fillText(ctaText, config.dimensions.width / 2, ctaY + 5);
  }

  private drawWrappedText(ctx: CanvasRenderingContext2D, text: string, x: number, y: number, maxWidth: number) {
    const words = text.split('');
    let line = '';
    let lineHeight = ctx.measureText('M').actualBoundingBoxAscent + ctx.measureText('M').actualBoundingBoxDescent;

    for (let n = 0; n < words.length; n++) {
      const testLine = line + words[n];
      const metrics = ctx.measureText(testLine);
      const testWidth = metrics.width;
      
      if (testWidth > maxWidth && n > 0) {
        ctx.fillText(line, x, y);
        line = words[n];
        y += lineHeight;
      } else {
        line = testLine;
      }
    }
    ctx.fillText(line, x, y);
  }

  private adjustBrightness(color: string, amount: number): string {
    // 色の明度を調整する簡易実装
    const hex = color.replace('#', '');
    const r = Math.max(0, Math.min(255, parseInt(hex.substr(0, 2), 16) + amount));
    const g = Math.max(0, Math.min(255, parseInt(hex.substr(2, 2), 16) + amount));
    const b = Math.max(0, Math.min(255, parseInt(hex.substr(4, 2), 16) + amount));
    
    return `#${r.toString(16).padStart(2, '0')}${g.toString(16).padStart(2, '0')}${b.toString(16).padStart(2, '0')}`;
  }

  private async saveImage(canvas: any, platform: string, title: string): Promise<string> {
    const buffer = canvas.toBuffer('image/png');
    const fileName = `${platform}_${title.replace(/[^a-zA-Z0-9]/g, '_')}_${Date.now()}.png`;
    const filePath = `./public/generated-images/${fileName}`;
    
    // ファイルの保存
    const fs = require('fs').promises;
    await fs.mkdir('./public/generated-images', { recursive: true });
    await fs.writeFile(filePath, buffer);
    
    return `/generated-images/${fileName}`;
  }
}
```

### 5. **バッチ処理による効率化**

```typescript
// src/actions/batchImageGeneration.ts
"use server";

import { generateMultiPlatformStatsImages } from './generateStatsImage';
import { StatsSchema } from '@/types/stats';
import { snsImageConfigs } from '@/config/snsImageConfigs';

export async function generateBatchSNSImages(
  statsDataArray: Array<{
    data: StatsSchema[];
    title: string;
    subtitle?: string;
    colorScheme: string;
    useMinValueForScale: boolean;
  }>
): Promise<{ success: boolean; results: Array<{ title: string; images: Record<string, string> }> }> {
  const results = [];
  
  for (const item of statsDataArray) {
    const config = {
      title: item.title,
      subtitle: item.subtitle,
      data: item.data,
      colorScheme: item.colorScheme as any,
      useMinValueForScale: item.useMinValueForScale,
      snsConfigs: Object.values(snsImageConfigs)
    };
    
    const result = await generateMultiPlatformStatsImages(config);
    
    if (result.success) {
      results.push({
        title: item.title,
        images: result.images
      });
    }
  }
  
  return {
    success: results.length > 0,
    results
  };
}
```

## **期待される効果**

1. **効率性の向上**: 一度の処理で複数プラットフォーム用画像を生成
2. **SNS集客の強化**: 各プラットフォームに最適化された画像でエンゲージメント向上
3. **ブランド統一性**: 一貫したデザインシステムによるブランド強化
4. **自動化**: 統計データ更新時の画像自動生成
5. **コスト削減**: 手動での画像作成作業の削減

このシステムにより、統計データの更新に合わせて、SNS集客に最適化された画像を自動生成できるようになります。
