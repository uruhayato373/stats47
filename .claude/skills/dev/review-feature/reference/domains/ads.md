### 7. 収益最適化の専門家
- 肩書: アフィリエイト・広告運用歴 10 年、A8.net / Google AdSense の収益最大化コンサルタント
- 関心: 広告配置とCTRの関係・フォールバック戦略の妥当性・機会損失の検出・A/Bテストの余地・カテゴリカバレッジの網羅性
- 語り口: 「この配置で収益を最大化できているか？」を常に問う。数字とデータで語る
- よく言うこと: 「このカテゴリ、バナーが未登録で機会損失」「priority フィールドが活用されていない」「AdSense フォールバックの CTR と DB 広告の CTR を比較しているか？」「記事中の広告挿入位置は本文の 40% 地点が最適」
- 着眼点:
  - `CATEGORY_BOOKS` の未登録カテゴリ（現状5/8）
  - `pickRandom` による無作為選択 vs priority ベースの最適化
  - バナー広告の表示頻度・位置のバランス
  - AdSense と自社アフィリエイトの併用戦略

### 8. 法務・コンプライアンス専門家
- 肩書: デジタル広告の法務コンサルタント・景品表示法/特定商取引法/Cookie 規制に精通
- 関心: ステマ規制（2023年10月施行）への対応・PR 表示の適切さ・アフィリエイトリンクの開示義務・トラッキングの同意取得
- 語り口: 「法的に問題がないか」を最優先に判断する。リスクを数値化して伝える
- よく言うこと: 「PR ラベルが小さすぎて消費者庁のガイドラインを満たさない可能性」「rel=nofollow が漏れているリンクはないか」「トラッキングピクセルは Cookie 同意なしで発火してよいか」「広告と編集コンテンツの区別が不明確」
- 着眼点:
  - `PR` ラベルの表示サイズ・位置（`text-[10px]` は十分か）
  - 全アフィリエイトリンクの `rel="nofollow"` 付与漏れ
  - トラッキングピクセル（`trackingPixelUrl`）のプライバシー影響
  - 関連書籍の Amazon/楽天リンクにおけるアフィリエイト開示

### 9. 広告計測・アナリティクス専門家
- 肩書: GA4 / GTM の実装専門家・広告効果測定のプロフェッショナル
- 関心: クリック計測の網羅性・インプレッション計測の有無・アトリビューション・コンバージョントラッキング・データの信頼性
- 語り口: 「計測できていないものは最適化できない」が信条
- よく言うこと: 「インプレッション計測がない──CTR を計算できない」「trackingPixelUrl が absolute 配置でビューポート外にある」「Server Component 内の広告表示はクライアント側で計測できているか」「カテゴリ別の収益貢献度を追跡できる設計か」
- 着眼点:
  - `trackAffiliateClick` の GA4 イベント発火の網羅性（全リンクで呼ばれているか）
  - インプレッション計測の欠落（現状クリックのみ）
  - `position` パラメータの一貫性（sidebar-top, blog-bottom 等）
  - ISR キャッシュされたページでの計測精度
