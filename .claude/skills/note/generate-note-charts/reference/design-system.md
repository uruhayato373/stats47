# note チャート デザインシステム

note 記事用 SVG チャートの共通規約。ブログ記事用（`/generate-article-charts`）の規約を踏襲し、note 向けの差分を記載。

## サイズ

| 種別 | viewBox | PNG 出力 | density |
|------|---------|----------|---------|
| チャート（散布図・棒グラフ等） | 600×540 前後 | 2x 解像度 | 288 |
| カバー画像 | 1280×670 | 等倍 | 72 |
| CSV サンプル | 332×任意 | 2x 解像度 | 288 |

## フォント

```css
font-family: 'Hiragino Sans', 'Noto Sans JP', sans-serif;
```

日本語は Hiragino Sans 優先。SVG 内で `<style>` ブロックに指定する。

## ドットサイズ

| 種別 | r |
|------|---|
| ハイライトあり | 5 |
| ハイライトなし | 3.5 |
| カバー画像（キー） | 6 |
| カバー画像（通常） | 4 |

## ラベル

- ハイライトされた都道府県名: `font-size="10.5"`, `font-weight="bold"`, 色はドットと同じ
- ラベルオフセット: `dx: ±8`, `dy: -8〜14` でドットとの重なりを回避
- `text-anchor`: ドットの左側なら `"end"`、右側なら `"start"`

## 凡例

- プロット領域の左下に配置
- 背景: `white` (opacity 0.85)、角丸 `rx="4"`
- 行高: 16px
- ドット: `r="4"`, テキスト: `font-size="10"`

## カバー画像デザイン指針

**最優先: スマホ表示で目に留まること。** note.com のフィードはスマホ閲覧が大半。サムネイルは小さく表示されるため、情報を詰め込まず「10文字前後のキャッチコピーを大きく配置」するのが最も効果的。

### 基本レイアウト（中央配置型）

```
┌─────────────────────────────────────────┐
│                              stats47    │
│                                         │
│     リード（44px, #64748b）              │
│                                         │
│   ██ キャッチコピー ██                   │
│   （150-200px, 900weight, テーマ色）     │
│                                         │
│   ── サブタイトル（26px, #94a3b8）──     │
│                                         │
│   [補足データ: バー・ファクト等]          │
│                                         │
│   ─── 区切り線 ───                       │
│   フッター（18px, #94a3b8）              │
└─────────────────────────────────────────┘
```

### 必須ルール

- **白背景**（`#ffffff`）
- **キャッチコピーは10文字前後**、font-size 150-200px、font-weight 900
- **散布図・複雑なチャートは載せない**（スマホで潰れて読めない）
- 補足データは簡素に: 比較バー（2本程度）、ファクト1行、凡例ドットなど
- 装飾は控えめ: 淡い円（`#f1f5f9` 系）+ radialGradient のグロー程度
- stats47 ウォーターマーク: 右上、18px、`#cbd5e1`

### テーマ色の選び方

記事のテーマに合わせてキャッチコピーの色を選ぶ:

| テーマ | 色 | 例 |
|--------|----|----|
| 危機・格差・借金 | `#ea580c` / `#dc2626` | 借金1位、56倍 |
| 日照・暖かさ | `#d97706` | 埼玉県 |
| 経済・成長 | `#2563eb` | — |
| 環境・健康 | `#059669` | — |

### PNG 変換

```js
sharp(Buffer.from(svg), { density: 144 })
  .resize(1280, 670)
  .png()
  .toFile('cover-1280x670.png');
```

density は 144（viewBox 1280×670 → 出力 1280×670px）。
