# チャート種別の選び方

## シリーズ別の推奨チャート構成

### B シリーズ（ランキング読み物）

2〜3 枚。テーマの核心を視覚化する。

- **散布図**: 2 指標の関係性を見せるメインチャート（最も汎用的）
- **カバー画像**: 必須。散布図のミニ版をカバー右側に配置

オプション:
- 横棒グラフ（上位5＋下位5）: 単一指標のランキングを視覚化
- フレームワーク図: 制度や分類の全体像を説明

### C シリーズ（統計調査ガイド）

3〜4 枚。stats47 の可視化能力のショーケース。

- 横棒グラフ（上位5＋下位5）: ランキングの「差」を実感させる
- タイルグリッドマップ: 全国一覧の体験を提供
- 散布図: 2 指標の相関で「こんな分析もできる」と訴求
- カバー画像: タイルマップのミニ版をカバーに配置

### D シリーズ（深掘り分析）

3〜4 枚。複数指標の関係性を掘り下げる。

- 散布図: 2 指標の相関（メイン）
- 折れ線グラフ: 時系列変化
- カバー画像: 散布図のミニ版をカバーに配置

## チャート種別と生成方法

| チャート | 生成方法 | viewBox | 備考 |
|----------|----------|---------|------|
| 散布図 | `scripts/scatter.js` | 600×540 | JSON 設定から自動生成 |
| カバー画像 | `scripts/cover-template.js` | 1280×670 | テンプレートをコピーして編集 |
| 横棒グラフ | `@stats47/svg-builder` | 960×624 | `generateBarChartSvg` |
| タイルマップ | `@stats47/svg-builder` | 600×700 | `generateChoroplethSvg` |
| 折れ線 | `@stats47/svg-builder` | 680×400 | `generateLineSvg` |
| フレームワーク図 | カスタムスクリプト | 任意 | design-system.md に従う |
| CSV サンプル | カスタムスクリプト | 332×任意 | テーブル形式 |

## 散布図の設計ポイント

散布図は note 記事で最も使用頻度が高い。効果的な散布図のポイント:

1. **グループ分類を色で表現**: 3〜5 グループに色分けし、凡例で説明
2. **キーとなる都道府県にラベル表示**: 8〜15 県程度。多すぎると読みにくい
3. **ラベル重なり回避**: `labelOffsets` で dx/dy/anchor を手動調整
4. **ゾーンラベル**: プロット四隅にパターン名を薄く表示（任意）
5. **軸ラベルに方向と意味を含める**: 「→高いほど〇〇」形式

## カバー画像の設計ポイント

1. **左テキスト＋右チャートの 2 カラム構成**
2. **タイトルは 2 行以内、36px bold**: 記事のフック（疑問形・対比）
3. **ミニチャートは詳細ラベル不要**: 6 県程度のキーラベルのみ
4. **アクセントカラーは記事テーマに合わせる**: 財政系はオレンジ、人口系は青等
5. **density: 72 で等倍出力**: 1280×670px ちょうどになるように
