## 推奨アクション

### 最優先 (今日中に着手推奨)

1. **宮城県地震 (★★★ / クロスソースヒット 3)** — 3 ソース同時報道で時事性 max。既存メトリクスに `disaster-damage-amount-per-person` (災害被害額) と `fire-earthquake-insurance-consumption-expenditure` (火災地震保険料) があり、東日本大震災以降の県別地震保険加入率と被害額の相関で「地震リスクの可視化」が即書ける。
2. **ナフサショック (★★☆ / クロスソースヒット 3)** — 「ナフサ高騰→食品包装→値上げ」連鎖がトレンド軸。都道府県別の石油化学業出荷額・プラスチック関連事業所数で「どの県の経済が中東依存か」マップ化。データは要 estat 取得だが切り口が新鮮。
3. **GSC_小麦粉消費量 / GSC_うどん消費量 (★★★ / コンテンツギャップ)** — 自サイト需要が +5,533%/+478% で急増。既存記事 `noodle-consumption-prefecture-character` の派生として、`wheat-flour-consumption-quantity` メトリクスを使った深堀記事を 1 本追加。確実に流入が見込める。

### 次点

4. **変動金利_住宅ローン (★★★)** — 「金利ある世界」が定着。estat に世帯別住宅ローン残高ありの統計表が豊富（registered=0、candidate=6）で、都道府県別の住宅ローン残高世帯率や持ち家比率の県差をテーマにできる。
5. **東京水道基本料金無償化 (★★★)** — 既存記事 3 本あり差別化必須だが、`utilities-expenditure-ratio` (光熱・水道費割合) を使い「東京以外で水道無償化したらいくら家計に効くか」シミュレーション系で書くと既存と被らない。
6. **ナシ婚 (★★★)** — `marriages` `divorces` `unmarried-ratio-*` メトリクス豊富。既存記事との差別化として「結婚式産業」の県別経済規模に着目した記事を提案。

### 経過観察 (要新規データ)
