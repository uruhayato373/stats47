## 推奨アクション

1. **完全養殖ウナギ（候補1）を最優先で執筆** — 世界初という明確なニュースフック、土用の丑前の季節性、`inland-aquaculture-harvest` 等の DB データが揃っており即執筆可能。既存の漁業記事（`fishery-catch-aquaculture-shift`）は漁業全体論なので、ウナギ・内水面養殖に特化すれば差別化できる。
2. **観光列車・鉄道輸送（候補2）** — 鉄道専用記事が stats47 に未整備の空白領域。`jr-passenger-transport` `railway-station-count` で即執筆可。JR東日本×西武の具体ニュースをフックに。
3. **移住・転入超過（候補3）** — note のデータ分析系競合が同テーマを量産＝検索需要が確実に高い。`moving-in-excess-rate`（2024）で即執筆可。stats47 はチャート＋出典明記の信頼性で差別化。
4. **データ取得を伴う中期候補** — 麻疹/予防接種（候補5）と熱中症搬送（候補4）は e-Stat 取得が必要。`/search-estat` → `/fetch-estat-data` を経てから着手。
5. **既存記事の改善で対応** — 物価・値上げ（候補6）と貯蓄（候補7）は既存記事 6〜2 本でカバー済み。新規より既存記事の鮮度更新を推奨。病床利用率（候補8）は GSC で自サイト実需要が出ているが既存 `hospital-bed-utilization-map` が流入を獲得済み＝関連記事の内部リンク強化が有効。
