## 基本構造

家計調査は**都道府県庁所在市**（＋政令指定都市）単位のデータ。
社会・人口統計体系（`lvArea=2` で都道府県取得）とは取得方法が異なる。

| パラメータ | 値 | 備考 |
|---|---|---|
| statsDataId | `0003348239` / `0003348235` | 金額 / 数量 |
| cat01 | 品目コード | 品目ごとに異なる |
| cat02 | `03` (二人以上の世帯) | `01`=勤労者世帯, `02`=勤労者以外, `03`=総数 |
| エリア | `lvArea` 指定なし | `lvArea=2` では該当データなし |
| 年次 | `2007000000`〜`2024000000` 等 | cdTime で指定 |

## 取得時の注意点

### 1. エリアフィルタ

`lvArea=2`（都道府県レベル）では該当データがない。**フィルタなし**で全エリアを取得し、プログラム側でフィルタする。

### 2. 政令指定都市の重複

県庁所在市が政令指定都市の場合、**同一県に複数行**が返る（2023年時点で52行）。

| area code | 都市名 | 都道府県 |
|---|---|---|
| `14003` | 横浜市 | 神奈川県 |
| `14004` | 川崎市 | 神奈川県 |
| `14150` | 相模原市 | 神奈川県 |
| `22003` | 静岡市 | 静岡県 |
| `22004` | 浜松市 | 静岡県 |
| `27003` | 大阪市 | 大阪府 |
| `27004` | 堺市 | 大阪府 |
| `40003` | 北九州市 | 福岡県 |
| `40004` | 福岡市 | 福岡県 |

**対処法**: area code の先頭2桁でグルーピングし、各都道府県の**最初の行**（県庁所在市）を採用する。

```javascript
const seen = new Set();
const prefData = [];
for (const v of values) {
  if (v['@area'] === '00000') continue; // 全国を除外
  const prefCode = v['@area'].substring(0, 2);
  if (seen.has(prefCode)) continue;
  seen.add(prefCode);
  prefData.push({ areaCode: prefCode, value: parseInt(v['$']) });
}
```

### 3. populate スクリプトは非対応

`packages/ranking/src/scripts/populate-all-rankings.ts` は社会・人口統計体系向け。
家計調査データは専用スクリプト（extract-items → bulk-register → populate-data）で投入済み。
追加登録が必要な場合は同様のパターンで直接 better-sqlite3 投入する。

### 制約事項

- **年収別データは全国集計のみ** — 都道府県×年収のクロス集計は標本数不足のため存在しない
- 都道府県別コロプレス地図とは直接組み合わせ不可
- ダッシュボードで使う場合は「全国レベルのチャート」として配置する設計が必要

## ダッシュボード・ブログでの活用パターン

### パターン A: 都道府県庁所在市別ダッシュボード（地図あり）

- **データ**: `0003348239`（品目別×47庁所在市）
- **地図**: 都道府県に代表値として割り当てたコロプレス
- **チャート**: Diverging Bar Chart（全国平均との差）、ドーナツ（費目構成比）
- **ドリルダウン**: 都道府県クリック → その庁所在市の費目別プロファイル

### パターン B: 年収別消費格差チャート（地図なし）

- **データ**: `0003348240`（五分位階級別×品目）
- **チャート**: Stacked Bar（五分位×10大費目）、折れ線（時系列推移）
- **テーマ**: 「所得格差と消費」「エンゲル係数の年収別推移」

### パターン C: 年収別 金融資産・負債分析（地図なし）

- **データ**: `0002210011`（五分位×貯蓄負債）
- **チャート**: Diverging Bar（貯蓄 vs 負債）、積み上げ棒（金融資産内訳）
- **テーマ**: 「貯蓄格差」「住宅ローンと年収」「投資の年収別浸透度」

### パターン D: 品目ランキング記事（ブログ向け）

- **データ**: `0003348239` の個別品目（level 4-5）
- **記事例**: 「ぎょうざ消費額ランキング」「ビール vs ワイン 地域差」「コーヒー vs 紅茶」
- **DB 登録済み**: 675品目が ranking_items に登録済み、即利用可能
