# EstatDataFetcher にヘッダー追加

## 概要

`EstatMetaInfoSidebar`と同様のヘッダースタイルを`EstatDataFetcher`に追加し、「検索条件」というタイトルと Search アイコンを表示する。

## 変更内容

### `src/features/estat-api/stats-data/components/EstatDataFetcher/EstatDataFetcher.tsx`

1. **ヘッダーセクションを追加**

`<Form>`の直後、`<form>`の前に以下のヘッダーを追加:

```tsx
<div className="flex items-center gap-2 p-4 border-b border-gray-200 dark:border-neutral-700">
  <Search className="w-5 h-5 text-indigo-600" />
  <h3 className="font-medium text-gray-900 dark:text-neutral-100">検索条件</h3>
</div>
```

2. **構造の調整**

全体構造:

```tsx
return (
  <Form {...form}>
    {/* ヘッダー */}
    <div className="flex items-center gap-2 p-4 border-b border-gray-200 dark:border-neutral-700">
      <Search className="w-5 h-5 text-indigo-600" />
      <h3 className="font-medium text-gray-900 dark:text-neutral-100">
        検索条件
      </h3>
    </div>

    {/* フォーム本体 */}
    <form onSubmit={form.handleSubmit(onSubmit)} className="space-y-6 p-4">
      {/* 既存のフォーム内容 */}
    </form>
  </Form>
);
```

3. **フォームに padding を追加**

`<form>`タグに`p-4`を追加して、ヘッダー下のコンテンツに余白を確保。

## 理由

- `EstatMetaInfoSidebar`と統一された UI デザイン
- サイドバーの役割が明確になる
- ユーザーがフォームの目的を理解しやすくなる
