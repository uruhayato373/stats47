# estat-api 共通レイアウト実装

## 概要

`src/app/admin/dev-tools/estat-api/`配下のページに共通の ResizablePanel レイアウトを適用し、各ページで異なるサイドバーコンポーネントを使用できるようにする。Next.js の並列ルート（Parallel Routes）機能を使用。

## アーキテクチャ

```
estat-api/
├── layout.tsx                    # 共通レイアウト（ResizablePanel）
├── @main/                        # メインコンテンツスロット
│   ├── default.tsx
│   ├── meta-info/
│   │   └── page.tsx
│   ├── stats-data/
│   │   └── page.tsx
│   └── stats-list/
│       └── page.tsx
└── @sidebar/                     # サイドバースロット
    ├── default.tsx
    ├── meta-info/
    │   └── page.tsx              # EstatMetaInfoSidebar
    ├── stats-data/
    │   └── page.tsx              # EstatDataFetcher
    └── stats-list/
        └── page.tsx              # (必要に応じて)
```

## 実装手順

### 1. 共通レイアウト作成

**`src/app/admin/dev-tools/estat-api/layout.tsx`**

```tsx
import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from "@/components/atoms/ui/resizable";

export default function EstatApiLayout({
  main,
  sidebar,
}: {
  main: React.ReactNode;
  sidebar: React.ReactNode;
}) {
  return (
    <ResizablePanelGroup direction="horizontal" className="min-h-screen">
      {/* メインコンテンツエリア */}
      <ResizablePanel defaultSize={70} minSize={50}>
        <div className="h-full overflow-auto">{main}</div>
      </ResizablePanel>

      {/* リサイズハンドル */}
      <ResizableHandle />

      {/* サイドバー（右側） */}
      <ResizablePanel defaultSize={30} minSize={20} maxSize={50}>
        <div className="h-full overflow-auto bg-gray-50 dark:bg-neutral-900 border-l border-border">
          {sidebar}
        </div>
      </ResizablePanel>
    </ResizablePanelGroup>
  );
}
```

### 3. メインコンテンツスロット移行

#### 3-1. meta-info

**`src/app/admin/dev-tools/estat-api/@main/meta-info/page.tsx`**

現在の`meta-info/page.tsx`からレイアウト部分を削除し、メインコンテンツのみを返す:

```tsx
import { fetchMetaInfo } from "@/features/estat-api/meta-info";
import {
  EstatMetaInfoDisplay,
  EstatMetaInfoFetcher,
} from "@/features/estat-api/meta-info/components";
import { buildEnvironmentConfig } from "@/lib/environment";
import { getMockMetaInfo } from "@data/mock/estat-api/meta-info";

export default async function MetaInfoMainSlot({
  searchParams,
}: {
  searchParams: Promise<{ statsId?: string }>;
}) {
  const { statsId } = await searchParams;
  const config = buildEnvironmentConfig();
  let metaInfo = null;
  let error = null;

  if (statsId) {
    try {
      if (config.isMock) {
        metaInfo = getMockMetaInfo(statsId);
        if (!metaInfo) {
          error = `モックデータが見つかりません: ${statsId}`;
        }
      } else {
        metaInfo = await fetchMetaInfo(statsId);
      }
    } catch (err) {
      error =
        err instanceof Error ? err.message : "メタ情報の取得に失敗しました";
    }
  }

  return (
    <div className="p-4 space-y-6">
      <EstatMetaInfoFetcher />
      <EstatMetaInfoDisplay
        metaInfo={metaInfo}
        statsId={statsId || null}
        error={error}
      />
    </div>
  );
}
```

#### 3-2. stats-data

**`src/app/admin/dev-tools/estat-api/@main/stats-data/page.tsx`**

現在の`stats-data/page.tsx`から ResizablePanel 部分を削除:

```tsx
import { fetchFormattedStatsData } from "@/features/estat-api/stats-data";
import { EstatDataDisplay } from "@/features/estat-api/stats-data/components";
import { buildEnvironmentConfig } from "@/lib/environment";
import { getMockStatsData } from "@data/mock/estat-api/stats-data";

export default async function StatsDataMainSlot({
  searchParams,
}: {
  searchParams: Promise<{
    statsDataId?: string;
    cdCat01?: string;
    cdArea?: string;
    cdTime?: string;
    // ... その他のパラメータ
  }>;
}) {
  const params = await searchParams;
  const config = buildEnvironmentConfig();
  let statsData = null;
  let error = null;

  const statsDataId =
    params.statsDataId || (config.isMock ? "0000010101" : undefined);
  const cdCat01 = params.cdCat01 || (config.isMock ? "A1101" : undefined);

  if (statsDataId && cdCat01) {
    try {
      if (config.isMock) {
        statsData = getMockStatsData(statsDataId, cdCat01);
        if (!statsData) {
          error = `モックデータが見つかりません: ${statsDataId}_${cdCat01}`;
        }
      } else {
        statsData = await fetchFormattedStatsData(statsDataId, {
          categoryFilter: cdCat01,
          ...(params.cdArea && { areaFilter: params.cdArea }),
          ...(params.cdTime && { yearFilter: params.cdTime }),
        });
      }
    } catch (err) {
      error = err instanceof Error ? err.message : "データの取得に失敗しました";
    }
  }

  return (
    <div className="p-4">
      <EstatDataDisplay data={statsData} loading={false} error={error} />
    </div>
  );
}
```

## 注意事項

1. **並列ルートの制約**

   - `@main`と`@sidebar`は同じレベルに存在する必要がある
   - 各スロットには必ず`default.tsx`が必要

2. **サイドバーの padding**

   - layout.tsx でサイドバーコンテナに`p-4`を追加するか
   - 各サイドバーコンポーネントで個別に設定するか検討

3. **stats-list**
   - 実装状況を確認して、同様に移行
