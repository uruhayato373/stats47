# EstatMetaInfoFetcher をサイドバーに統合

## 概要

`EstatMetaInfoFetcher`を`EstatMetaInfoSidebar`に統合し、サイドバー内で統計表 ID 検索と保存済みデータ一覧を表示できるようにする。

## 現状

- **メインコンテンツ**: `EstatMetaInfoFetcher` + `EstatMetaInfoDisplay`
- **サイドバー**: `EstatMetaInfoSidebar`（保存済みデータ一覧のみ）

## 変更後

- **メインコンテンツ**: `EstatMetaInfoDisplay`のみ
- **サイドバー**: `EstatMetaInfoFetcher` + `EstatMetaInfoSidebar`（検索フォーム + 保存済みデータ一覧）

## レイアウト構成（変更後）

```
┌─────────────────────────────────────────────┬──────────────────┐
│ メインコンテンツ                            │ サイドバー       │
│                                             │                  │

│ │ - 地域                                  │ │ │ データ一覧   │ │

│ └─────────────────────────────────────────┘ │ └──────────────┘ │
```

## メリット

1. **UI の統一性**: stats-data と同様にサイドバーで検索条件を入力
2. **スペースの有効活用**: メインエリアがより広く使える
3. **操作性の向上**: 検索と保存済みデータが同じ場所にある
4. **一貫性**: 全ページで同じレイアウトパターン

## 注意事項

- `EstatMetaInfoFetcher`はクライアントコンポーネント（"use client"）
- `EstatMetaInfoSidebar`もクライアントコンポーネント
- インポートパスに注意（相対パス）
