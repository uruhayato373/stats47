# eStat-API 統計リスト (stats-list)

## 概要

e-Stat API の `getStatsList` エンドポイントを使用して、利用可能な統計表の一覧を取得・検索する機能を担います。

## 主要な責務

- **統計表の検索**: 指定されたキーワード（政府統計名、統計表題名等）に基づいて統計表を探索。
- **一覧取得**: 分野別や更新日などでの絞り込み。
- **正規化**: API レスポンスに含まれる統計表情報を、アプリケーション共通の `StatListItem` 形式へ変換。

## 主要なデータ構造

### StatListItem
- `id`: 統計表 ID (StatsDataId)
- `statName`: 政府統計名
- `title`: 統計表のタイトル
- `govOrg`: 作成機関
- `surveyDate`: 調査年月
- `updatedDate`: 更新日

## API パラメータ

| パラメータ | 説明 |
| :--- | :--- |
| `searchWord` | 検索キーワード |
| `searchKind` | 検索対象 (1:統計名, 2:表題, 3:項目) |
| `statsField` | 統計分野コード |
| `limit` | 取得件数 (最大 10,000) |
| `startPosition` | 開始位置 |

## 実装上の注意

- **検索精度**: キーワード検索は e-Stat 側の仕様により、部分一致として機能します。
- **動的検索**: 管理画面での統計表探索ツール等で使用されます。
